# /simplify と /batch：ビルトインスラッシュコマンドの追加

## 原文（日本語に翻訳）

`/simplify` と `/batch` のバンドルされたスラッシュコマンドが追加されました。

## 原文（英語）

Added `/simplify` and `/batch` bundled slash commands

## 概要

Claude Code v2.1.63で、2つの新しいビルトインスラッシュコマンド `/simplify` と `/batch` が追加されました。`/simplify` はコードの品質・再利用性・効率性をレビューして改善するコマンド、`/batch` は複数の操作をまとめて実行するコマンドです。これらはClaude Codeに標準搭載されており、すぐに使用できます。

## 基本的な使い方

### /simplify コマンド

変更したコードをレビューして品質改善を行います：

```
/simplify
```

コードの再利用性、品質、効率性を確認し、問題があれば修正します。

### /batch コマンド

複数の操作をまとめて実行します：

```
/batch
```

## 実践例

### /simplify でリファクタリング後のコードを最適化する

```
# 機能を実装した後に実行
/simplify
```

Claudeが変更箇所を分析し、以下の観点で改善提案・修正を行います：
- 重複コードの排除
- より効率的な実装への置き換え
- 可読性の向上
- 不要な抽象化の削除

### /batch で複数タスクを一括処理する

```
# 複数のファイルや操作をまとめて処理
/batch
```

## 注意点

- これらはビルトインのバンドルコマンドとして提供されており、カスタム設定なしにすぐ使用可能です
- `/simplify` は変更されたコードに対して機能します。レビュー対象のコンテキストを事前に共有しておくと効果的です
- スラッシュコマンドは `/` を入力するとコマンド候補が表示されます
