## 原文（日本語に翻訳）

長いIDE選択範囲が絵文字の途中で切り詰められて文字化け（mojibake）が発生する問題と、ツール実行エラーが静かに握りつぶされてしまうケースを修正しました

## 原文（英語）

Fixed mojibake when a long IDE selection was truncated mid-emoji, and a case where a tool executor error could be silently dropped

## 概要

今回の修正は2つの独立したバグに対応したものです。1つ目は、IDEから送信された長いテキスト選択範囲を切り詰めて表示・処理する際、絵文字などのマルチバイト文字（サロゲートペア）の途中で切断してしまい、文字化け（mojibake）が発生する問題です。修正後は文字の境界を認識したうえで切り詰めが行われるようになりました。2つ目は、ツール実行時に発生したエラーが特定のケースで表面化されずに静かに破棄されてしまう問題で、修正後はエラーが正しく報告されるようになりました。

## 注意点

- Claude Code v2.1.218で修正
- 絵文字を含む長いテキストをIDEから選択してClaude Codeに渡す際の文字化けが解消
- 切り詰め処理は文字（コードポイント/サロゲートペア）境界を尊重するように変更
- 一部のケースでツール実行エラーが表示されずに失敗していた問題も併せて修正され、エラーが正しくユーザーに通知されるようになった
