## 原文（日本語に翻訳）

展開されたツール結果内のリンクをクリックするとリンクが開く代わりにセクションが折りたたまれる問題を修正しました。

## 原文（英語）

Fixed clicking a link inside an expanded tool result collapsing the section instead of opening the link

## 概要

ツール実行結果（例：Bash の出力、Read の内容など）を展開して表示している状態で、その中に含まれるリンクをクリックすると、リンクが開かずにセクションが折りたたまれてしまう問題が修正されました。v2.1.152 以降は、展開されたツール結果内のリンクを正常にクリックして開くことができます。

## 影響を受けていた状況

- ツール実行結果（Bash 出力、ファイル内容など）を展開して表示している状態
- 結果内に URL やリンクが含まれる場合

## 注意点

- v2.1.152 以降では自動的に修正されており、特別な操作は不要です
