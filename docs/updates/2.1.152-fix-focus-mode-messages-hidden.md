## 原文（日本語に翻訳）

非表示アクティビティがないターンでフォーカスモードが誤って「N messages hidden」カウントを表示する問題を修正しました。

## 原文（英語）

Fixed focus mode showing a spurious "N messages hidden" count on turns with no hidden activity

## 概要

フォーカスモード（focus mode）において、実際には非表示になっているメッセージがないにもかかわらず「N messages hidden」というカウントが表示される場合があった問題が修正されました。v2.1.152 以降は、実際に非表示になっているアクティビティがある場合にのみ、このカウントが表示されます。

## 影響を受けていた状況

- フォーカスモードが有効な状態でのセッション
- 非表示にすべきアクティビティがないターンでの誤表示

## 注意点

- v2.1.152 以降では自動的に修正されており、特別な操作は不要です
