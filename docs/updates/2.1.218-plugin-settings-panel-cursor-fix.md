## 原文（日本語に翻訳）

プラグインパネルおよび設定パネルでフォーカスしている行にターミナルカーソルが移動しないバグを修正し、スクリーンリーダーや画面拡大鏡が矢印キーによるナビゲーションを追従できるようにしました

## 原文（英語）

Fixed plugin and settings panels not moving the terminal cursor to the focused row, so screen readers and magnifiers can follow arrow-key navigation

## 概要

Claude Code v2.1.218で修正された、アクセシビリティに関するバグです。プラグインブラウザや設定パネルでは、矢印キーを使って項目間を移動しても、実際のターミナルカーソルの位置は移動していませんでした。ターミナルのカーソル位置を頼りに画面を追従するスクリーンリーダーや画面拡大鏡では、キーボード操作についていくことができませんでした。修正後は、フォーカスしている行にあわせてターミナルカーソルが正しく移動するようになり、支援技術がナビゲーションに同期して追従できるようになりました。

## 注意点

- Claude Code v2.1.218で修正
- 影響範囲はプラグインパネルおよび設定パネルでの矢印キー操作
- スクリーンリーダーや画面拡大鏡を利用するユーザーの操作性が向上
- ユーザー側で追加設定は不要
