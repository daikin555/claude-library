## 原文（日本語に翻訳）

Vim モード：NORMAL モードでの `/` が bash/zsh の vi-mode に合わせて逆方向履歴検索（Ctrl+R 相当）を開くようになりました。

## 原文（英語）

Vim mode: `/` in NORMAL mode now opens reverse history search (like Ctrl+R), matching bash/zsh vi-mode

## 概要

Claude Code の Vim モードで、NORMAL モード中に `/` を押すと、bash や zsh の vi-mode と同様に逆方向履歴検索が開くようになりました。これにより、過去に入力したプロンプトをインクリメンタルに検索して再利用できます。bash/zsh の vi-mode に慣れているユーザーにとって、一貫した操作感が得られます。

## 基本的な使い方

```
1. /config → Editor mode → Vim を選択して Vim モードを有効化

2. NORMAL モードに切り替える（Esc を押す）

3. / を押して逆方向履歴検索モードを開く

4. 検索したいプロンプトの一部を入力してフィルタリング

5. Enter で選択、Esc でキャンセル
```

## 実践例

### 過去のプロンプトを素早く再利用

長いプロンプトや頻繁に使うコマンドを再入力せずに呼び出す例です。

```
# NORMAL モードで / を押す
/

# 検索クエリを入力（例：テストを実行）
テスト

# → 「テスト」を含む過去のプロンプトが一覧表示される
# Enter で選択して実行
```

### bash/zsh vi-mode との一貫した操作

ターミナルの vi-mode と同じ感覚で Claude Code の履歴を検索できます。

```
# bash/zsh vi-mode での操作と同じ
# Esc（NORMAL モードへ） → / → 検索ワード → Enter
```

## 注意点

- この機能は Vim モードを有効にしている場合のみ利用できます（`/config` → Editor mode → Vim）
- INSERT モードでの `/` は従来通り、そのままテキストとして入力されます
- NORMAL モードでの `/` の旧動作（Vim の通常前方検索）は履歴検索に置き換えられました
- `Ctrl+R` も従来通り逆方向履歴検索として機能します
