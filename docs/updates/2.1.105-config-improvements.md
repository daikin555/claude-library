## 原文（日本語に翻訳）

`/config`のラベルと説明をわかりやすく改善しました。

## 原文（英語）

Improved `/config` labels and descriptions for clarity

## 概要

`/config`コマンドの設定UIが改善され、各設定項目のラベルと説明がより明確になりました。これにより、設定を変更する際に各オプションの意味や影響をより正確に理解できるようになります。

## 基本的な使い方

```
/config
```

設定メニューが表示され、各項目にわかりやすいラベルと説明が付くようになりました。

## 実践例

### モデル設定の変更

```
/config

設定メニュー
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
モデル              claude-sonnet-4-5
  └ 使用するClaudeモデルを選択します

言語設定            日本語 (ja)
  └ Claudeの応答言語を設定します

パーミッションモード   acceptEdits
  └ ファイル編集の自動承認設定
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 設定項目の確認

ラベルが改善されたことで、設定の意味を推測する必要がなくなり、適切な設定を素早く見つけられるようになりました。

## 注意点

- UIの表示が変わりましたが、設定値自体や機能に変更はありません
- `settings.json`を直接編集する場合、UIに表示される説明を参考にできます
