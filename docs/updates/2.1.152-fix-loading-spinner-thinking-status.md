## 原文（日本語に翻訳）

ツール実行中にローディングスピナーが「still thinking」や「almost done thinking」と表示される問題を修正しました。また、各ツール実行後に思考ステータスが「thinking」にリセットされるようになりました。

## 原文（英語）

Fixed the loading spinner showing "still thinking"/"almost done thinking" while a tool is running, and reset the thinking status to "thinking" after each tool

## 概要

ツール（Bash、Read、Edit など）が実行されている最中に、ローディングスピナーが誤って「still thinking」または「almost done thinking」というメッセージを表示することがありました。実際にはモデルが思考中ではなくツールが実行中であるため、これは誤解を招く表示でした。v2.1.152 以降は、ツール実行中は適切なステータスが表示され、ツール完了後は思考ステータスが正常に「thinking」にリセットされます。

## 影響を受けていた状況

- ツール実行（Bash コマンド、ファイル読み取り、編集など）中
- ツール実行後に次のモデル思考が始まる際

## 注意点

- v2.1.152 以降では自動的に修正されており、特別な操作は不要です
