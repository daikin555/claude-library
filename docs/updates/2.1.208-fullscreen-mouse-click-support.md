## 原文（日本語訳）

フルスクリーンモードにおいて、複数選択メニューと「その他（Other）」入力行に対するマウスクリック操作のサポートを追加。

## 原文（英語）

Added mouse-click support for multi-select menus and "Other" input rows in fullscreen mode

## 概要

Claude Codeの[フルスクリーン描画モード](https://code.claude.com/docs/en/fullscreen)は、マウス操作にも対応したちらつきのないレンダリングモードです。これまで単一選択のメニュー（パーミッションプロンプトや `/model`、`/config` など）はクリックで選べるようになっていましたが、複数の項目をトグルできる複数選択メニューと、選択肢にない自由入力用の「その他」行はクリック操作に対応していませんでした。今回のアップデートでこれらもマウスクリックで操作できるようになり、キーボードだけでなくマウスでも一貫した操作が可能になりました。

## 基本的な使い方

フルスクリーンモード（`/tui fullscreen` で有効化）で複数選択メニューが表示された際、選びたい項目をクリックするとトグルでき、送信ボタンをクリックすると選択内容を確定できます。

```text
/tui fullscreen
```

その後、複数選択形式の質問が表示されたら、対象の項目をクリックして選択・解除し、送信ボタンをクリックして確定します。

## 実践例

### アンケート形式の複数選択質問にマウスだけで回答する

Claude Codeが複数の選択肢から該当するものをすべて選ぶよう尋ねてきた場合、キーボードで矢印キーとスペースを操作する代わりに、該当する項目を順にクリックしてから送信ボタンをクリックするだけで回答できます。

```text
[ ] Reactを使っている
[x] TypeScriptを使っている ← クリックしてチェック
[x] テストにVitestを使っている ← クリックしてチェック
[ 送信 ] ← クリックして確定
```

### 選択肢にない項目を「その他」行から自由入力する

用意された選択肢に該当するものがない場合、「その他（Other）」の入力行をクリックするとその場でフォーカスが移り、自由にテキストを入力できます。

```text
( ) TypeScript
( ) Python
( ) その他: ___________ ← クリックしてフォーカスし、テキストを入力
```

## 注意点

- 本機能はフルスクリーン描画モード（`/tui fullscreen` または `CLAUDE_CODE_NO_FLICKER=1`）でのみ有効です。クラシックレンダラーでは対象外です。
- 利用にはClaude Code v2.1.208以降が必要です。
- マウス操作全般が業務の妨げになる場合は、環境変数 `CLAUDE_CODE_DISABLE_MOUSE_CLICKS=1` でクリック・ドラッグ・ホバーだけを無効化しつつホイールスクロールは維持できます。マウスキャプチャ自体を完全に無効化したい場合は `CLAUDE_CODE_DISABLE_MOUSE=1` を使用します。
