## 原文（日本語に翻訳）

サブスクリプション切り替えの提案が、トーストではなくスタートアップアナウンスメントスロットに表示されるようになりました。

## 原文（英語）

The subscription-switch suggestion now shows in the startup announcement slot instead of a toast

## 概要

これまでトースト通知（画面下部に一時的に表示されるポップアップ）として表示されていたサブスクリプション切り替えの案内が、スタートアップアナウンスメントスロット（起動時に表示されるメッセージ領域）に移動しました。作業中に突然通知が表示されることがなくなり、よりスムーズな作業体験を提供します。

## 基本的な使い方

変更は自動的に適用されます。Claude Code の起動時に、サブスクリプション切り替えの案内が必要な場合は起動メッセージとして表示されます。

## 実践例

### トーストと起動時アナウンスメントの違い

**以前（トースト）：**
- 作業中に突然画面下部に通知が表示される
- 数秒後に自動的に消えるため見逃す可能性がある
- 作業の集中を妨げる

**現在（スタートアップアナウンスメント）：**
- Claude Code の起動時に表示される
- 作業開始前に確認できる
- 邪魔にならない

## 注意点

- この変更は UI の表示位置のみの変更です。サブスクリプションの内容や切り替え方法に変更はありません。
- スタートアップアナウンスメントは Claude Code の起動時のみ表示されます。
