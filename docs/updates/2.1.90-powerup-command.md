## 原文（日本語に翻訳）

`/powerup` を追加 — アニメーションデモを使ってClaude Codeの機能を教えるインタラクティブレッスン

## 原文（英語）

Added `/powerup` — interactive lessons teaching Claude Code features with animated demos

## 概要

`/powerup` は、Claude Codeの機能をインタラクティブなレッスン形式で学べる新しいスラッシュコマンドです。アニメーションデモが内蔵されており、実際の動作を見ながら機能を習得できます。初めてClaude Codeを使うユーザーや、見落としがちな機能を発見したいユーザーに最適です。

## 基本的な使い方

```
/powerup
```

セッション内でコマンドを実行するだけで、利用可能なレッスン一覧が表示されます。

## 実践例

### 初回セットアップ時の活用

Claude Codeをインストールしたばかりのユーザーは、まず `/powerup` を実行して基本機能を学ぶのがおすすめです：

```
/powerup
```

表示されるメニューから興味のあるトピックを選択し、アニメーションデモで視覚的に機能を確認できます。

### 特定の機能を深掘りしたいとき

普段使っていない機能や、聞いたことはあるけど使い方がわからない機能を `/powerup` で学ぶことができます。例えば、hooks や MCP など高度な機能のデモも確認できます。

### チームへのオンボーディング

新しいチームメンバーにClaude Codeを紹介する際、`/powerup` を活用することで自己学習を促進できます。ドキュメントを読むよりも直感的に機能を把握できます。

## 注意点

- `/powerup` はセッション内でいつでも実行可能です
- アニメーションデモはターミナルの表示能力に依存するため、一部の環境では表示が異なる場合があります
- レッスンの内容はClaude Codeのバージョンアップに伴い更新されます
