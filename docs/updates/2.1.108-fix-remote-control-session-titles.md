# 修正：Remote ControlセッションのタイトルがWeb UIの設定を上書きするバグ

## 原文（日本語に翻訳）

Web UIで設定したRemote Controlセッションのタイトルが、3番目のメッセージ以降に自動生成タイトルで上書きされていた問題を修正した。

## 原文（英語）

Fixed Remote Control session titles set in the web UI being overwritten by auto-generated titles after the third message.

## 概要

Claude CodeのRemote Control機能を使用してWeb UIからセッションタイトルを設定した場合、3番目のメッセージ以降で自動生成タイトルによって上書きされてしまうバグが修正されました。Web UIで意図的に設定したセッション名が維持されるようになり、セッション管理の一貫性が保たれます。

## 基本的な使い方

### 修正後の正しい動作

```
# Web UIでRemote Controlセッションのタイトルを設定
タイトル: "プロダクションデプロイ準備"

# メッセージを送受信
1番目のメッセージ... → タイトル維持
2番目のメッセージ... → タイトル維持
3番目のメッセージ... → タイトル維持（修正前は自動生成タイトルで上書きされていた）
4番目のメッセージ... → タイトル維持
```

## 注意点

- **v2.1.108で修正済み**: アップデート後は自動的に修正が適用されます。
- **Remote Control機能について**: Claude CodeのRemote Control（リモートコントロール）は、Web UIからClaude Codeセッションを操作する機能です。
