## 原文（日本語に翻訳）

エージェントビューを Esc キーで終了する際に発生するターミナルのズレと数秒間のハングを修正。

## 原文（英語）

Fixed terminal misalignment and a multi-second hang when exiting the agent view by pressing Esc

## 概要

`claude agents` のエージェントビューを Esc キーで終了しようとすると、ターミナルの表示がズレたり、数秒間応答がなくなったりするバグが修正されました。Esc キーによるビューの終了がスムーズに行えるようになります。

## 基本的な使い方

修正後は特別な操作は不要です。`claude agents` でエージェントビューを開き、Esc キーで終了しても正常に動作します：

```bash
claude agents
# エージェントビューが開く
# Esc キーで終了 → ターミナルのズレやハングなし
```

## 注意点

- この問題はターミナルの UI に関するバグであり、データやセッションの内容には影響していませんでした。
- Esc キー以外の終了方法（`q` キーなど）は影響を受けていませんでした。
