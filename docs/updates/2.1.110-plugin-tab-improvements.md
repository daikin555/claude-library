## 原文（日本語に翻訳）

`/plugin` インストール済みタブを改善 — 注意が必要なアイテムとお気に入りが上部に表示、無効なアイテムは折りたたみの後ろに隠れ、`f` キーで選択中のアイテムをお気に入り登録できる

## 原文（英語）

Improved `/plugin` Installed tab — items needing attention and favorites appear at the top, disabled items are hidden behind a fold, and `f` favorites the selected item

## 概要

`/plugin` コマンドのインストール済みタブ（Installed tab）の表示が整理され、より使いやすくなりました。注意が必要なプラグインやお気に入り登録したプラグインが上部に優先表示されるようになり、無効化されたプラグインは折りたたみセクションの中に隠れます。また、`f` キーでその場でお気に入り登録ができるようになりました。

## 基本的な使い方

```bash
# プラグイン管理メニューを開く
/plugin

# Installed タブに移動（インストール済みプラグイン一覧）

# キー操作:
# f  → 選択中のプラグインをお気に入り登録/解除
# ↑↓ → リストのナビゲーション
```

## 実践例

### お気に入りプラグインを素早く見つける

よく使うプラグインをお気に入り登録して、リストの上部に固定表示する：

```bash
/plugin
# → Installed タブへ

# お気に入り登録したいプラグインを選択
# f を押してお気に入り登録

# 以降は常にリスト上部に表示される
```

### 注意が必要なプラグインを確認する

設定が不完全だったり、更新が必要なプラグインは自動的に上部に表示されます：

```bash
/plugin
# → Installed タブを開くと、注意が必要なプラグインが最上部に表示される
# → 問題の詳細を確認して対処
```

### 無効なプラグインの管理

無効化されたプラグインは折りたたみセクションに格納されるため、通常は表示されません：

```bash
/plugin
# → Installed タブ

# 無効プラグインを確認したい場合:
# → 「Disabled items」折りたたみセクションを展開
# → 有効化が必要なものを選んで再有効化
```

### プラグインのステータス一覧

| 表示位置 | 対象 |
|---------|------|
| 最上部 | 注意が必要なプラグイン（エラー、設定不足など） |
| 上部 | お気に入り登録済みプラグイン |
| 中部 | 通常の有効なプラグイン |
| 折りたたみ | 無効化されたプラグイン |

## 注意点

- `f` キーでのお気に入り登録は、`/plugin` の Installed タブ内でのみ有効です
- 注意が必要と判断される条件は、プラグインの状態（認証エラー、依存関係の不足など）によって異なります
- 無効化されたプラグインを再有効化するには、折りたたみセクションを展開する必要があります
