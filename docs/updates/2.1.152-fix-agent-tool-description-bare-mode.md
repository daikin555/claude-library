## 原文（日本語に翻訳）

`--bare` フラグや添付ファイル無効で実行している場合に、Agent ツールの説明が提供されないエージェントリストを参照する問題を修正しました。

## 原文（英語）

Fixed the Agent tool description referencing an agent list that is never delivered when running with `--bare` or with attachments disabled

## 概要

`--bare` フラグを使用した実行や添付ファイルが無効な設定では、Agent ツールの説明文が「エージェントリストを参照してください」のような参照を含んでいましたが、そのリストは実際には提供されていませんでした。v2.1.152 以降は、このような状況でも Agent ツールの説明が正確な情報を提供します。

## 影響を受けていた状況

- `claude --bare` フラグを使用して非インタラクティブ実行している場合
- 添付ファイル機能が無効になっている設定で使用している場合
- Agent ツールを使用するスキルやワークフローを実行している場合

## 注意点

- v2.1.152 以降では自動的に修正されており、特別な操作は不要です
