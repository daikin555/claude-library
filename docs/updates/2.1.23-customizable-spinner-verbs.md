## 原文（日本語訳）
カスタマイズ可能なスピナー動詞設定（`spinnerVerbs`）を追加しました

## 原文（英語）
Added customizable spinner verbs setting (`spinnerVerbs`)

## 概要
Claude が処理中に表示するスピナーメッセージ（「Thinking」「Working」など）をカスタマイズできる `spinnerVerbs` 設定が追加されました。チームで共通の用語を使用したり、プロジェクト固有の表現を設定できます。

## 基本的な使い方

設定ファイルに `spinnerVerbs` を追加します。

```json
{
  "spinnerVerbs": {
    "mode": "append",
    "verbs": ["Cooking...", "Brewing...", "Crafting..."]
  }
}
```

設定ファイルの場所：
- Windows: `%USERPROFILE%\.claude\settings.json`
- macOS/Linux: `~/.claude/settings.json`

## 実践例

### チーム共通のスピナーメッセージ

```json
{
  "spinnerVerbs": {
    "mode": "append",
    "verbs": [
      "コードをレビュー中...",
      "最適解を検討中...",
      "テストを実行中..."
    ]
  }
}
```

これにより、チーム全体で統一された表現を使用できます。

### プロジェクト特有の表現

```json
{
  "spinnerVerbs": {
    "mode": "replace",
    "verbs": [
      "魔法をかけています✨",
      "アイデアを練っています💡",
      "コーヒーを飲んでいます☕"
    ]
  }
}
```

`mode: "replace"` を使用すると、デフォルトのスピナーメッセージを完全に置き換えられます。

### 多言語対応

```json
{
  "spinnerVerbs": {
    "mode": "append",
    "verbs": [
      "Thinking...",
      "考え中...",
      "思考中...",
      "検討中..."
    ]
  }
}
```

複数の言語を混在させることも可能です。

### シンプルな進捗表示

```json
{
  "spinnerVerbs": {
    "mode": "replace",
    "verbs": [
      "Processing",
      "Working",
      "Computing"
    ]
  }
}
```

シンプルな英語表現に統一することもできます。

### チームのトーンに合わせる

```json
{
  "spinnerVerbs": {
    "mode": "append",
    "verbs": [
      "Analyzing your request",
      "Generating solution",
      "Reviewing codebase",
      "Running diagnostics"
    ]
  }
}
```

フォーマルなビジネス環境向けの表現も設定可能です。

## 注意点

- **モード設定**: `mode` には `"append"`（追加）または `"replace"`（置換）を指定します
- **配列形式**: `verbs` は文字列の配列として定義する必要があります
- **設定の反映**: 設定ファイルを変更すると即座に反映されます（再起動不要）
- **チーム共有**: プロジェクトの `.claude/settings.json` に設定すると、チーム全体で共有できます
- **文字数制限**: スピナーメッセージは端末の幅に収まるよう適切な長さに保つことを推奨します
- **絵文字の使用**: 絵文字を使用する場合、端末のフォントが対応していることを確認してください
