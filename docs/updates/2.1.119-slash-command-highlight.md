## 原文（日本語に翻訳）

スラッシュコマンドの候補が、クエリにマッチした文字をハイライト表示するようになった

## 原文（英語）

Slash command suggestions now highlight the characters that matched your query

## 概要

スラッシュコマンドを入力する際、候補リストに表示されるコマンド名のうち、入力した文字列にマッチした部分がハイライト（強調表示）されるようになった。どの文字が検索クエリにマッチしているかが視覚的にわかりやすくなり、目的のコマンドを素早く見つけられる。

## 基本的な使い方

スラッシュコマンドの入力時に自動的に適用される。特別な設定は不要。

```
/ag     ← 入力
```

候補リストに以下のように表示される（**ag** 部分がハイライト）：

```
/[ag]ents         - エージェントを管理
/[ag]e            - チャット履歴を確認
```

## 実践例

### あいまい検索でのハイライト

Claude Codeのスラッシュコマンドは前方一致だけでなく部分一致でも候補を表示する。マッチした箇所がハイライトされることで、候補が表示された理由がひと目でわかる：

```
/exp    ← 入力
```

```
/[exp]ort         - 会話をエクスポート
```

### 複数文字のマッチ確認

```
/st     ← 入力
```

```
/[st]atus         - 設定状態を表示
/[st]op           - 実行を停止
```

### 長いコマンド名の検索

```
/reload  ← 入力
```

```
/[reload]-plugins  - プラグインをリロード
```

## 注意点

- ハイライトはあくまで視覚的なヒントであり、コマンドの動作には影響しない
- この機能は自動的に有効になっており、設定変更は不要
- ターミナルのカラー設定によってハイライトの見え方が異なる場合がある
