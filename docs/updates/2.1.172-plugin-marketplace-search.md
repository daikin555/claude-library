## 原文（日本語に翻訳）

`/plugin` でマーケットプレイスのプラグインを閲覧する際に検索バーが追加されました

## 原文（英語）

Added a search bar when browsing a marketplace's plugins in `/plugin`

## 概要

`/plugin` コマンドでプラグインマーケットプレイスを閲覧する際、プラグインを名前やキーワードで検索できる検索バーが追加されました。プラグインの数が増えている中、目的のプラグインを素早く見つけられるようになります。

## 基本的な使い方

```
/plugin
```

マーケットプレイス画面を開いた後、検索バーにキーワードを入力してフィルタリング：

```
/plugin
> [検索バーに "git" と入力]
→ git関連のプラグインのみ表示される
```

## 実践例

### 特定カテゴリのプラグインを検索

```
/plugin
# マーケットプレイス一覧が表示される
# 検索バーに "test" と入力
# テスト関連のプラグインがリアルタイムでフィルタリングされる
```

### 開発ツール系プラグインを探す

```
/plugin
# 検索バーに "linter" や "formatter" などを入力
# 該当するプラグインを素早く発見
```

## 注意点

- 検索はプラグイン名と説明文に対して実行されます
- 検索バーは `/plugin` でマーケットプレイスを開いた後に表示されます
- `Esc` キーで検索をクリアし、全プラグイン一覧に戻れます
