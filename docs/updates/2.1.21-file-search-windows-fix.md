## 原文（日本語訳）
VS Code拡張のWindows環境でファイル検索が動作しない問題を修正しました

## 原文（英語）
Fixed file search not working in VS Code extension on Windows

## 概要
Windows上のVS Code拡張でファイル検索機能が正常に動作しない問題が修正されました。これにより、Windowsユーザーも`@`メンションやファイル選択機能を問題なく使用できます。

## 実践例

### @メンションでのファイル参照

```bash
# VS Code の Claude Code 拡張内で
@src/comp[TAB]
# → ファイル候補が正しく表示される

# 修正前は Windows で候補が表示されなかった
```

### ファイル検索の利用

```bash
# VS Code Claude Code 拡張で
このファイル @README.md の内容を要約してください

# Windows でもファイルが正しく見つかり、読み込まれる
```

## 注意点

- **VS Code専用**: この修正はVS Code拡張に特化したものです
- **パス形式**: Windowsのパス形式（バックスラッシュ）も正しく処理されます
- **再起動不要**: VS Code拡張を最新版に更新すれば自動的に修正が適用されます
