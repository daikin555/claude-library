## 原文（日本語に翻訳）

「bashコマンドはサンドボックス化されます」という起動時のバナーを削除しました。サンドボックスのステータスは引き続き `/status` コマンドおよびコマンドがブロックされた際に表示されます。

## 原文（英語）

Removed the "bash commands will be sandboxed" startup banner — sandbox status still shows in `/status` and when a command is blocked

## 概要

Claude Code の起動時に表示されていた「bashコマンドはサンドボックス化されます」というバナーメッセージが削除されました。これはUIのノイズを減らすための改善で、起動画面がよりすっきりします。サンドボックスが有効かどうかの情報は、`/status` コマンドの実行時やコマンドがブロックされた際に引き続き確認できます。

## 基本的な使い方

サンドボックスのステータスを確認するには、セッション中に `/status` コマンドを実行してください。

```
/status
```

コマンドがサンドボックスによってブロックされた場合は、その時点でも通知が表示されます。

## 注意点

- 起動バナーは削除されましたが、サンドボックス機能自体は引き続き動作しています
- サンドボックスの有効/無効は `/status` で確認可能です
- コマンドがブロックされた場合は、依然として通知が表示されます
- v2.1.157 以降で自動的に適用されます
