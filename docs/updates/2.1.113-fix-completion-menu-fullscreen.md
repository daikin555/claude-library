## 原文（日本語に翻訳）

フルスクリーンモードでスラッシュ/@補完メニューがプロンプト枠に密着して表示されない問題を修正しました。

## 原文（英語）

Fixed slash/@ completion menu not sitting flush against the prompt border in fullscreen mode

## 概要

Claude Codeをフルスクリーンモードで使用している場合、`/` や `@` を入力したときに表示されるオートコンプリートメニューが、プロンプト入力欄の枠に正しく密着して表示されず、ずれて表示される問題がありました。今回の修正でフルスクリーンモードでもメニューが正しい位置に表示されます。

## 基本的な使い方

```
# フルスクリーンモードでの補完メニューの動作

# / を入力すると補完メニューが表示される
/copy
/clear
/review
/compact
...

# @ を入力するとファイルやMCPツールの補完が表示される
@src/components/App.tsx
@package.json
...

# 修正後: これらのメニューがプロンプト枠に密着して表示される
```

## 実践例

### フルスクリーンモードへの切り替え

```
# フルスクリーンモードでClaude Codeを起動する方法

# ターミナルをフルスクリーンにする場合:
# macOS: Cmd+Ctrl+F (ターミナルのフルスクリーン)
# Windows: F11 (Windows Terminalのフルスクリーン)

# または Claude Code 内でウィンドウを最大化
# Ctrl+Z や Cmd+Shift+F などの設定によっては
# Claude Code自体がフルスクリーンUIを提供
```

### スラッシュ補完の使い方

```
# / を入力すると利用可能なコマンドが一覧表示される
# フルスクリーンモードでも正しく表示されるようになった

/c...
┌─────────────────────────────────────────┐  ← プロンプト枠に密着
│ > /clear    会話履歴をクリア              │
│   /compact  コンテキストをコンパクト化    │
│   /copy     テキストをコピー             │
│   /cost     コスト情報を表示             │
└─────────────────────────────────────────┘
```

### @ 補完（ファイル参照）の使い方

```
# @ を入力するとファイル補完が表示される
@src/
┌─────────────────────────────────────────┐  ← プロンプト枠に密着
│ > src/components/                        │
│   src/services/                          │
│   src/utils/                             │
│   src/types.ts                           │
│   src/App.tsx                            │
└─────────────────────────────────────────┘

# 修正前: この補完メニューがずれて表示されていた
# 修正後: プロンプト枠のすぐ上（または下）に正しく表示される
```

### 補完メニューを使った効率的な作業

```
# フルスクリーンモードでの効率的な操作フロー

# 1. ファイルを参照してレビュー依頼
@ → src/components/UserProfile.tsx を選択
「このコンポーネントのパフォーマンスを改善してください」

# 2. コマンドの素早い実行
/ → /compact を選択（コンテキストを整理）

# 3. MCPツールの指定
@ → @github: などのMCPツールを選択

# フルスクリーンでの開発作業がより快適になった
```

## 注意点

- この修正はフルスクリーンモード使用時の**表示位置のずれ**を修正するものです
- 補完メニューの内容や機能自体は変わりません
- 通常の（非フルスクリーン）モードでは以前から正しく表示されていました
- ターミナルの種類（macOS Terminal、iTerm2、Windows Terminal等）によって動作が異なる場合がありましたが、今回の修正で各環境での一貫性が改善されます
