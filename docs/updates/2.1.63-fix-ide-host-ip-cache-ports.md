# IDEホストIPキャッシュがポートをまたいで結果を誤って共有していた問題を修正

## 原文（日本語に翻訳）

IDEホストIPキャッシュが複数のポートにわたって結果を誤って共有していた問題を修正しました。

## 原文（英語）

Fixed IDE host IP detection cache incorrectly sharing results across ports

## 概要

Claude Code v2.1.63で、IDEとの通信に使用するホストIPアドレスのキャッシュが、異なるポートで誤った結果を共有していた問題が修正されました。この問題は複数のIDEインスタンスや異なるポートでClaude Codeを使用する場合に、誤ったIPアドレスが使用される原因となっていました。

## 基本的な使い方

この修正はバックグラウンドで自動的に適用されます。ユーザー側での操作は不要です。

## 実践例

### 影響を受けるシナリオ

- VS CodeとJetBrainsなど複数のIDEで同時にClaude Codeを使用する場合
- 複数の異なるポートでClaude Code IDE統合を使用する場合

### 修正後の改善

- 各ポートで独立したホストIP検出が行われる
- 複数IDEの同時使用時の接続信頼性が向上

## 注意点

- v2.1.63にアップデートすることで自動的に修正が適用されます
- 複数のIDE統合を同時に使用するユーザーに特に効果があります
