## 原文（日本語に翻訳）

Markdown テーブルのセル境界がインラインコードの色を継承する問題、折り返された継続行のスタイルが失われる問題、狭いターミナルのスタックレイアウトで空のヘッダーセルにラベルが表示される問題を修正しました。

## 原文（英語）

Fixed markdown table cell borders inheriting the color of inline code, wrapped continuation lines losing their style, and empty header cells showing a label in the narrow-terminal stacked layout

## 概要

Markdown テーブルに関連する3つの表示バグが修正されました。（1）テーブルのセル境界線がインラインコード（`` ` `` で囲まれたテキスト）の色を誤って継承する問題、（2）長いテキストが折り返された際に継続行のスタイル（色・強調など）が失われる問題、（3）狭いターミナルのスタックレイアウトで空のヘッダーセルに不要なラベルが表示される問題がすべて修正されました。

## 影響を受けていた状況

- Claude の応答にインラインコードを含む Markdown テーブルが出力される場合
- 長いセル内容が折り返される場合
- 狭いターミナルウィンドウや小さいフォントサイズで使用している場合

## 注意点

- v2.1.152 以降では自動的に修正されており、特別な操作は不要です
