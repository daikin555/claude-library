## 原文（日本語に翻訳）

プラグインレジストリが再構築された後、git ブランチを追跡するプラグインが静かに更新を受け取れなくなる問題を修正しました。

## 原文（英語）

Fixed plugins that track a git branch silently no longer receiving updates after the plugin registry was rebuilt

## 概要

git リポジトリのブランチを追跡するように設定されたプラグインが、プラグインレジストリの再構築後に更新を受け取らなくなるという問題が修正されました。この問題は通知なく発生していたため、プラグインが古いバージョンのまま使われ続けることがありました。v2.1.152 以降は、レジストリ再構築後もブランチ追跡プラグインが正常に更新を受け取り続けます。

## 影響を受けていた状況

- `source: git` 形式でブランチを指定してインストールしたプラグインを使用している場合
- プラグインのレジストリ再構築（`claude plugin`関連のコマンド実行後など）が発生した場合

## 注意点

- v2.1.152 以降では自動的に修正されており、特別な操作は不要です
- 以前にこの問題に遭遇した場合は、プラグインを再インストールすることで最新版に更新できます
