# 修正：/loginのコードプロンプトでペーストが動作しないバグ（2.1.105リグレッション）

## 原文（日本語に翻訳）

`/login` のコードプロンプトでペーストが動作しないバグを修正した（2.1.105でのリグレッション）。

## 原文（英語）

Fixed paste not working in the `/login` code prompt (regression in 2.1.105).

## 概要

v2.1.105で発生したリグレッションにより、`/login` コマンド実行時の認証コード入力フォームにクリップボードからコードをペーストできない問題が修正されました。これにより、認証コードをコピーして貼り付けるという標準的なログインフローが正常に動作するようになりました。

## 基本的な使い方

### /loginの正常な使用方法

```
/login

# 認証コードの入力画面が表示される
# ブラウザで表示された認証コードをコピー
# プロンプトにCtrl+V（またはCmd+V）でペースト → 正常に動作するように修正
```

## 注意点

- **v2.1.105〜v2.1.107のバージョンのみ影響**: このバグはv2.1.105で導入され、v2.1.108で修正されました。
- **影響を受けた操作**: `/login` フローの認証コード入力のみが影響を受けていました。通常のプロンプト入力へのペーストは問題ありませんでした。
