## 原文（日本語に翻訳）

折りたたまれたグループ内の思考サマリーが最低3秒間は読めるようになりました。Markdownでレンダリングされ、最大10行に制限されます（`Ctrl+O` で全思考を表示）。

## 原文（英語）

Thinking summaries in the collapsed group now stay readable for at least 3 seconds, render as markdown, and cap at 10 lines (`Ctrl+O` shows the full thinking)

## 概要

Claude が思考（thinking）を使う際に表示される折りたたみグループ内のサマリー表示が改善されました。サマリーは最低3秒間表示されるため素早く読め、Markdown としてレンダリングされるようになりました。また表示は最大10行に制限され、全内容は `Ctrl+O` で確認できます。

## 基本的な使い方

```
# Claude が思考を実行すると、折りたたみグループにサマリーが表示される
# サマリーは最低 3 秒間表示される

# 全思考内容を見たい場合
Ctrl+O  # 全思考を展開表示
```

## 実践例

### 思考サマリーの確認

Claude が複雑な問題を考える際、サマリーで進行状況を確認できます。

```
# 複雑なタスクを依頼
→ Claude の思考グループが表示される
→ サマリーが最低 3 秒間、Markdown 形式で表示される
→ 折りたたまれた後も Ctrl+O で全内容を確認可能
```

### 長い思考への対処

思考が長い場合でも10行でキャップされ、読みやすい表示になります。

```
# 長い思考が発生しても
→ サマリーは最大 10 行で表示
→ 「Ctrl+O で全思考を表示」という案内が表示される
→ Ctrl+O で完全な思考プロセスを確認
```

## 注意点

- 思考機能（extended thinking）が有効な場合にのみ、この表示が現れます
- サマリーは Claude が自動生成したもので、思考の全内容ではありません
- `Ctrl+O` は思考の完全な内容を展開表示します
- この変更はビジュアル表示のみに関するもので、思考の質や動作には影響しません
