# サードパーティプロバイダー利用時の誤解を招く料金表示を削除

## 原文（日本語に翻訳）
サードパーティプロバイダー（Bedrock、Vertex、Foundry）のユーザーに対して、モデル選択画面から誤解を招くAnthropic APIの料金情報を削除しました。

## 原文（英語）
Removed misleading Anthropic API pricing from model selector for third-party provider (Bedrock, Vertex, Foundry) users

## 概要
AWS Bedrock、Google Cloud Vertex AI、またはAnthropic Foundryを経由してClaude Codeを使用している場合、モデル選択画面に表示されるAnthropic API（直接API）の料金情報が誤解を招く可能性がありました。この修正により、サードパーティプロバイダー利用時には不正確な料金情報が表示されなくなりました。

## 基本的な使い方

### Bedrock経由での利用
```bash
# AWS Bedrock設定
export AWS_PROFILE=your-profile
export AWS_REGION=us-east-1

claude --provider bedrock

# モデル選択画面
/model

# 以前: Anthropic APIの料金（$3/MTok入力, $15/MTok出力）が表示
# 現在: 料金情報は表示されない（Bedrock独自の料金体系）
```

### Vertex AI 経由での利用
```bash
# Google Cloud Vertex AI設定
export GOOGLE_APPLICATION_CREDENTIALS=/path/to/credentials.json

claude --provider vertex

# モデル選択
/model

# Vertex AI独自の料金が適用される
# 誤解を招くAnthropic直接APIの料金は表示されない
```

### Foundry 経由での利用
```bash
# Anthropic Foundry（エンタープライズ）
claude --provider foundry

# カスタム料金契約が適用される
# 公開API料金は表示されない
```

## 実践例

### プロバイダーごとの料金確認
各プロバイダーの正しい料金は、それぞれの公式ドキュメントで確認：

```bash
# AWS Bedrock の料金
# https://aws.amazon.com/bedrock/pricing/

# Google Cloud Vertex AI の料金
# https://cloud.google.com/vertex-ai/pricing

# Anthropic Foundry の料金
# 契約によりカスタム（営業担当に問い合わせ）
```

### コスト最適化のためのプロバイダー選択
```bash
# 異なるプロバイダーでコストを比較

# 1. Anthropic API（直接）
claude
# 料金: 標準のAnthropic API料金

# 2. AWS Bedrock
claude --provider bedrock
# 料金: Bedrock料金（リージョンにより異なる）

# 3. Google Cloud Vertex AI
claude --provider vertex
# 料金: Vertex AI料金（リージョンにより異なる）

# モデル選択画面で誤解を招く料金情報は表示されない
```

### エンタープライズ環境での利用
大規模組織でカスタム契約を使用：

```bash
# Foundry経由（エンタープライズ契約）
claude --provider foundry

# モデル選択
/model

# カスタム料金契約が適用される
# 標準API料金は表示されない（混乱を避ける）
```

### プロバイダー間のモデル比較
```bash
# 各プロバイダーで利用可能なモデルを確認

# Bedrock
claude --provider bedrock
/model
# Claude 3.5 Sonnet, Claude 3 Opus など

# Vertex AI
claude --provider vertex
/model
# 同様のモデルラインナップ（Google Cloud統合）

# 料金は各プロバイダーの公式情報を参照
```

## 注意点

- **料金の確認先**:
  - **Anthropic API（直接）**: [https://www.anthropic.com/api/pricing](https://www.anthropic.com/api/pricing)
  - **AWS Bedrock**: [https://aws.amazon.com/bedrock/pricing/](https://aws.amazon.com/bedrock/pricing/)
  - **Google Cloud Vertex AI**: [https://cloud.google.com/vertex-ai/pricing](https://cloud.google.com/vertex-ai/pricing)
  - **Anthropic Foundry**: カスタム契約（営業担当に問い合わせ）

- **料金の違い**: サードパーティプロバイダー経由の料金は、Anthropic直接APIとは異なります。リージョン、ボリューム、契約条件により大きく変動します。

- **プロバイダーの選択**:
  ```bash
  # デフォルト: Anthropic API
  claude

  # AWS Bedrock
  claude --provider bedrock

  # Google Cloud Vertex AI
  claude --provider vertex

  # Anthropic Foundry
  claude --provider foundry
  ```

- **料金計算ツール**: 各プロバイダーは料金計算ツールを提供しています。詳細なコスト見積もりには、それらを使用してください。

- **プロンプトキャッシング**: プロンプトキャッシングの料金もプロバイダーごとに異なります。Bedrockでは別料金、Vertex AIでは含まれる場合があります。

- **リージョン差**: 特にBedrockとVertex AIでは、リージョンによって料金が異なります。最適なリージョンを選択してください。
