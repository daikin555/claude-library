## 原文（日本語に翻訳）

`/release-notes` がインタラクティブなバージョン選択機能になりました

## 原文（英語）

`/release-notes` is now an interactive version picker

## 概要

`/release-notes` コマンドがインタラクティブなバージョン選択UIに刷新されました。これまでは最新のリリースノートのみ表示されていましたが、新しいUIでは過去のバージョンも含めて選択・閲覧できるようになりました。気になるバージョンのリリースノートを直接参照できます。

## 基本的な使い方

```bash
# インタラクティブバージョン選択を起動
/release-notes
```

起動後の操作：
```
バージョンを選択してください:
❯ 2.1.92 （最新）
  2.1.91
  2.1.90
  2.1.89
  ...

↑↓ で選択、Enter で確認、Esc でキャンセル
```

## 実践例

### 特定バージョンの変更内容を確認

```bash
/release-notes
# 矢印キーで目的のバージョンを選択
# 例: 2.1.90 を選択して Enter
# → 2.1.90 のリリースノートが表示される
```

### アップデート後の変更確認

```bash
# Claude Code を更新した後に変更点を確認
npm update -g @anthropic-ai/claude-code

# 新しいバージョンのリリースノートを確認
/release-notes
# 最新バージョンが先頭に表示されている
```

### バージョン間の機能追加を調べる

```bash
# 過去数バージョンを遡って機能変更を確認
/release-notes
# 2.1.85 から 2.1.92 まで順に確認することで
# いつどの機能が追加されたかを把握できる
```

## 注意点

- **バージョン一覧の範囲**: 表示されるバージョンの範囲はインストール済みの履歴または配信サーバーの情報に依存します
- **ナビゲーション**: 矢印キー（↑↓）でバージョンを選択し、Enterで確定、Escでキャンセルします
- **最新版の確認**: バージョン一覧の先頭が常に最新バージョンです
