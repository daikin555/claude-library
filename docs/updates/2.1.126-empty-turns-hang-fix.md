# 思考完了後に出力が表示されないハングを修正

## 原文（日本語に翻訳）

アシスタントが思考を終えても、空のターンの連続後に出力が表示されないハングを修正しました。

## 原文（英語）

Fixed a hang where the assistant could finish thinking but show no output after a run of empty turns

## 概要

特定の条件下でClaude Codeの応答処理が完了しているにもかかわらず、画面に何も表示されないままフリーズする問題が修正されました。空のターン（コンテンツなしの応答）が連続した後に発生していたこのハングにより、ユーザーはClaude Codeを強制終了するしかない状況でした。今回の修正で正常に出力が表示されます。

## 基本的な使い方

```bash
# v2.1.126以前: 特定条件でフリーズすることがあった
claude "複雑なタスクを実行して"
# → 処理完了後も何も表示されずフリーズ...

# v2.1.126以降: 正常に出力が表示される
claude "複雑なタスクを実行して"
# → 処理完了後に正常に結果が表示される
```

## 注意点

- この問題は特定の状況（空のターンが連続する場合）にのみ発生していました
- 修正は自動的に適用されます。設定変更は不要です
- もし依然としてハングが発生する場合は `Ctrl+C` で中断して再試行してください
