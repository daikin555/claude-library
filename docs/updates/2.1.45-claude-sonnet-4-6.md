## 原文（日本語に翻訳）

Claude Sonnet 4.6 のサポートを追加

## 原文（英語）

Added support for Claude Sonnet 4.6

## 概要

Claude Code 2.1.45 において、最新の Claude Sonnet 4.6 モデルがサポートされました。これにより、Claude Code から最新の Sonnet モデルを直接利用できるようになります。Claude Sonnet 4.6 は前バージョンと比較して性能が向上しており、より高精度なコード生成・分析が期待できます。

## 基本的な使い方

Claude Code の設定でモデルを指定することで Sonnet 4.6 を利用できます。

```bash
# 環境変数でモデルを指定
ANTHROPIC_MODEL=claude-sonnet-4-6 claude

# または --model フラグで指定
claude --model claude-sonnet-4-6
```

`settings.json` に永続設定として記述する場合：

```json
{
  "model": "claude-sonnet-4-6"
}
```

## 実践例

### プロジェクトごとにモデルを切り替える

プロジェクトの `.claude/settings.json` に設定することで、そのプロジェクトのみ Sonnet 4.6 を使用できます。

```json
{
  "model": "claude-sonnet-4-6"
}
```

### API キーと合わせて利用する

Anthropic API を直接使う場合は、モデル ID として `claude-sonnet-4-6` を指定します。

```python
import anthropic

client = anthropic.Anthropic()
message = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "このコードをレビューしてください。"}
    ]
)
```

## 注意点

- Sonnet 4.6 は最新モデルのため、旧バージョンの Claude Code では利用できない場合があります。`claude --version` でバージョンを確認し、2.1.45 以上にアップデートしてください。
- モデルの利用可否はサブスクリプションプランや API キーの権限によって異なります。
- `claude-sonnet-4-6` はフルモデル ID です。エイリアス（短縮形）が使える場合もありますが、確実に最新版を指定する場合はフル ID を使用することを推奨します。
