# 修正：/feedbackの送信失敗後にEnterキーで再送信できないバグ

## 原文（日本語に翻訳）

`/feedback` の再試行を修正：送信失敗後にまず説明を編集せずにEnterを押して再送信しようとしても動作しなかった問題を修正した。

## 原文（英語）

Fixed `/feedback` retry: pressing Enter to resubmit after a failure now works without first editing the description.

## 概要

`/feedback` コマンドでフィードバックを送信しようとして失敗した際、内容を編集せずにそのままEnterキーを押して再送信を試みても動作しないバグが修正されました。以前は、再送信するために必ず何らかの編集操作が必要でしたが、修正後は失敗後に直接Enterキーで再送信できるようになりました。

## 基本的な使い方

### /feedbackの使用方法

```
/feedback

# フィードバック入力画面が表示される
# メッセージを入力してEnterで送信

# 送信失敗した場合（ネットワークエラーなど）：
# 修正前: 再度Enterを押しても再送信できず、編集が必要だった
# 修正後: そのままEnterを押すだけで再送信できる
```

## 注意点

- **v2.1.108で修正済み**: アップデート後は自動的に修正が適用されます。
- **再送信の判断**: ネットワークエラーによる一時的な失敗の場合は再試行が有効です。サーバーエラーの場合は `status.claude.com` を確認してください。
