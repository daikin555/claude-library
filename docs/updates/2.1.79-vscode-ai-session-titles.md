# [VSCode] セッションタブにAI生成タイトル：最初のメッセージから自動命名

## 原文（日本語に翻訳）

[VSCode] セッションタブが最初のメッセージに基づいてAIが生成したタイトルを取得するようになりました

## 原文（英語）

[VSCode] Session tabs now get AI-generated titles based on your first message

## 概要

VS Code拡張機能のClaude Codeで、セッションタブが自動的にAI生成タイトルを持つようになりました。セッションを開始すると最初のメッセージを元にAIが適切なタイトルを生成し、タブに表示します。これにより、複数のセッションを同時に管理する際に各セッションの内容を一目で識別できるようになります。

## 基本的な使い方

特別な操作は不要です。VS CodeでClaude Codeセッションを開始し、最初のメッセージを送信すると、タブのタイトルが自動的に更新されます。

例：
- 「このPythonコードのバグを直してください」→ タブタイトル: "Pythonバグ修正"
- 「READMEの書き方を教えて」→ タブタイトル: "README作成ガイド"
- 「テストを追加してほしい」→ タブタイトル: "テスト追加"

## 実践例

### 複数セッションの並行管理

複数のタスクを並行して進める場合：

```
セッション1: 「認証機能のリファクタリングをお願いします」
→ タブ: "認証機能リファクタリング"

セッション2: 「データベーススキーマの設計をレビューして」
→ タブ: "DBスキーマレビュー"

セッション3: 「CIパイプラインの設定を確認してほしい」
→ タブ: "CIパイプライン確認"
```

各タブに意味のあるタイトルが付くことで、どのセッションに切り替えればよいかを即座に判断できます。

### セッション履歴の活用

過去のセッションをサイドバーから参照する際、タイトルを見るだけで内容を把握できます：

```
セッション履歴:
- "ログイン機能バグ修正" (3日前)
- "APIドキュメント生成" (1週間前)
- "パフォーマンス最適化" (2週間前)
```

## 注意点

- タイトルは最初のメッセージを送信した後に生成されるため、送信直後は一瞬「新しいセッション」などのデフォルト名が表示されることがあります
- タイトルはAIが自動生成するため、意図した内容と若干異なる場合があります
- この機能はVS Code拡張機能専用です
- セッションタイトルを手動で変更する機能については、今後のアップデートで対応される可能性があります
