## 原文（日本語に翻訳）

バックグラウンドタスク完了通知が箇条書きでプロアクティブに表示されるよう修正しました

## 原文（英語）

Fixed background task completion notifications to appear proactively with bullet point

## 概要

Claude Code v2.1.0で修正された、バックグラウンドタスク完了通知の表示タイミングと書式改善です。以前は、タスクが完了しても通知が遅れたり、見づらい形式で表示されていました。修正後は、完了直後に箇条書き形式（bullet point）で即座に通知されるようになり、ユーザーがタスク完了をすぐに認識できます。

## 注意点

- Claude Code v2.1.0で実装
- プロアクティブな通知表示（即座に通知）
- 箇条書き形式で見やすく
- タスク完了の認識性向上
