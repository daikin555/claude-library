## 原文（日本語訳）
読み込み・検索の進捗インジケーターを改善し、進行中は「Reading…」、完了時は「Read」と表示するようにしました

## 原文（英語）
Improved read/search progress indicators to show "Reading…" while in progress and "Read" when complete

## 概要
ファイルの読み込みや検索操作の進捗状況が、より明確に表示されるようになりました。進行中は動作を示す表現、完了後は過去形で表示されるため、状態が一目で分かります。

## 基本的な使い方

```bash
claude
> src/ディレクトリの全ファイルを確認してください

# 実行中
Reading 15 files… (進行中を示す)

# 完了後
Read 15 files (完了を示す過去形)
```

## 実践例

### 複数ファイルの読み込み

```bash
claude
> プロジェクトの TypeScript ファイルをすべて確認

# 進行中の表示
Reading… src/components/Button.tsx
Reading… src/components/Input.tsx
Reading… src/utils/helpers.ts

# 完了後の表示
Read 3 files
```

### 検索操作の進捗

```bash
claude
> コードベース全体で "TODO" を検索してください

# 検索中
Searching for "TODO"…

# 検索完了
Searched for "TODO" (14 matches in 8 files)
```

### グループ化された表示

```bash
claude
> プロジェクト全体を分析してください

# 進行中（折りたたまれた表示）
Reading 47 files…

# 完了後（折りたたまれた表示）
Read 47 files
```

## 注意点

- **視覚的フィードバック**: 省略記号（…）により、処理が進行中であることが明確になります
- **過去形での完了表示**: 完了した操作は過去形で表示され、状態の変化が分かりやすくなります
- **一貫性**: 読み込み、検索、その他のツール操作すべてで統一された表示形式が使用されます
- **折りたたみ表示**: 多数のファイルを扱う場合は、進捗が折りたたまれて表示されます
