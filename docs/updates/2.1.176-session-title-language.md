## 原文（日本語に翻訳）

セッションタイトルが会話の言語で生成されるようになりました（特定の言語に固定したい場合は `language` 設定を指定してください）。

## 原文（英語）

Session titles are now generated in the language of your conversation (set the `language` setting to pin a specific language)

## 概要

これまでセッションタイトルは会話の内容にかかわらず英語で生成されていましたが、会話で使われている言語に合わせて生成されるようになりました。日本語で会話していれば日本語のタイトルが付くため、セッション一覧での見分けがつきやすくなります。常に特定の言語でタイトルを生成したい場合は `language` 設定で固定できます。

## 基本的な使い方

特別な設定は不要です。日本語で会話を始めると、セッションタイトルも自動的に日本語で生成されます。

会話の言語に依存せず、常に特定の言語でタイトルを生成したい場合は、設定ファイル（`~/.claude/settings.json` など）に `language` を指定します。

```json
{
  "language": "ja"
}
```

## 実践例

### 日本語で会話してタイトルも日本語にする

日本語でやり取りすれば、追加設定なしでセッションタイトルも日本語になります。

```
> ログイン画面のバリデーションを追加して
```

このようなセッションには「ログイン画面のバリデーション追加」のような日本語タイトルが付きます。

### 言語を固定する

英語と日本語を混在させて会話するプロジェクトでも、タイトルだけは常に日本語にしたい場合は `language` を固定します。

`~/.claude/settings.json`:
```json
{
  "language": "ja"
}
```

### プロジェクト単位で言語を統一する

チームで共有するプロジェクトでは、`.claude/settings.json` に設定をコミットすることで、メンバー全員のセッションタイトルの言語を揃えられます。

`.claude/settings.json`:
```json
{
  "language": "ja"
}
```

## 注意点

- `language` を指定しない場合は、会話で使われている言語が自動的に判定されます。
- 会話の途中で言語が混在している場合、自動判定では意図しない言語になることがあります。確実に固定したい場合は `language` 設定を使ってください。
- すでに生成済みのセッションタイトルが遡って変更されるわけではありません。
