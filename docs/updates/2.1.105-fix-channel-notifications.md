## 原文（日本語に翻訳）

Team/EnterpriseユーザーでFirst messageの後にインバウンドチャンネル通知がサイレントに失われていた問題を修正しました。

## 原文（英語）

Fixed inbound channel notifications being silently dropped after the first message for Team/Enterprise users

## 概要

Claude Code Team/Enterpriseプランのユーザーで、最初のメッセージ後にインバウンドチャンネルからの通知が静かに失われてしまう問題が修正されました。チャンネル経由での継続的なメッセージ受信が正常に機能するようになりました。

## 修正内容

```
Before: 最初の通知は受信 → 以降の通知はサイレントにドロップ
After:  全ての通知を正常に受信
```

## 注意点

- この修正はTeam/Enterpriseプランのユーザーに影響します
- インバウンドチャンネル通知を使用している場合、この問題が解消されました
