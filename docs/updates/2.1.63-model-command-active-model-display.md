# /model コマンドでアクティブなモデルをメニューに表示

## 原文（日本語に翻訳）

`/model` コマンドが改善され、スラッシュコマンドメニューに現在アクティブなモデルが表示されるようになりました。

## 原文（英語）

Improved `/model` command to show the currently active model in the slash command menu

## 概要

Claude Code v2.1.63から、`/model` コマンドのスラッシュコマンドメニューに現在使用中のモデルが表示されるようになりました。これにより、モデルを切り替える前に現在の設定を確認でき、誤ったモデルで作業するリスクを減らせます。

## 基本的な使い方

### /model コマンドの使用

```
/model
```

スラッシュコマンドメニューを開くと、現在アクティブなモデル名が表示されます。選択肢から別のモデルに切り替えることができます。

## 実践例

### 現在のモデルを確認してから切り替える

```
# スラッシュコマンドメニューを開く
/model

# メニューに表示される情報:
# 現在のモデル: claude-sonnet-4-6（アクティブ）
# 選択肢:
# - claude-opus-4-6
# - claude-sonnet-4-6 ✓（現在選択中）
# - claude-haiku-4-5
```

### コスト最適化のためのモデル切り替え

```
# 簡単なタスクにはHaikuを使用
/model
# → claude-haiku-4-5 を選択

# 複雑な作業にはOpusを使用
/model
# → claude-opus-4-6 を選択
```

### APIキーやBedrockの確認

```
# Bedrock/Vertex AI環境でのモデル確認
/model
# 現在アクティブなモデルがメニューに表示される
```

## 注意点

- この改善はUIの表示に関するもので、モデルの切り替え機能自体は変わりません
- アクティブなモデルはメニュー内でハイライトまたはチェックマークなどで識別されます
- `--model` CLIフラグで起動時に指定したモデルも正しく表示されます
