## 原文（日本語に翻訳）

- PowerShellツールが `-ExecutionPolicy Bypass` を渡すようになりました。`CLAUDE_CODE_POWERSHELL_RESPECT_EXECUTION_POLICY=1` でオプトアウトできます
- PowerShellツールが、Bedrock・Vertex・Foundryユーザー向けのWindowsでデフォルト有効になりました。`CLAUDE_CODE_USE_POWERSHELL_TOOL=0` でオプトアウトできます

## 原文（英語）

- PowerShell tool now passes `-ExecutionPolicy Bypass`. Opt out with `CLAUDE_CODE_POWERSHELL_RESPECT_EXECUTION_POLICY=1`
- The PowerShell tool is now enabled by default on Windows for Bedrock, Vertex, and Foundry users. Opt out with `CLAUDE_CODE_USE_POWERSHELL_TOOL=0`

## 概要

Claude CodeのPowerShellツールが `-ExecutionPolicy Bypass` フラグを自動で付与するようになり、スクリプトの実行ポリシーによる実行ブロックを防ぎます。また、AWS Bedrock・Google Vertex AI・Azure Foundryを利用するWindowsユーザー向けに、PowerShellツールがデフォルトで有効化されました。

## 基本的な使い方

### PowerShellツールの自動Bypass

追加の設定なしで、PowerShellスクリプトが実行ポリシーに関係なく動作します：

```powershell
# Claude Code が内部的に実行するコマンドイメージ
powershell.exe -ExecutionPolicy Bypass -Command "..."
```

### 実行ポリシーを尊重させたい場合（オプトアウト）

```bash
# 環境変数で実行ポリシーのBypassを無効化
export CLAUDE_CODE_POWERSHELL_RESPECT_EXECUTION_POLICY=1
```

または `.claude/settings.json` の `env` セクションで設定：

```json
{
  "env": {
    "CLAUDE_CODE_POWERSHELL_RESPECT_EXECUTION_POLICY": "1"
  }
}
```

### PowerShellツール自体を無効化したい場合

```bash
# PowerShellツールを無効化
export CLAUDE_CODE_USE_POWERSHELL_TOOL=0
```

## 実践例

### Bedrock/Vertex/FoundryユーザーのWindows環境

Bedrock、Vertex AI、Azure Foundry経由でClaude Codeを使用しているWindowsユーザーは、追加設定なしでPowerShellツールが使えるようになります：

```bash
# 設定確認（PowerShellツールが有効か）
claude --version
# Windows + Bedrock/Vertex/Foundry 環境では自動でPowerShellツールが有効
```

### セキュリティポリシーが厳格な環境での対応

企業環境など、PowerShell実行ポリシーをセキュリティコントロールとして使用している場合：

```json
// .claude/settings.json - 実行ポリシーを尊重する設定
{
  "env": {
    "CLAUDE_CODE_POWERSHELL_RESPECT_EXECUTION_POLICY": "1"
  }
}
```

## 注意点

- `-ExecutionPolicy Bypass` は現在のPowerShellセッションにのみ適用され、システム設定を恒久的に変更しません
- セキュリティポリシーとして実行ポリシーを使用している環境では、`CLAUDE_CODE_POWERSHELL_RESPECT_EXECUTION_POLICY=1` でオプトアウトすることを検討してください
- Bedrock/Vertex/Foundry以外のWindowsユーザーは、以前と同様に手動でPowerShellツールを有効化する必要があります
- PowerShellツールを完全に無効化したい場合は `CLAUDE_CODE_USE_POWERSHELL_TOOL=0` を設定してください
