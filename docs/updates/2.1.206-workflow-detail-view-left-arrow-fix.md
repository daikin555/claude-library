## 原文（日本語訳）

ワークフロー詳細ビューにおいて、フェーズやエージェントの階層から一段戻る際に左矢印キーが機能していなかった不具合を修正した。

## 原文（英語）

Fixed left arrow not stepping back out of a phase or agent in the workflow detail view

## 概要

`/workflows` から開けるワークフロー詳細ビューでは、上下矢印でフェーズやエージェントを選択し、Enterまたは右矢印で一段深く（フェーズ→エージェント→エージェントの詳細）ドリルダウンし、逆に一段戻るには左矢印またはEscを使う操作体系になっています。しかしこれまでは、フェーズやエージェントの階層に入り込んだ状態で左矢印キーを押しても一段戻る動作が効かない不具合がありました。今回の修正により、左矢印キーでも正しく一段外側の階層へ戻れるようになりました。

## 基本的な使い方

ワークフロー詳細ビューを開くには、セッション内で以下を実行します。

```text
/workflows
```

実行中・完了済みのワークフロー一覧から対象を選択すると、進捗ビュー（詳細ビュー）が開きます。基本操作は次の通りです。

- 上下矢印: フェーズやエージェントを選択
- Enterまたは右矢印: 選択した項目にドリルダウン
- 左矢印またはEsc: 一段戻る
- j/k: エージェント詳細がオーバーフローした際のスクロール
- f: 選択中フェーズのエージェント一覧をステータスでフィルタ

## 実践例

### フェーズ一覧からエージェント詳細に入り、元に戻る

複数フェーズからなるワークフローを実行した後、特定フェーズのエージェントが何をしたかを確認し、また別のフェーズを見たい場合の操作です。

```text
/workflows
→ 対象ワークフローを選択
→ 上下矢印でフェーズを選択、右矢印またはEnterでフェーズ内のエージェント一覧へ
→ 上下矢印でエージェントを選択、右矢印またはEnterでエージェント詳細（プロンプト・ツール呼び出し・結果）へ
→ 左矢印で一段戻ってエージェント一覧に戻る
→ さらに左矢印でフェーズ一覧に戻る
```

修正前はこの最後の「左矢印で戻る」操作が効かず、Escキーを使わざるを得ませんでした。修正後は矢印キーだけで階層を行き来できます。

### キーボードのみで複数エージェントの結果を素早く見比べる

矢印キーだけで完結する操作性が復元されたことで、Escに持ち替えることなく左右矢印のみで階層移動と選択ができるようになり、多数のエージェント結果を連続で見比べる作業がスムーズになります。

```text
右矢印でドリルダウン → 内容確認 → 左矢印で戻る → 上下矢印で次のエージェントを選択 → 右矢印でドリルダウン...
```

## 注意点

- Escキーでも同様に一段戻ることができるため、これまでEscで回避していた場合は引き続きそのままでも問題ありません。
- 本修正はキー操作の挙動に関するものであり、ワークフロー自体の実行結果やデータ内容には影響しません。
- 詳細ビューではエージェント詳細の中身が長い場合にj/kでスクロールする操作もあるため、左矢印（戻る）とj/k（スクロール）を混同しないよう注意してください。
