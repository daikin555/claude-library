## 原文（日本語に翻訳）

- バンドルサイズを約510KB削減

## 原文（英語）

- Reduced bundle size by ~510 KB

## 概要

Claude Codeのバンドルサイズが約510KB削減されました。これにより、インストール時のダウンロード量が減少し、起動時の読み込み速度が向上します。bashコマンド解析のネイティブモジュール化などの最適化により実現されたものです。

## 基本的な使い方

この改善はClaude Codeのアップデートにより自動的に適用されます:

```bash
# 最新版にアップデート
npm update -g @anthropic-ai/claude-code

# インストールサイズが以前より約510KB小さくなる
```

## 実践例

### CI/CDパイプラインでの効果

Claude Codeをci環境で使用している場合、インストール時間の短縮につながります:

```yaml
# GitHub Actionsの例
- name: Install Claude Code
  run: npm install -g @anthropic-ai/claude-code
  # インストールが以前より若干高速に
```

### ディスク容量が限られた環境での利用

コンテナやリモート環境など、ディスク容量に制約がある環境でも軽量に動作します:

```dockerfile
# Dockerfileでの利用例
FROM node:20-slim
RUN npm install -g @anthropic-ai/claude-code
# バンドルサイズの削減により、イメージサイズが抑えられる
```

## 注意点

- バンドルサイズの削減は内部的な最適化によるものであり、機能の削除によるものではありません
- ユーザーが体感できる変化としては、インストール速度と起動速度のわずかな改善です
- npm / yarn / pnpm いずれのパッケージマネージャでも恩恵を受けられます
