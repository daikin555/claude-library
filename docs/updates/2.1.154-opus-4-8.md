## 原文（英語）

- Opus 4.8 is here! Now defaults to high effort · /effort xhigh for your hardest tasks
- Fast mode on Opus 4.8 is now available at a fraction of its previous cost: 2x the standard rate for 2.5x the speed
- Deprecated `CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE` (will be removed on 06/01). To use fast mode on Opus 4.6, switch with `/model claude-opus-4-6[1m]` and then `/fast on`

## 概要

Claude Code に新モデル **Opus 4.8** が追加されました。デフォルトでハイエフォート（high effort）動作となり、複雑なタスクに対してより深い推論を行います。さらに、Opus 4.8 の Fast モードが大幅にコスト最適化され、標準の2倍のコストで2.5倍の速度を実現します。Opus 4.6 向けの Fast モード環境変数は非推奨となり、2026年6月1日に削除される予定です。

## 基本的な使い方

```bash
# Opus 4.8 モデルに切り替える
/model claude-opus-4-8

# エフォートレベルを指定する（デフォルトは high）
/effort high      # 高エフォート（デフォルト）
/effort xhigh     # 超高エフォート：最も難しいタスク向け
/effort low       # 低エフォート：軽量・高速なタスク向け

# Opus 4.8 で Fast モードをオンにする
/fast on
```

## 実践例

### 複雑なリファクタリングに xhigh エフォートを使う

大規模なコードベースの全体的なアーキテクチャ改善など、最も難しいタスクには `/effort xhigh` を指定します。

```bash
# モデルを Opus 4.8 に切り替え
/model claude-opus-4-8

# 超高エフォートに設定
/effort xhigh

# 複雑なタスクを依頼
大規模なモノリスを複数のマイクロサービスに分割する設計と実装をしてください。
```

### Fast モードでコストと速度を最適化する

Opus 4.8 の Fast モードは標準の2倍のコストながら、2.5倍の速度で動作します。速度優先のタスクに有効です。

```bash
# Opus 4.8 で Fast モードを有効化
/model claude-opus-4-8
/fast on

# 繰り返しのレビューやドラフト作成など、速度が重要なタスクを実行
コードレビューのフィードバックを素早くまとめてください。
```

### Opus 4.6 で Fast モードを使う（移行方法）

`CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE` 環境変数は非推奨です。代わりに以下の方法を使用してください。

```bash
# 旧来の方法（非推奨・06/01削除予定）
# export CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE=true  ← 使用しないこと

# 新しい方法：モデルを明示的に指定してから /fast on
/model claude-opus-4-6[1m]
/fast on
```

### エフォートレベルの使い分け

タスクの難易度に応じてエフォートを調整することで、コストと品質のバランスを取れます。

```bash
# 簡単な質問や素早い確認 → low エフォート
/effort low
ファイルの最終更新日を確認してください。

# 通常の開発作業 → high エフォート（デフォルト）
/effort high
このバグを修正してテストを追加してください。

# 最難関タスク（大規模設計・難解なデバッグ）→ xhigh エフォート
/effort xhigh
このパフォーマンス問題の根本原因を特定し、包括的な解決策を提案してください。
```

## 注意点

- Opus 4.8 はデフォルトで **high** エフォートで動作します。以前のモデルより思考時間が長くなることがあります
- **xhigh** エフォートはトークン消費が大幅に増加するため、費用対効果を考慮した上で使用してください
- Opus 4.8 の Fast モードは「標準レートの2倍のコスト・2.5倍の速度」です。コスト削減が目的の場合は Haiku や Sonnet モデルの使用も検討してください
- `CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE` 環境変数は **2026年6月1日**に削除されます。早めに新しい方法（`/model claude-opus-4-6[1m]` + `/fast on`）へ移行してください
- Fast モードは応答速度を優先するため、通常モードより推論の深さが若干低下する場合があります
