## 原文（日本語に翻訳）

- Auto モードで「claude-opus-4-7 is temporarily unavailable」と表示される問題を修正しました

## 原文（英語）

- Fixed "claude-opus-4-7 is temporarily unavailable" for auto mode

## 概要

v2.1.111 で導入された Auto モード（Max サブスクライバー向け Opus 4.7 対応）において、「claude-opus-4-7 is temporarily unavailable（claude-opus-4-7 は一時的に利用できません）」というエラーが誤って表示される問題が修正されました。実際には利用可能であっても誤ってエラーが出るバグだったため、v2.1.112 で修正されています。

## 基本的な使い方

### 修正後の動作確認

```bash
# v2.1.112 以降にアップデートしてから実行
claude update

# Auto モードが正常に動作することを確認
claude --model claude-opus-4-7 "テストメッセージ"
```

## 注意点

- このバグが発生していた場合は、`claude update` で v2.1.112 以降にアップデートしてください
- アップデート後もエラーが続く場合は、API キーや Max サブスクリプションの状態を確認してください
- 本当に一時的な障害が発生している場合は [status.anthropic.com](https://status.anthropic.com) を確認してください
