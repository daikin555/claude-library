## 原文（日本語に翻訳）

履歴の書き込みが競合または失敗した際に、プロンプト履歴のエントリが失われたり重複したりする問題を修正しました

## 原文（英語）

Fixed prompt history entries being dropped or duplicated when history writes raced or failed

## 概要

上矢印キーで呼び出せるプロンプト履歴（過去に入力したプロンプトの履歴）は、複数の履歴書き込みが同時に発生（競合）したり、書き込みが途中で失敗したりすると、エントリが失われたり重複して表示されたりすることがありました。この修正により、書き込み処理が安全に行われるようになり、履歴が正確に保たれます。

## 注意点

- Claude Code v2.1.218で修正されました
- 複数セッションを並行して利用している場合や、頻繁にプロンプトを入力する使い方で発生しやすかった問題です
- ユーザー側での追加設定は不要で、自動的に改善が適用されます
