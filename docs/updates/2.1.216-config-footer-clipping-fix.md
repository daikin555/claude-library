## 原文（日本語訳）

フルスクリーンモードにおいて、`/config` の設定リストがキーボードヒントのフッターを見切れさせてしまう不具合を修正しました。

## 原文（英語）

Fixed the `/config` settings list in fullscreen mode clipping its keyboard-hint footer

## 概要

`/config` コマンドはテーマやエディタモード、自動スクロールなど各種設定を一覧・変更できる画面を開きます。[フルスクリーンレンダリングモード](https://code.claude.com/docs/en/fullscreen)では、この設定リストの項目数が多い場合や端末の高さが限られている場合に、画面下部に表示されるべきキーボードヒント（例: 選択・決定・閉じるためのキー案内）のフッター行が見切れて見えなくなることがありました。v2.1.216ではこのクリッピング（表示の切れ）が修正され、フッターのヒントが常に正しく表示されるようになりました。

## 基本的な使い方

見た目の修正であり、特別な設定は不要です。Claude Codeをv2.1.216以降にアップデートし、フルスクリーンモードで `/config` を開くだけで改善された表示を確認できます。

```
/tui fullscreen
/config
```

## 実践例

### 設定リストの表示を確認する

`/config` を開き、フッターに操作ヒント（矢印キーでの移動、Enterでの選択、Escでの終了など）が最後まで表示されているかを確認します。

```
/config
```

### 端末を低い高さにしてフッターの見切れがないか確認

ターミナルウィンドウの高さを小さくした状態で `/config` を開くと、以前は項目数が多いときにフッターが隠れてしまうことがありましたが、修正後は正しく表示されます。

```
# ターミナルの高さを小さくしてから
/config
```

### Vimモードやテーマなど個別設定へのアクセス

`/config` からエディタモードやテーマなどの項目にアクセスする操作自体は変更されていないため、通常通り矢印キーとEnterで操作できます。

```
/theme
```

## 注意点

- フルスクリーンレンダリングはリサーチプレビュー機能であるため、今後も表示関連の改善が継続的に行われる可能性があります
- クラシックレンダラー（非フルスクリーンモード）を使用している場合は元々この不具合の影響を受けません
- 表示上の問題を見つけた場合は `/feedback` またはGitHub Issueで報告してください
