## 原文（日本語に翻訳）

4月1日のために `/buddy` が登場 — コーディング中を見守る小さな生き物を孵化させます

## 原文（英語）

`/buddy` is here for April 1st — hatch a small creature that watches you code

## 概要

v2.1.89に4月1日限定（？）のイースターエッグとして `/buddy` コマンドが追加されました。このコマンドを実行すると、コーディング中を見守る小さな生き物を「孵化」させることができます。作業の合間の小さな楽しみとして、ぜひ試してみてください。

## 基本的な使い方

```bash
# buddyを孵化させる
/buddy

# Claude Codeのインタラクティブセッション内で実行
claude
> /buddy
```

## 実践例

### buddyと一緒にコーディング

```bash
# 1. Claude Codeを起動
claude

# 2. buddyを孵化させる
> /buddy
# 🥚 ...
# 🐣 Buddy が生まれました！

# 3. buddyが見守る中でコーディング開始
> src/app.ts のバグを修正してください
# 🐾 Buddy: 頑張ってるね！
```

## 注意点

- `/buddy` は4月1日の遊び心ある機能です。実際の挙動は試してみてのお楽しみ！
- 4月1日以降も使用できるかどうかは不明です
- 真剣なコーディングセッションの前に精神的な安らぎとしてどうぞ
