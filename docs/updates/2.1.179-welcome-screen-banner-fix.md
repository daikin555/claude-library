## 原文（日本語に翻訳）

ウェルカム画面に複数のプロモーショナルバナーが積み重なる問題を修正 — 1セッションにつき最大1つのプロモが表示されるようになりました

## 原文（英語）

Fixed the welcome screen stacking multiple promotional banners — at most one promo now shows per session

## 概要

Claude Codeの起動時に表示されるウェルカム画面で、複数のプロモーショナルバナーが重なって表示されるUIの問題がありました。この修正により、1セッションにつき最大1つのプロモーションのみが表示されるようになり、画面が煩雑にならずすっきりした表示になりました。

## 基本的な使い方

この修正は自動的に適用されます。Claude Codeを起動した際のウェルカム画面がより整理された状態で表示されます。

```bash
# Claude Code 起動
claude

# ウェルカム画面表示
# 修正後：プロモバナーは最大1つのみ表示
```

## 実践例

### 通常の起動フロー

```
# Claude Code 起動時のウェルカム画面（修正後）
╭─────────────────────────────────╮
│  Claude Code へようこそ          │
│                                  │
│  [プロモーション 1つのみ表示]    │
│                                  │
│  何でも聞いてください...        │
╰─────────────────────────────────╯

# 修正前（問題のある状態）：
# [プロモーション A]
# [プロモーション B]  ← 重複表示されていた
# [プロモーション C]  ← これも重複
```

### 新機能のお知らせ表示

プロモーションバナーは新機能の紹介やお知らせを表示するために使用されます。修正後は最も重要な1つだけが表示されるため、情報が埋もれにくくなりました。

## 注意点

- この修正はUIの表示のみに関係し、機能への影響はありません
- プロモーション表示はセッションごとにリセットされます
- バナーを閉じた場合、同じセッション中は再表示されません
