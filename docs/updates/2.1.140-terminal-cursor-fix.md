## 原文（日本語に翻訳）

ターミナルがフォーカスを失った際にネイティブターミナルのカーソルが入力キャレットに留まらない問題を修正

## 原文（英語）

Fixed native terminal cursor not staying at the input caret when the terminal loses focus

## 概要

ターミナルウィンドウがフォーカスを失った際（別のウィンドウに切り替えるなど）、ターミナルカーソルが入力キャレット（テキスト入力位置）から離れてしまう問題が修正されました。これにより、フォーカス切り替え後にターミナルに戻った際の視覚的な混乱が解消されます。

## 注意点

- この修正はClaude Codeの入力UIの視覚的な安定性を向上させるものです
- フォーカスを失った際に入力キャレットとカーソル位置が同期するようになりました
- 特にマルチウィンドウ環境や頻繁にウィンドウを切り替える作業で恩恵があります
