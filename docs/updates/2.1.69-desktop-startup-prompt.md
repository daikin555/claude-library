## 原文（日本語に翻訳）

macOSとWindowsでClaude Code Desktopを提案する一回限りの起動時プロンプトを追加（最大3回表示、非表示にできます）

## 原文（英語）

Added a one-time startup prompt suggesting Claude Code Desktop on macOS and Windows (max 3 showings, dismissible)

## 概要

macOSまたはWindowsでターミナル版Claude Codeを使用しているユーザーに対し、起動時にClaude Code Desktopアプリへの移行を提案するプロンプトが表示されるようになりました。このプロンプトは最大3回まで表示され、その後は自動的に非表示になります。デスクトップアプリを使うと、ビジュアルなコードdiff表示やGUI操作など、より快適な体験が得られます。

## プロンプトの動作

- **表示回数**: 最大3回（3回見た後は自動的に表示されなくなります）
- **非表示方法**: プロンプトで「Don't show again」を選択
- **対象プラットフォーム**: macOSとWindowsのみ（Linux環境では表示されません）

## Claude Code Desktopの利点

ターミナル版と比較したデスクトップ版の主な利点：

- ビジュアルなコードdiff（変更箇所がGUIで確認できる）
- `cmd+V` / `ctrl+V` で画像をそのまま貼り付け可能
- ターミナル設定不要のセットアップ体験
- `Teleport` 機能でモバイルやWebブラウザのセッションとの連携が容易
- ネイティブアプリとしての快適なパフォーマンス

## プロンプトへの対応方法

起動時にプロンプトが表示されたら：

1. **「Open Desktop App」** - Claude Code Desktopをダウンロード・起動
2. **「Skip」** - 今回はスキップ（次回起動時に再度表示）
3. **「Don't show again」** - 以後表示しない

## ターミナル版の継続使用

デスクトップ版への移行を希望しない場合：

```bash
# 通常通りターミナルで使用し続けられます
claude
```

「Don't show again」を選択するか、3回スキップすればプロンプトは表示されなくなります。

## 注意点

- このプロンプトはLinux環境では表示されません
- デスクトップ版とターミナル版は同じClaude Codeエンジンを使用しており、機能差はほとんどありません
- 既にデスクトップ版を使用しているユーザーには表示されません
