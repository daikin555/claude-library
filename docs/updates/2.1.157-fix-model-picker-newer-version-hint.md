## 原文（日本語に翻訳）

選択されたモデルがすでにそのファミリーの最新バージョンである場合に `/model` ピッカーが誤った「新しいバージョンが利用可能」ヒントを表示する問題を修正しました。また、固定モデルの行に生のIDではなくモデルの説明が表示されるようになりました。

## 原文（英語）

Fixed the `/model` picker showing an incorrect "Newer version available" hint when the selected model is already the newest in its family; the pinned-model row now shows the model's description instead of its raw ID

## 概要

`/model` コマンドでモデルを選択する際、すでに最新モデルを選択しているにもかかわらず「新しいバージョンが利用可能です」というヒントが誤って表示されるバグがありました。あわせて、固定されたモデルの表示行にモデルの内部IDではなく、人間が読みやすい説明文が表示されるよう改善されました。このバグはv2.1.157で修正されました。

## 影響を受けていた状況

- `/model` コマンドを使ってモデルを選択した際に、最新モデルを選択しているにもかかわらず更新通知が表示されていた場合
- 固定モデルの行に `claude-opus-4-5-20251101` のような生のモデルIDが表示されて分かりにくかった場合
- モデルのファミリー内で最新バージョンを明示的に指定して使用していた場合

## 注意点

- v2.1.157 以降では自動的に修正されており、特別な操作は不要です
