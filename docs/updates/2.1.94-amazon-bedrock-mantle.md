## 原文（日本語に翻訳）

Amazon Bedrock powered by Mantleのサポートを追加。`CLAUDE_CODE_USE_MANTLE=1`を設定することで有効化できます。

## 原文（英語）

Added support for Amazon Bedrock powered by Mantle, set `CLAUDE_CODE_USE_MANTLE=1`

## 概要

Claude CodeがAmazon Bedrock上で動作するMantleインフラストラクチャをサポートしました。MantleはAWSが提供する高度な推論基盤で、環境変数`CLAUDE_CODE_USE_MANTLE=1`を設定するだけで切り替えられます。AWS環境でClaudeを利用しているチームや企業にとって、より柔軟なインフラ選択肢が増えることになります。

## 基本的な使い方

環境変数を設定してClaudeを起動します：

```bash
export CLAUDE_CODE_USE_MANTLE=1
claude
```

または、単一コマンドとして実行：

```bash
CLAUDE_CODE_USE_MANTLE=1 claude
```

## 実践例

### AWS環境での設定

シェルの設定ファイル（`.bashrc` / `.zshrc`）に追加して常時有効化：

```bash
# ~/.bashrc または ~/.zshrc
export CLAUDE_CODE_USE_MANTLE=1
export AWS_REGION=us-east-1  # 使用するリージョン
```

### CI/CDパイプラインでの利用

GitHub ActionsなどのCIでMantle経由のBedrockを使用：

```yaml
- name: Run Claude Code
  env:
    CLAUDE_CODE_USE_MANTLE: "1"
    AWS_REGION: us-east-1
  run: claude --print "コードをレビューしてください"
```

### settings.jsonでの設定

プロジェクトの`settings.json`から環境変数として設定することも可能：

```json
{
  "env": {
    "CLAUDE_CODE_USE_MANTLE": "1"
  }
}
```

## 注意点

- Bedrock経由でClaudeを使用している場合にのみ有効な設定です
- Mantleが利用可能なAWSリージョンかどうか事前に確認してください
- 通常のBedrockアクセス権限に加え、Mantleへのアクセス権限が必要な場合があります
- 既存のBedrock設定（`AWS_REGION`、IAMロールなど）はそのまま引き継がれます
