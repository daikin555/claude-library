## 原文（日本語に翻訳）

プラグインプロセスが更新中にファイルを開き続けている際、マーケットプレースの自動更新が公式マーケットプレースを壊れた状態にしてしまう問題を修正しました。

## 原文（英語）

Fixed marketplace auto-update leaving the official marketplace in a broken state when a plugin process holds files open during the update

## 概要

マーケットプレースの自動更新中に、実行中のプラグインプロセスがファイルを開いたままにしていると、更新が不完全な状態で終了し、公式マーケットプレースが壊れた状態になってしまう問題が修正されました。

## 修正内容

プラグインプロセスがファイルを保持している場合でも、マーケットプレースの自動更新が安全に完了するようになりました。

```
Before: 自動更新中 + プラグインがファイルを保持 → マーケットプレースが壊れた状態になる
After:  自動更新中 + プラグインがファイルを保持 → 安全に更新が完了
```
