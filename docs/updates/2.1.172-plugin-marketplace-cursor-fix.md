## 原文（日本語に翻訳）

`/plugin` マーケットプレイスリストで長いプラグインリストから戻った後にカーソルが消える問題と、プラグインブラウザからEscキーを押すと間違ったタブに戻る問題を修正しました

## 原文（英語）

Fixed the `/plugin` marketplace list losing its cursor after backing out of a long plugin list, and Esc from the plugin browser returning to the wrong tab

## 概要

`/plugin` コマンドのマーケットプレイスブラウザにおいて2つのナビゲーション問題が修正されました。1つ目は、長いプラグインリストを閲覧して戻ったときにカーソル（選択位置）が消えてしまう問題、2つ目はプラグインブラウザで `Esc` キーを押して戻ると、元々いたタブではない別のタブに遷移してしまう問題です。

## 基本的な使い方

```
/plugin
# マーケットプレイスを開く

# 長いプラグインリストをスクロールして戻る
# 修正後: カーソル位置が保持される

# Escキーを押す
# 修正後: 元々いたタブ（プラグインリスト等）に正しく戻る
```

## 実践例

### プラグインブラウジングの一般的な操作

```
/plugin
↓ マーケットプレイスを開く
→ プラグイン一覧が表示される
↓ スクロールして様々なプラグインを閲覧
← バックする（Backspace/←）
# 修正後: カーソルが最後に見ていた位置に戻る

→ あるプラグインの詳細を開く
← Escで戻る
# 修正後: プラグインブラウザのリスト画面に正しく戻る
```

## 注意点

- この修正はナビゲーションのUX改善です
- 関連修正として、同バージョンで `/plugin` マーケットプレイスに検索バーも追加されました
