## 原文（日本語に翻訳）

圧縮スタートアップモードでサンドボックス有効化の警告が表示されない問題を修正しました。すべてのレイアウトで表示されるようになりました。

## 原文（英語）

Fixed the sandbox-enabled warning not appearing in condensed startup mode — it now shows in every layout

## 概要

サンドボックスが有効な状態で Claude Code を起動した際に表示されるべき警告メッセージが、圧縮スタートアップモード（condensed startup mode）では表示されない問題が修正されました。v2.1.152 以降は、スタートアップモードの種類に関わらず、すべてのレイアウトで警告が表示されます。

## 影響を受けていた状況

- サンドボックス機能が有効な環境で Claude Code を起動した場合
- コンパクト表示や特定のターミナルサイズで起動した場合

## 注意点

- v2.1.152 以降では自動的に修正されており、特別な操作は不要です
- サンドボックスが有効な場合は、起動時に必ず警告が表示されるようになりました
