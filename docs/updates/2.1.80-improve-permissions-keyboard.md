## 原文（日本語に翻訳）

`/permissions` を改善しました — リスト内からもTabキーと矢印キーでタブを切り替えられるようになりました

## 原文（英語）

Improved `/permissions` — Tab and arrow keys now switch tabs from within a list

## 概要

`/permissions` コマンドのパーミッション管理画面のキーボードナビゲーションが改善されました。以前はリスト内の項目にフォーカスがある状態ではTabキーや矢印キーでタブを切り替えることができませんでしたが、今回の改善によりリスト内からでもTabキーと矢印キーでタブを直接切り替えられるようになりました。キーボードのみでパーミッション管理をスムーズに行えるようになります。

## 基本的な使い方

`/permissions` 画面でリストを操作中でも、TabキーまたはTab/矢印キーでタブを切り替えられるようになりました。

```
# パーミッション管理画面を開く
/permissions

# リスト内の項目を選択中でもタブ切り替えが可能
↑↓     : リスト内の項目を移動
Tab     : タブを切り替え（リスト内からでも可能）
← →    : タブを切り替え（リスト内からでも可能）
Enter   : 選択した項目を操作
```

## 実践例

### キーボードのみでのパーミッション管理

```
# /permissions を実行
/permissions

# タブ1: すべてのパーミッション一覧
↓ ↓    # リスト内の項目を下に移動
Tab     # リスト内にいてもタブを切り替え可能（改善後）

# タブ2: 許可済みパーミッション
↓      # 許可済みリストを移動
→      # 次のタブへ（改善後: リスト内からタブ切り替え）

# タブ3: ブロック済みパーミッション
# キーボードのみで全タブを操作できる
```

### 特定ツールのパーミッションを素早く確認・変更

```
# Bashツールのパーミッションを変更する手順（キーボードのみ）
/permissions         # 画面を開く
/bash <Enter>        # 検索（対応している場合）
→                   # 「許可済み」タブに移動
↑↓                  # Bashのルールを選択
Enter               # パーミッションを変更
```

### パーミッション設定の全体確認

```
# すべてのタブを順番にキーボードで確認
/permissions

Tab → Tab → Tab  # 各タブを順番に切り替えて全体を確認
# 各タブで ↑↓ キーでリストを確認しながら Tab でタブを切り替え

# 修正前: リスト内にいるとタブ切り替えができず、一度リストのフォーカスを外す必要があった
# 改善後: リスト内からシームレスにタブを切り替えられる
```

## 注意点

- **関連する修正**: v2.1.79で `/permissions` の矢印キーナビゲーションの基本修正が行われ、v2.1.80でリスト内からのタブ切り替えが追加されました
- **Tabキーの二重機能**: リスト内でのTabキーはタブ切り替えに使われます。リスト項目間の移動は上下矢印キーを使用してください
- **フォーカスの状態**: タブが切り替わるとフォーカスは新しいタブの最初のリスト項目に移動します
- **マウスとの併用**: キーボードとマウスを組み合わせて使用することも引き続き可能です
