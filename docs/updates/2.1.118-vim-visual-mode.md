## 原文（日本語に翻訳）

Vimビジュアルモード（`v`）とビジュアルラインモード（`V`）を追加。選択、オペレータ、ビジュアルフィードバックに対応。

## 原文（英語）

Added vim visual mode (`v`) and visual-line mode (`V`) with selection, operators, and visual feedback

## 概要

Claude Codeのvimモードにビジュアルモードが追加され、`v`で文字単位の選択、`V`で行単位の選択が可能になりました。選択範囲に対してyank（コピー）、delete（削除）、changeなどのオペレータを直接適用できます。ビジュアルフィードバックにより選択範囲が視覚的に確認できるようになっています。

## 基本的な使い方

vimモードが有効な状態でノーマルモードにいるとき、以下のキーでビジュアルモードに入れます。

```
v   → ビジュアルモード（文字単位選択）
V   → ビジュアルラインモード（行単位選択）
```

選択中はモーションキーで選択範囲を拡張できます：

```
v → モーション（h/j/k/l/w/b/e/$など）→ オペレータ
```

## 実践例

### テキストの選択とコピー

```
# ビジュアルモードで単語を選択してヤンク
v → w → y     # 次の単語を選択してコピー

# ビジュアルモードで行末まで選択してヤンク  
v → $ → y     # カーソルから行末までコピー
```

### 行単位の選択と削除

```
# ビジュアルラインモードで複数行を選択して削除
V → j → j → d    # 3行を選択して削除
```

### 選択範囲の変更

```
# ビジュアルモードで単語を選択して置換（change）
v → e → c    # 単語末尾まで選択して変更モードへ
```

### 選択範囲のヤンクとペースト

```
# 行を選択してコピー→別の場所にペースト
V → y → j → p    # 行をコピーして次の行の後にペースト
```

## 注意点

- ブロックビジュアルモード（`Ctrl+V`）は現在サポートされていません
- ビジュアルモードを抜けるには `Esc` キーを使用します
- 選択範囲はハイライト表示され、視覚的に確認できます
- vimモードはClaude Codeのインタラクティブモードでのみ利用可能です
