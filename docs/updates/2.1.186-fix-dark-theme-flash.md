# ライトターミナルでclaude agentsからバックグラウンドセッションを開く際のダークテーマの一瞬の点滅を修正

## 原文（日本語に翻訳）
ライトターミナルで `claude agents` からバックグラウンドセッションを開く際に、ダークテーマが一瞬点滅する問題を修正しました。

## 原文（英語）
Fixed a dark-theme flash when opening a background session from `claude agents` on a light terminal

## 概要
ライトカラーのターミナルテーマを使用している環境で、`claude agents` からバックグラウンドセッションを開いた際に、ダークテーマが一瞬（フラッシュ）表示される視覚的な問題がありました。この修正により、テーマの切り替え時に画面がちらつく問題が解消されました。

## 基本的な使い方

この修正は自動的に適用されます。ライトテーマのターミナルユーザーに特に効果があります。

### 影響を受けていたシナリオ
- ライトカラーのターミナル（白背景など）を使用
- `claude agents` でバックグラウンドエージェントを選択
- セッションを開く際にダークテーマが一瞬点滅

### 修正後
セッション切り替え時に画面がちらつくことなく、現在のターミナルテーマが維持されます。

## 注意点
- UIの表示（レンダリング）に関するバグ修正です
- ライトカラーターミナルのユーザーに特に影響していた問題です
- 機能的な動作への影響はありません

## 関連情報
- バックグラウンドエージェント管理: `claude agents`
- Claude Code のテーマ設定: `~/.claude/settings.json`
