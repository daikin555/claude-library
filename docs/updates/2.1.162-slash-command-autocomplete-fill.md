# スラッシュコマンドのオートコンプリート：クリックで入力欄に挿入、Enterで実行

## 原文（日本語に翻訳）

オートコンプリートメニューでスラッシュコマンドをクリックすると、すぐに実行される代わりにプロンプト入力欄に挿入されるようになりました。実行するには Enter キーを押してください。

## 原文（英語）

Clicking a slash command in the autocomplete menu now fills it into your prompt instead of running it immediately; press Enter to run

## 概要

これまで、スラッシュコマンドのオートコンプリートメニューでコマンドをクリックすると、即座に実行されていました。v2.1.162 から、クリックするとコマンドが入力欄に挿入されるだけとなり、Enter キーを押すことで初めて実行されます。これにより、引数を追加したり、コマンドを確認してから実行できるようになります。

## 基本的な使い方

```
# 1. "/" を入力してオートコンプリートメニューを表示
/

# 2. メニューからコマンドをクリック
# → コマンドが入力欄に挿入される（例: /review）

# 3. 必要に応じて引数を追加
/review --fix

# 4. Enter キーで実行
```

## 実践例

### 引数付きコマンドの使い方

```
# オートコンプリートで /effort を選択すると入力欄に挿入される
/effort
# → その後 "high" を追加して /effort high として実行可能
```

### コマンドの確認から実行まで

```
# /clear を選択して内容を確認
/clear
# Enter を押す前に、本当にクリアするか確認できる
```

### コマンドを組み合わせた利用

```
# /compact を挿入後、引数を指定
/compact
# → /compact 50000 のようにトークン数を追加してから実行
```

## 注意点

- 従来の動作（クリックで即実行）から変更されているため、慣れが必要な場合があります
- Enter キーを押すまでコマンドは実行されないため、誤操作を防ぎやすくなっています
- キーボードでオートコンプリートを選択（Tab/矢印キー）した場合も同様に、Enter で実行されます
- 引数が必要なコマンド（例: `/effort <level>`）では、挿入後に引数を入力することを忘れないようにしましょう
