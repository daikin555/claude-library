# 従量課金制のEnterprise・Teamサブスクライバーにセッションコストが表示されない問題を修正

## 原文（日本語に翻訳）
従量課金制のEnterpriseおよびTeamサブスクライバーにセッションコストが表示されない問題を修正しました。

## 原文（英語）
Fixed session cost not showing for usage-based Enterprise and Team subscribers

## 概要
従量課金制（usage-based）のEnterpriseまたはTeamプランを使用しているユーザーに対して、セッションのコスト情報が表示されない問題がありました。この修正により、従量課金制プランのユーザーもセッションごとのコストをリアルタイムで確認できるようになりました。

## 基本的な使い方

この修正は自動的に適用されます。従量課金制のEnterprise・Teamサブスクライバーはセッションコストが表示されるようになります。

### セッションコストの確認
セッション中の右下やステータスバーに表示されるコスト情報を確認できます。また：

```
/cost
```

または`/stats` コマンドでセッションのコスト詳細を確認できます。

## 実践例

### コスト管理の活用
```
/cost
# → セッションのAPIコストを表示（修正後は従量課金Enterpriseでも表示）
```

### コスト意識的な利用
セッションコストを確認しながら：
- 長時間のエージェントセッションのコストを把握
- チームでのAPI使用量を管理
- コスト最適化のための参考データとして活用

## 注意点
- 従量課金制（usage-based）のEnterprise・Teamプランに適用されます
- 定額制プランでは以前から表示されていました
- コスト情報はリアルタイムで更新されます
- 表示されるコストは概算です。正確な請求は請求ダッシュボードで確認してください
