# Cursor・VS Code 1.92〜1.104の統合ターミナルで過度に速いトラックパッドスクロールを修正

## 原文（日本語に翻訳）

CursorおよびVS Code 1.92〜1.104の統合ターミナルで発生していた過度に速いトラックパッドスクロールを修正しました。

## 原文（英語）

Fixed overly fast trackpad scrolling in Cursor and VS Code 1.92–1.104 integrated terminals

## 概要

CursorエディタおよびVS Code バージョン1.92から1.104の統合ターミナルでClaude Codeを使用する際、トラックパッドでのスクロールが異常に速く操作しにくい問題が修正されました。特にMacのトラックパッドを使用しているユーザーにとって、長い出力のスクロール操作が改善されます。

## 基本的な使い方

この修正は自動的に適用されます。Cursor または VS Code 1.92〜1.104の統合ターミナルをアップデートするだけで問題が解消されます：

```bash
# Claude Code v2.1.126にアップデート後、VS Code/Cursor を再起動
# → トラックパッドのスクロール速度が正常になる
```

## 実践例

### 影響を受けるバージョンの確認

```bash
# VS Codeのバージョン確認
code --version
# 1.92.0 〜 1.104.x の場合、この修正が適用される

# Cursorも同様に確認
cursor --version
```

## 注意点

- この修正はVS Code / Cursor の**統合ターミナル**でClaude Codeを使用している場合のみ関係します
- 外部ターミナル（Terminal.app、iTerm2など）では問題は発生していません
- VS Code 1.105以降では別の修正が適用されている可能性があります
