## 原文（日本語に翻訳）

エグレスプロキシが有効な場合に Claude Code Remote セッションでリモート MCP サーバーへの接続が失敗する問題を修正しました。

## 原文（英語）

Fixed remote MCP servers failing to connect in Claude Code Remote sessions when the egress proxy is enabled

## 概要

Claude Code Remote セッション（SSH やリモート開発環境）でエグレスプロキシが有効な環境において、リモート MCP サーバーへの接続が失敗する問題が修正されました。v2.1.152 以降は、エグレスプロキシが設定されている環境でも、リモート MCP サーバーに正常に接続できます。

## 影響を受けていた状況

- 企業のプロキシ環境や特定のネットワーク設定でエグレスプロキシが有効な場合
- Claude Code Remote（SSH 接続や VS Code Remote などを通じたリモート開発）セッションを使用している場合
- リモート MCP サーバーを設定している場合

## 注意点

- v2.1.152 以降では自動的に修正されており、特別な操作は不要です
