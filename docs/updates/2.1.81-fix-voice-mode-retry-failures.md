# 修正：音声モードでリトライ失敗が握りつぶされていた問題

## 原文（日本語に翻訳）

音声モードがリトライ失敗を黙って握りつぶし、実際のエラーの代わりに誤解を招く「ネットワークを確認してください」というメッセージを表示していた問題を修正。

## 原文（英語）

Fixed voice mode silently swallowing retry failures and showing a misleading "check your network" message instead of the actual error

## 概要

音声モード使用中に接続エラーが発生した場合、以前は「ネットワークを確認してください」という誤解を招くメッセージが表示されていました。実際の原因がネットワーク以外（APIエラー、認証エラーなど）であっても同じメッセージが出ていたため、トラブルシューティングが困難でした。この修正により、実際のエラーメッセージが正確に表示されるようになります。

## 基本的な使い方

音声モードでエラーが発生した際、実際のエラー内容が表示されるようになりました：

```
# 以前（修正前）
エラー: ネットワークを確認してください

# 修正後
エラー: APIレート制限に達しました (429 Too Many Requests)
```

## 実践例

### ### エラーの正確な診断

修正後は、エラーの種類に応じた適切な対処が可能：

- **認証エラー**: APIキーや認証設定を確認
- **レート制限エラー**: しばらく待ってから再試行
- **サーバーエラー**: Anthropicのステータスページを確認
- **実際のネットワークエラー**: ネットワーク接続を確認

## 注意点

- 音声モードは現在実験的な機能のため、設定で有効化する必要がある場合がある
- エラーメッセージが英語で表示される場合がある
