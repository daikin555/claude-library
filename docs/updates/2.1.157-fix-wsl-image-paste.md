## 原文（日本語に翻訳）

WSL: 画像ペースト（`alt+v` キーバインド）の修正、Windows 11 でのスクリーンショットペーストの修正、および Windows Explorer からの画像ドラッグ対応を追加しました。

## 原文（英語）

WSL: fixed image paste (`alt+v` keybinding), screenshot paste on Windows 11, and added support for dragging images from Windows Explorer

## 概要

WSL（Windows Subsystem for Linux）環境でのClaude Codeにおける画像関連の操作に複数の問題がありました。`alt+v` キーバインドによる画像ペーストが動作しない問題と、Windows 11 でのスクリーンショットペーストが機能しない問題が修正されました。さらに、Windows Explorer から直接画像をドラッグしてClaude Codeに送れる機能も追加されました。これらの修正・改善はv2.1.157に含まれています。

## 影響を受けていた状況

- WSL環境でClaude Codeを使用しており、`alt+v` で画像を貼り付けようとしたが動作しなかった場合
- Windows 11 上のWSLで、PrintScreenキーやSnipping Toolで取得したスクリーンショットをClaude Codeに貼り付けられなかった場合
- WSL環境でWindows側のファイルシステムにある画像をClaude Codeに渡す手段がなかった場合

## 注意点

- v2.1.157 以降では自動的に修正されており、特別な操作は不要です
- Windows Explorer からのドラッグ機能は新機能として追加されたものです。WSL環境で画像ファイルをエクスプローラーからClaude Codeウィンドウにドラッグすることで画像を直接送信できます
