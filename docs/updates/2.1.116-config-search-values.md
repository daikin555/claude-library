## 原文（日本語訳）

`/config` の検索がオプション値にもマッチするようになりました（例：「vim」で検索するとEditorモード設定が見つかります）。

## 原文（英語）

`/config` search now matches option values (e.g. searching "vim" finds the Editor mode setting)

## 概要

`/config` コマンドの検索機能が強化され、設定項目の名前だけでなく**オプション値**（選択肢の値）にもマッチするようになりました。たとえば「vim」と検索すると、Editorモードの選択肢にvimが含まれているため、そのEditor mode設定が検索結果に表示されます。設定の正式名称を覚えていなくても、関連するキーワードから設定にたどり着けるようになりました。

## 基本的な使い方

```bash
# /config を開いて検索フィールドにキーワードを入力
claude /config
# → 検索ボックスで "vim" と入力 → Editor mode 設定が表示される
```

## 実践例

### エディタモード設定を値名で検索

```bash
claude /config
# 検索: "vim"
# → "Editor mode" 設定がヒット（vimが選択肢の一つのため）

# 検索: "emacs"
# → "Editor mode" 設定がヒット（emacsが選択肢の一つのため）
```

### テーマ設定をカラー名で検索

```bash
claude /config
# 検索: "dark"
# → テーマ関連の設定がヒット（darkモードが選択肢に含まれるため）

# 検索: "solarized"
# → そのテーマが選択肢に含まれる設定がヒット
```

### モデル名で設定を検索

```bash
claude /config
# 検索: "opus"
# → モデル選択設定がヒット

# 検索: "haiku"
# → モデル選択設定がヒット
```

## 注意点

- 検索は設定項目名・説明文・**オプション値**の全てにマッチします
- 大文字・小文字は区別されません（"VIM" でも "vim" でもマッチします）
- 設定の値として存在するキーワードであれば検索でたどり着けます
- スペルミスには対応していないため、正確なキーワードで検索してください
