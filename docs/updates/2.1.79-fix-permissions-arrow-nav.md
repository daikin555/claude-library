## 原文（日本語に翻訳）

`/permissions` における左右矢印キーのタブナビゲーションを修正しました

## 原文（英語）

Fixed left/right arrow tab navigation in `/permissions`

## 概要

`/permissions` コマンドで開くパーミッション管理画面において、左右の矢印キーを使ったタブ間の移動が機能しない問題がありました。本来はタブを切り替えるために矢印キーを使えるはずですが、キー入力が反応しない状態でした。この修正により、`/permissions` 画面での左右矢印キーを使ったタブナビゲーションが正しく動作するようになります。

## 基本的な使い方

`/permissions` コマンドを実行後、左右の矢印キーでタブを切り替えられるようになりました。

```
# パーミッション管理画面を開く
/permissions

# 修正後のキー操作
← （左矢印）: 前のタブに移動
→ （右矢印）: 次のタブに移動
Tab: 項目間の移動（別の用途）
```

## 実践例

### パーミッション設定の確認と変更

```
# Claude Codeのプロンプトで実行
/permissions

# 画面が開いたら矢印キーでタブを切り替え
→ 「許可済み」タブを表示
← 「ブロック済み」タブに戻る

# 各タブでのパーミッション設定を確認・変更
```

### ツール別パーミッションの管理

```
# /permissions 画面でのタブナビゲーション
# タブ1: すべてのパーミッション
→ キーを押す
# タブ2: 許可済みパーミッション
→ キーを押す
# タブ3: ブロック済みパーミッション
← キーを押して前のタブに戻る
```

### キーボードのみでのパーミッション管理

```
# マウスを使わずにキーボードだけでパーミッションを管理
/permissions     # コマンド実行
→               # 次のタブへ
↑↓             # タブ内のアイテムを選択
Enter           # 選択したパーミッションを変更
```

## 注意点

- **以前の回避策**: 修正前はTabキーや他のキーでナビゲーションを試みる必要がありましたが、今回の修正で矢印キーが正しく機能します
- **Tabキーとの違い**: 左右矢印キーはタブ切り替え用、Tabキーは要素間のフォーカス移動用として使い分けられます
- **関連する修正**: v2.1.80でも `/permissions` のキーボード操作に関する改善が行われています
