# 長時間セッションで無制限に増加するJSONパースキャッシュのメモリリーク修正

## 原文（日本語に翻訳）

長時間セッションで無制限に増加していたJSONパースキャッシュのメモリリークを修正しました。

## 原文（英語）

Fixed memory leak in JSON parsing cache that grew unbounded over long sessions

## 概要

Claude Code v2.1.63で、JSON解析に使用するキャッシュのメモリリークが修正されました。このキャッシュは長時間セッション中に制限なく増加し続け、最終的にパフォーマンスの低下やメモリ不足の原因となる可能性がありました。修正後はキャッシュが適切に管理され、長時間セッションでもメモリ使用量が安定します。

## 基本的な使い方

この修正はバックグラウンドで自動的に適用されます。ユーザー側での操作は不要です。

## 実践例

### 影響を受けるシナリオ

- JSON設定ファイルや大量のJSONデータを扱う長時間セッション
- APIレスポンスなど多くのJSONを解析する作業
- 長時間継続するCI/CDパイプラインでのClaude Code使用

### 修正後の改善

- 長時間セッションでのメモリ使用量が安定
- JSONを多く扱うプロジェクトでのパフォーマンスが維持される

## 注意点

- v2.1.63にアップデートすることで自動的に修正が適用されます
- 長時間セッションで作業するすべてのユーザーに恩恵があります
