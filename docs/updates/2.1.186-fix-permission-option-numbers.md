# 権限プロンプトでオプションテキストが折り返す際のオプション番号の位置ずれを修正

## 原文（日本語に翻訳）
オプションテキストがあふれる際に、権限プロンプトのオプション番号の位置がずれる問題を修正しました。

## 原文（英語）
Fixed misaligned option numbers in permission prompts when the option text overflows

## 概要
権限承認ダイアログで、選択肢のテキストが長くて折り返す場合に、オプション番号（1, 2, 3...）の表示位置がずれる視覚的な問題がありました。この修正により、テキストが折り返す場合でもオプション番号が正しく整列して表示されるようになりました。

## 基本的な使い方

この修正は自動的に適用されます。権限プロンプトの表示が改善されます。

### 影響を受けていたシナリオ
長いパスやコマンドを含む権限プロンプトで：
```
[1] 許可: /very/long/path/that/overflows/the/terminal/width/file.txt
[2] 拒否
[3] 常に許可
```
（以前）番号がずれて読みにくい表示になっていた

### 修正後
長いテキストが折り返す場合でも、番号が正しく整列して読みやすく表示されます。

## 注意点
- UIの表示に関するバグ修正です
- 特に長いファイルパスや長いコマンドが含まれる権限プロンプトで効果が見られます

## 関連情報
- 権限設定の管理: `.claude/settings.json` の `permissions`
