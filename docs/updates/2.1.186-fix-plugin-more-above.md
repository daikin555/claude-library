# /plugin のInstalledタブで最上部にスクロール済みの場合に「上に続きあり」が表示される問題を修正

## 原文（日本語に翻訳）
`/plugin` の Installed タブで、すでに最上部にスクロールしているのに「上に続きあり」インジケーターが表示される問題を修正しました。

## 原文（英語）
Fixed `/plugin` Installed showing a "more above" indicator when already scrolled to the top

## 概要
`/plugin` コマンドのインストール済みタブを最上部まで スクロールした後も、「上に続きあり（more above）」を示すインジケーターが誤って表示される問題がありました。この修正により、インジケーターが正確なスクロール位置を反映するようになりました。

## 基本的な使い方

この修正は自動的に適用されます。`/plugin` の Installed タブのスクロールインジケーターが正確に表示されます。

### 影響を受けていたシナリオ
1. `/plugin` を開いて Installed タブを表示
2. 最上部まで上スクロール
3. （以前）最上部にいるのに「↑ more above」インジケーターが消えなかった

### 修正後
最上部にスクロールすると、「上に続きあり」インジケーターが正しく非表示になります。

## 注意点
- UIの表示に関するバグ修正です
- 機能的な動作への影響はありません
