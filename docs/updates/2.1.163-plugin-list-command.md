## 原文（日本語に翻訳）

インストール済みプラグインを一覧表示する `/plugin list` コマンドを追加。`--enabled` / `--disabled` フィルターに対応。

## 原文（英語）

Added `/plugin list` command to list installed plugins, with `--enabled`/`--disabled` filters

## 概要

`/plugin list` コマンドを使用して、現在インストールされているすべてのプラグインを確認できるようになりました。`--enabled` または `--disabled` フラグを使えば、有効・無効のプラグインを絞り込んで表示できます。プラグインの管理や動作確認がより直感的になります。

## 基本的な使い方

Claude Code のチャット内で以下のコマンドを入力します：

```
/plugin list
```

インストール済みのすべてのプラグインが一覧表示されます。

## 実践例

### 有効なプラグインのみを確認

```
/plugin list --enabled
```

現在有効になっているプラグインだけを表示します。

### 無効なプラグインを確認

```
/plugin list --disabled
```

インストールはされているが無効になっているプラグインを表示します。動作していないプラグインのトラブルシューティングに役立ちます。

### プラグインの状態確認フロー

1. `/plugin list` で全プラグインを確認
2. 問題のあるプラグインを `/plugin list --disabled` で特定
3. `/plugin enable <plugin-name>` で必要なプラグインを有効化

## 注意点

- `/plugin list` は `/plugin` コマングループの一部です。`/plugin` とだけ入力するとサブコマンドの一覧が表示されます。
- プラグインのインストール・削除・有効化・無効化には、それぞれ別のサブコマンドを使用します。
- フィルターなしの場合、有効・無効に関わらずすべてのインストール済みプラグインが表示されます。
