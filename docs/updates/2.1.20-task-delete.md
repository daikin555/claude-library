## 原文（日本語訳）
`TaskUpdate` ツールを介してタスクを削除する機能が追加されました

## 原文（英語）
Added ability to delete tasks via the `TaskUpdate` tool

## 概要
Claude が `TaskUpdate` ツールを使用してタスクを削除できるようになりました。不要になったタスクや完了したタスクを整理し、タスクリストを常にクリーンな状態に保てます。

## 基本的な使い方

Claude がタスクを削除する際、自動的に `TaskUpdate` ツールを使用します。

```bash
claude
> 完了したタスクを削除してください
```

Claude が適切なタスクを識別して削除します。

## 実践例

### 完了済みタスクの削除

```bash
claude
> /tasks
# 現在のタスクリストを確認

# タスクリスト:
# ✅ 1. ユーザー認証APIの実装
# ✅ 2. データベースマイグレーション
# ⏳ 3. フロントエンド統合

> 完了したタスク（1と2）を削除して

# Claude が TaskUpdate ツールを使用してタスクを削除
```

### 不要なタスクのクリーンアップ

```bash
claude
> 新しい方向性に変更したので、古いタスクをすべて削除して

# Claude が関連するタスクを特定して削除
```

### 特定のタスクの削除

```bash
claude
> 「レガシーコードのリファクタリング」タスクはもう必要ないので削除して

# Claude が該当タスクを検索して削除
```

## タスクシステムの概要

### タスクの作成から削除まで

```bash
# 1. タスク作成
claude
> 大規模なリファクタリングを計画してください

# Claude がタスクを作成:
# - タスク1: コードベース分析
# - タスク2: リファクタリング計画
# - タスク3: 実装
# - タスク4: テスト

# 2. タスク実行
> タスク1を実行してください

# 3. タスク削除
> タスク1は完了したので削除してください
```

### タスクステータスの管理

```bash
# タスクの状態遷移
Created → In Progress → Completed → Deleted
          ↓
          Cancelled → Deleted
```

## TaskUpdate ツールの機能

`TaskUpdate` ツールは以下の操作をサポート：

- タスクの更新（ステータス、説明、依存関係）
- タスクの削除（新機能）
- タスクの優先順位変更

## 注意点

- **復元不可**: 削除したタスクは復元できません
- **依存関係**: 他のタスクが依存している場合、削除前に警告される可能性があります
- **バックグラウンドタスク**: 実行中のバックグラウンドタスクは削除できません（まず停止が必要）
- **セッション固有**: タスクはセッションに紐付いており、削除もそのセッション内でのみ有効です
