## 原文（日本語に翻訳）

ファイルウォッチャーが発火後のクリーンアップを見逃した際に、ワンショットのスケジュールタスクが繰り返し再起動する問題を修正しました。

## 原文（英語）

Fixed one-shot scheduled tasks re-firing repeatedly when the file watcher missed the post-fire cleanup

## 概要

一度だけ実行されるはずのスケジュールタスク（ワンショットタスク）が、ファイルウォッチャーのタイミング問題により複数回実行されてしまうバグが修正されました。タスクの発火後のクリーンアップ処理がファイルウォッチャーに適切に伝わらない場合に発生していた問題です。

## 修正内容

ワンショットスケジュールタスクが確実に1回だけ実行されるようになりました。
