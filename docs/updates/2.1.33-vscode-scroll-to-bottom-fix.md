## 原文（日本語に翻訳）

VSCodeのスクロール最下部まで到達しない問題（under-scrolling）を修正しました。

## 原文（英語）

VSCode scroll-to-bottom under-scrolling fix

## 概要

VSCode拡張で会話の最下部までスクロールしようとしても、完全には到達せず、最後のメッセージが部分的に隠れてしまう問題が修正されました。これにより、VSCode内でClaude Codeを使用する際のユーザー体験が向上します。

## 基本的な使い方

VSCode拡張でClaude Codeを使用する際、自動的にスクロールが適切に機能します。

```bash
# VSCode内でClaude Code拡張を開く
# 会話が進むと自動的に最下部までスクロールされる
```

## 実践例

### 長い応答の表示

Claudeが長い応答を生成する場合、最後まで適切に表示されます。

```typescript
// VSCode内でClaude Codeに質問
"このファイルのリファクタリング案を複数提案してください"

// 修正前：最後の提案が部分的に隠れる
// 修正後：すべての提案が完全に表示される
```

### コード生成時のスクロール

複数のファイルを生成する際のスクロール動作。

```typescript
// 複数ファイルの生成を依頼
"REST APIのCRUD操作を実装してください"

// Claudeが複数のファイルを生成
// 各ファイルの内容が完全に表示される
// 最後のファイルも隠れない
```

### インタラクティブな対話

質問と回答を繰り返す場合。

```typescript
// 連続した質問
"この関数の説明をしてください"
// 応答を確認

"次にこの関数を最適化してください"
// 応答を確認

// すべての応答が完全に表示される
```

## 注意点

- この修正はVSCode拡張にのみ適用されます
- ターミナル版のClaude Codeには影響しません
- 自動スクロールは応答のストリーミング中も機能します
- 手動でスクロールした場合、自動スクロールは一時的に無効になります
- VSCode拡張は最新バージョンに更新してください
