## 原文（英語）

Added Voice STT support for 10 new languages (20 total) — Russian, Polish, Turkish, Dutch, Ukrainian, Greek, Czech, Danish, Swedish, Norwegian

## 概要

Claude CodeのVoice STT（音声テキスト変換）機能に10言語が新たに追加され、合計20言語に対応しました。これにより、ヨーロッパ言語を母語とする多くのユーザーが音声でClaudeに指示を出せるようになります。

## 対応言語一覧（20言語）

今回追加された10言語：
- 🇷🇺 ロシア語（Russian）
- 🇵🇱 ポーランド語（Polish）
- 🇹🇷 トルコ語（Turkish）
- 🇳🇱 オランダ語（Dutch）
- 🇺🇦 ウクライナ語（Ukrainian）
- 🇬🇷 ギリシャ語（Greek）
- 🇨🇿 チェコ語（Czech）
- 🇩🇰 デンマーク語（Danish）
- 🇸🇪 スウェーデン語（Swedish）
- 🇳🇴 ノルウェー語（Norwegian）

## 基本的な使い方

音声入力はスペースバーを長押しして起動します：

```
[スペースバーを長押し] → 話す → 離す → テキストに変換
```

## 実践例

### 音声でコード生成を指示する

ロシア語での例：
```
"Создай функцию для сортировки массива чисел"
（数値配列をソートする関数を作成して）
```

オランダ語での例：
```
"Schrijf een unit test voor de login functie"
（ログイン機能のユニットテストを書いて）
```

### 長いコマンドを声で入力する

技術的な英語の用語が混在する場合も、母語で自然に話せます：

```
"Git commitしたあとで、テストを実行してエラーがあったら教えて"
（日本語も引き続き対応）
```

## 注意点

- 音声入力はスペースバーの長押しで起動します
- 言語は自動検出されます（手動設定不要）
- ノイズの多い環境では誤認識が増える場合があります
- 音声入力機能はデスクトップ版Claude Codeおよびモバイルアプリで利用可能です
