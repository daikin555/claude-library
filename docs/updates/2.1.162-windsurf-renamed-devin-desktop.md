# Windsurf が Devin Desktop にリネーム：/ide メニューなどに反映

## 原文（日本語に翻訳）

エディターのリブランドに伴い、`/ide` メニュー、`/terminal-setup`、`/scroll-speed` で Windsurf が Devin Desktop に改称されました。

## 原文（英語）

Renamed Windsurf to Devin Desktop in the `/ide` menu, `/terminal-setup`, and `/scroll-speed`, following the editor's rebrand

## 概要

Windsurf エディターがリブランドにより「Devin Desktop」に名称変更されたことを受け、Claude Code の各種コマンドメニューでも名称が更新されました。`/ide`、`/terminal-setup`、`/scroll-speed` コマンドで表示される IDE 一覧で「Windsurf」の代わりに「Devin Desktop」が表示されます。機能自体に変更はなく、名称の変更のみです。

## 基本的な使い方

```
# IDEを選択するコマンド（Devin Desktop が選択肢に表示される）
/ide

# ターミナル設定（Devin Desktop 向けの設定が適用される）
/terminal-setup

# スクロール速度の設定
/scroll-speed
```

## 実践例

### /ide コマンドで Devin Desktop を選択

```
/ide
# メニューに以下が表示される:
# - VS Code
# - JetBrains
# - Devin Desktop  ← (旧: Windsurf)
# - その他のエディター
```

### Devin Desktop 向けのターミナル設定

```
/terminal-setup
# Devin Desktop 用の最適な設定が自動適用される
```

### スクロール速度の調整

```
/scroll-speed
# Devin Desktop を使用している場合の最適な速度設定
```

## 注意点

- 機能や設定の内容は変わっておらず、名称のみの変更です
- 以前 Windsurf として設定していた構成は Devin Desktop として引き継がれます
- Windsurf の設定ファイルや環境変数は変更が必要な場合があります
- このリブランドはエディター側（Windsurf → Devin Desktop）のもので、Claude Code はそれに追従しています
