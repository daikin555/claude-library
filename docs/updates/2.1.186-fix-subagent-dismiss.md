# エージェントパネルで完了したサブエージェントにxを押しても消えない問題を修正

## 原文（日本語に翻訳）
エージェントパネルで完了したサブエージェントに `x` キーを押しても非表示にならない問題を修正しました。

## 原文（英語）
Fixed pressing `x` on a finished subagent in the agent panel not dismissing it

## 概要
`claude agents` などのエージェントパネルで完了（finished）ステータスになったサブエージェントに対して `x` キーを押しても、一覧から消えない（dismiss できない）問題がありました。この修正により、完了したサブエージェントを `x` キーで正常に非表示にできるようになりました。

## 基本的な使い方

この修正は自動的に適用されます。

### 完了したサブエージェントの非表示
1. `claude agents` でエージェント一覧を表示
2. 完了（finished）したサブエージェントにカーソルを移動
3. `x` キーを押す → エージェントが一覧から非表示になる（修正後は正常に動作）

## 注意点
- `x` キーはサブエージェントを完全に削除するのではなく、パネルから非表示にします
- 実行中のサブエージェントは `x` では非表示にできません
