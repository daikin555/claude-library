## 原文（日本語に翻訳）

/ メニューに表示される組み込みコマンドとスキルの説明がより明確になりました。

## 原文（英語）

Clearer descriptions for built-in commands and skills in the / menu

## 概要

Claude Code の `/` メニュー（スラッシュコマンドメニュー）に表示される組み込みコマンドおよびスキルの説明文が改善されました。各コマンドの目的や使い方が一目でわかりやすくなり、初めてのユーザーや、使用頻度の低いコマンドを探すときに便利です。

## 基本的な使い方

チャット内で `/` を入力するとコマンドメニューが表示されます。改善された説明文により、目的のコマンドを素早く見つけられます：

```
/     ← これを入力するとメニューが開く
```

メニュー内の各コマンドの説明が以前よりも具体的になっており、コマンドを選択する前に何をするものか判断しやすくなっています。

## 実践例

### コマンドを探すときの活用方法

1. チャットで `/` を入力
2. コマンド名の一部を入力してフィルタリング
3. 改善された説明文を読んで目的のコマンドを選択

例えば、デバッグ関連のコマンドを探す場合：

```
/debug    ← 入力するとデバッグ関連コマンドが絞り込まれ、説明も表示される
```

## 注意点

- これは UI テキストの改善であり、コマンドの動作自体に変更はありません。
- カスタムスキル（ユーザーが追加したスキル）の説明はスキルの定義ファイルに記述したものが表示されます。組み込みコマンドのみが今回の改善対象です。
