## 原文（日本語に翻訳）

フルスクリーンモードで処理中メッセージのテキストにバッククォートやアスタリスクなどのマークダウン記号がそのまま表示される問題を修正しました。

## 原文（英語）

Fixed literal markdown markers (backticks, asterisks) appearing in the in-progress message text in fullscreen mode

## 概要

フルスクリーンモードでClaudeが処理中に表示されるメッセージテキスト内で、バッククォート（`` ` ``）やアスタリスク（`*`）などのマークダウン記号がレンダリングされずにそのままの文字として表示されるバグがありました。これによりメッセージの可読性が低下し、整形されていない生のテキストが表示されていました。このバグはv2.1.157で修正されました。

## 影響を受けていた状況

- フルスクリーンモード（`/fullscreen` またはキーボードショートカットで切り替えた状態）でClaudeを使用していた場合
- Claudeがコードブロックやボールドテキストなどのマークダウン書式を含む返答を生成している最中に表示が乱れていた場合
- 処理中（ストリーミング中）のメッセージにマークダウン記号が多く含まれていた場合

## 注意点

- v2.1.157 以降では自動的に修正されており、特別な操作は不要です
