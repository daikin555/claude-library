## 原文（日本語に翻訳）

[VSCode] 最近閉じたセッションタブを再度開くためにCmd/Ctrl+Shift+Tを押すことができます。`claudeCode.enableReopenClosedSessionShortcut`で設定可能

## 原文（英語）

[VSCode] Press Cmd/Ctrl+Shift+T to reopen the most recently closed session tab, configurable via `claudeCode.enableReopenClosedSessionShortcut`

## 概要

VSCode拡張機能において、最後に閉じたClaude Codeセッションタブを`Cmd/Ctrl+Shift+T`で再開できるようになりました。ブラウザのタブ復元と同様の操作感で、誤ってセッションを閉じてしまった場合に素早く復元できます。

## 基本的な使い方

| OS | ショートカット |
|----|-------------|
| macOS | `Cmd+Shift+T` |
| Windows/Linux | `Ctrl+Shift+T` |

## 実践例

### 誤って閉じたセッションの復元

```
# Claude Codeセッションを誤って閉じてしまった場合
Cmd/Ctrl+Shift+T
→ 最後に閉じたセッションタブが復元される
```

### 設定でのカスタマイズ

VSCodeの設定でショートカットを有効/無効にできます：

```json
// settings.json
{
  "claudeCode.enableReopenClosedSessionShortcut": true
}
```

デフォルトでは有効になっています。無効にしたい場合は`false`に設定します。

## 注意点

- この機能はVSCode拡張機能でのみ利用可能です（CLI版では使用不可）
- 復元されるのは最後に閉じたセッションのタブのみです
- セッションの会話履歴はセッションIDで管理されており、タブを再開すると前回の会話が復元されます
- `Cmd/Ctrl+Shift+T`はVSCode自体のタブ復元ショートカットと同じキーバインドです
