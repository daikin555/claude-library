## 原文（日本語に翻訳）

フルスクリーンモードでマウスホイールのスクロール加速を無効化する `wheelScrollAccelerationEnabled` 設定が追加されました。

## 原文（英語）

Added `wheelScrollAccelerationEnabled` setting to disable mouse-wheel scroll acceleration in fullscreen mode

## 概要

フルスクリーンモードでマウスホイールを使ったスクロールには加速（勢いをつけると一気に大きくスクロールする挙動）が適用されますが、これを `wheelScrollAccelerationEnabled` 設定で無効化できるようになりました。スクロール量を一定に保ちたいユーザーや、加速によって意図せず行き過ぎてしまうのを避けたいユーザー向けの設定です。

## 基本的な使い方

`settings.json` に設定を追加してスクロール加速を無効化します。

```json
{
  "wheelScrollAccelerationEnabled": false
}
```

デフォルトでは加速が有効（`true`）のままなので、無効にしたい場合のみ明示的に `false` を指定します。

## 実践例

### スクロールの行き過ぎを防ぐ

```json
{
  "wheelScrollAccelerationEnabled": false
}
```

加速があると、ホイールを素早く回したときに意図した位置を大きく通り過ぎてしまうことがあります。`false` にすると1回のホイール操作あたりのスクロール量が一定になり、目的の位置で止めやすくなります。

### プロジェクト単位での設定

特定のプロジェクトでのみ無効化したい場合は、プロジェクトの `.claude/settings.json` に記述します。

```json
{
  "wheelScrollAccelerationEnabled": false
}
```

## 注意点

- この設定はフルスクリーンモードでのマウスホイールスクロールに対する加速挙動を制御します
- デフォルト値は有効（加速あり）です
- 加速の有無は好みによる部分が大きいため、まずデフォルトで使ってみて、スクロールが扱いづらいと感じた場合に無効化を検討するとよいでしょう
