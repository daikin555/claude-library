## 原文（日本語に翻訳）

起動時にマネージド設定のセキュリティダイアログを承認した後にターミナルがフリーズする問題を修正しました。

## 原文（英語）

Fixed the terminal freezing after approving the managed-settings security dialog at startup

## 概要

Claude Codeの起動時に表示されるマネージド設定のセキュリティダイアログで承認操作を行った後、ターミナルが応答しなくなるバグがありました。この問題により、Claude Codeを再起動するか、ターミナルウィンドウを強制終了する必要がありました。このバグはv2.1.157で修正されました。

## 影響を受けていた状況

- 組織や管理者によってマネージド設定が構成されている環境でClaude Codeを使用していた場合
- Claude Codeの起動時に表示されるセキュリティ確認ダイアログで「承認」を選択した直後にターミナルが操作不能になっていた場合
- 企業環境や管理されたデプロイメント環境でClaude Codeを利用していた場合

## 注意点

- v2.1.157 以降では自動的に修正されており、特別な操作は不要です
