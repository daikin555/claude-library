## 原文（日本語に翻訳）

マウスホイールのスクロール速度をライブプレビュー付きで調整できる`/scroll-speed`コマンドを追加

## 原文（英語）

Added `/scroll-speed` command to tune mouse wheel scroll speed with a live preview

## 概要

`/scroll-speed`コマンドにより、Claude Codeのターミナル内でマウスホイールのスクロール速度をインタラクティブに調整できるようになりました。設定変更はライブプレビューで即座に確認でき、好みの速度に微調整することが可能です。

## 基本的な使い方

```
/scroll-speed
```

コマンドを実行すると速度調整UIが表示され、ライブプレビューを見ながらスクロール速度を変更できます。

## 実践例

### スクロール速度の調整

```
/scroll-speed
```

UIが表示されたら：
- 矢印キーまたはスライダーで速度を調整
- 変更がリアルタイムでプレビューされる
- Enterキーで確定

### 推奨設定の目安

| 環境 | 推奨設定 |
|------|---------|
| 通常のターミナル | 中速（デフォルト） |
| 長いログを頻繁に見る | 高速 |
| 精密な操作が必要 | 低速 |
| トラックパッド使用 | 低〜中速 |

## 注意点

- 設定はClaude Codeの設定ファイルに保存され、次回起動時も維持されます
- マウスホイールとトラックパッドで別々に動作する場合があります
- VSCode/Cursorなどのエディタ統合環境では、ホストエディタの設定も影響することがあります
- バージョン2.1.139では、VSCode 1.92〜1.104でのマウスホイールスクロール速度の修正も同時に行われています
