# /plugin の Installedタブに「Skills」セクションを追加

## 原文（日本語に翻訳）
`/plugin` のインストール済みタブに「Skills」セクションを追加しました。

## 原文（英語）
Added a "Skills" section to the `/plugin` Installed tab

## 概要
`/plugin` コマンドのインストール済みタブに、新たに「Skills」セクションが追加されました。これにより、インストール済みのスキルをプラグインと同じインターフェースから確認・管理できるようになりました。スキルとプラグインの管理が一元化され、使用中のツールの全体像を把握しやすくなっています。

## 基本的な使い方

```
/plugin
```
コマンドを実行後、「Installed」タブに切り替えると「Skills」セクションが表示されます。

## 実践例

### インストール済みスキルの確認
```
/plugin
# → Installed タブを選択
# → Skills セクションにインストール済みスキルが一覧表示される
```

### スキルとプラグインの統合管理
```
/plugin
```
- Installed タブで、プラグインとスキルを一覧で確認
- 各スキルの説明・タグ・設定を参照
- スキルの有効/無効状態を管理

### スキルの確認後に実行
```
/plugin
# Skills セクションでスキル一覧を確認

# 確認したスキルを実行
/skill-name
```

## 注意点
- スキルは `~/.claude/skills/` ディレクトリや、プラグインのスキルディレクトリから読み込まれます
- スキルがインストールされていない場合、Skills セクションは表示されない場合があります
- プラグイン経由でインストールされたスキルもこのセクションに表示されます

## 関連情報
- スキルの作成: `SKILL.md` ファイルをスキルディレクトリに配置
- プラグイン管理: `/plugin` コマンド
- [Claude Code スキルシステム](https://docs.anthropic.com/en/docs/claude-code/skills)
