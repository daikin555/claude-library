# claude agentsでテキストを削除した後もマウス選択したテキストがハイライトされたままになる問題を修正

## 原文（英語）

## 概要
`claude agents` インターフェース内でマウスを使ってテキストを選択し、削除した後も、削除済みのテキストが選択（ハイライト）状態のまま視覚的に残り続ける問題がありました。この修正により、テキストを削除すると選択状態が正しくクリアされるようになりました。

## 基本的な使い方

この修正は自動的に適用されます。`claude agents` でのテキスト編集体験が改善されます。

### 影響を受けていたシナリオ
1. `claude agents` 内でマウスを使ってテキストを選択
2. 選択したテキストを削除（BackspaceやDeleteキー）
3. （以前）削除後もテキストのハイライトが残っていた

### 修正後
テキストを削除すると選択状態が正しくクリアされます。
