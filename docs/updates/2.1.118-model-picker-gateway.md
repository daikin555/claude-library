## 原文（日本語に翻訳）

`/model`ピッカーがカスタム`ANTHROPIC_BASE_URL`ゲートウェイ使用時に`ANTHROPIC_DEFAULT_*_MODEL_NAME`/`_DESCRIPTION`オーバーライドを反映するようになった。

## 原文（英語）

The `/model` picker now honors `ANTHROPIC_DEFAULT_*_MODEL_NAME`/`_DESCRIPTION` overrides when using a custom `ANTHROPIC_BASE_URL` gateway

## 概要

企業や組織が独自のAnthropicゲートウェイ（プロキシ）を使用している場合に、`ANTHROPIC_DEFAULT_*_MODEL_NAME`および`ANTHROPIC_DEFAULT_*_MODEL_DESCRIPTION`環境変数で設定したモデル名の上書き設定が`/model`ピッカーに正しく反映されるようになりました。カスタムゲートウェイ経由で内部名称が異なるモデルを使用する環境でも、管理者が定義したモデル名をUIに表示できます。

## 基本的な使い方

環境変数でモデル名をオーバーライドして設定：

```bash
# カスタムゲートウェイのURLを設定
export ANTHROPIC_BASE_URL="https://proxy.internal.company.com"

# モデル名をオーバーライド
export ANTHROPIC_DEFAULT_SMALL_MODEL_NAME="company-claude-haiku"
export ANTHROPIC_DEFAULT_SMALL_MODEL_DESCRIPTION="社内承認済み高速モデル"

export ANTHROPIC_DEFAULT_LARGE_MODEL_NAME="company-claude-sonnet"
export ANTHROPIC_DEFAULT_LARGE_MODEL_DESCRIPTION="社内承認済み高性能モデル"
```

`/model`コマンドを実行すると、上記の名前と説明が表示されます。

## 実践例

### 企業プロキシ環境での設定

```bash
# ~/.bashrc または ~/.zshrc
export ANTHROPIC_BASE_URL="https://ai-gateway.corp.example.com/v1"
export ANTHROPIC_DEFAULT_SMALL_MODEL_NAME="approved-haiku-v2"
export ANTHROPIC_DEFAULT_SMALL_MODEL_DESCRIPTION="コスト効率重視モデル（IT承認済み）"
export ANTHROPIC_DEFAULT_LARGE_MODEL_NAME="approved-sonnet-v2"
export ANTHROPIC_DEFAULT_LARGE_MODEL_DESCRIPTION="高性能モデル（IT承認済み）"
```

Claude Codeの`/model`ピッカーで：

```
モデルを選択:
  ● approved-sonnet-v2 - 高性能モデル（IT承認済み）
  ○ approved-haiku-v2  - コスト効率重視モデル（IT承認済み）
```

### settings.jsonでの設定（環境変数が使えない場合）

```json
{
  "env": {
    "ANTHROPIC_BASE_URL": "https://ai-gateway.corp.example.com/v1",
    "ANTHROPIC_DEFAULT_LARGE_MODEL_NAME": "internal-sonnet",
    "ANTHROPIC_DEFAULT_LARGE_MODEL_DESCRIPTION": "社内高性能LLM",
    "ANTHROPIC_DEFAULT_SMALL_MODEL_NAME": "internal-haiku",
    "ANTHROPIC_DEFAULT_SMALL_MODEL_DESCRIPTION": "社内高速LLM"
  }
}
```

### モデルを切り替える

```bash
# /modelコマンドでカスタム名のモデルを選択
/model
# → カスタムゲートウェイで定義した名前のモデルが表示される
```

## 注意点

- この機能は`ANTHROPIC_BASE_URL`でカスタムゲートウェイを設定している場合にのみ効果があります
- `ANTHROPIC_DEFAULT_*_MODEL_NAME`は実際にAPIに送信されるモデルIDを変更します（UI表示だけでなく）
- `ANTHROPIC_DEFAULT_*_MODEL_DESCRIPTION`はUIの説明文のみを変更します
- ゲートウェイがサポートするモデル名と一致させてください
