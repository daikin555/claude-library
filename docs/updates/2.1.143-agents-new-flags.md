## 原文（日本語に翻訳）

- `claude agents` が `--add-dir`、`--settings`、`--mcp-config`、`--plugin-dir` を受け付け、ダッシュボードとそこから起動されるバックグラウンドセッションに適用するようになりました
- `claude agents` が `--permission-mode`、`--model`、`--effort`、`--dangerously-skip-permissions` を受け付け、ビューから起動されるセッションのデフォルトを設定するようになりました

## 原文（英語）

- `claude agents` now accepts `--add-dir`, `--settings`, `--mcp-config`, and `--plugin-dir` and applies them to the dashboard and to background sessions dispatched from it
- `claude agents` accepts `--permission-mode`, `--model`, `--effort`, and `--dangerously-skip-permissions` to set defaults for sessions dispatched from the view

## 概要

`claude agents` コマンドに多数の新フラグが追加され、エージェントダッシュボードから起動するバックグラウンドセッションの構成を細かく制御できるようになりました。MCP設定・プラグインディレクトリ・パーミッションモード・モデル指定などをダッシュボード起動時に一括設定できます。

## 基本的な使い方

```bash
# 新しいフラグの概要
claude agents [オプション]

# 設定系フラグ
  --add-dir <path>        追加のディレクトリを含める
  --settings <path>       カスタム設定ファイルを指定
  --mcp-config <path>     MCPサーバー設定ファイルを指定
  --plugin-dir <path>     プラグインディレクトリを指定

# セッションデフォルト系フラグ
  --permission-mode       デフォルトのパーミッションモード
  --model <model>         デフォルトモデルを指定
  --effort <level>        デフォルトの処理レベルを指定
  --dangerously-skip-permissions  パーミッション確認をスキップ
```

## 実践例

### MCP設定を適用してエージェントダッシュボードを起動

```bash
# プロジェクト専用のMCPサーバー設定を使用
claude agents --mcp-config ./mcp-servers.json

# ダッシュボードから起動した全バックグラウンドセッションに
# mcp-servers.json の設定が適用される
```

### 特定モデルをデフォルトにしてダッシュボード起動

```bash
# ダッシュボードから起動するセッションのデフォルトモデルを指定
claude agents \
  --model claude-opus-4-7 \
  --effort high \
  --permission-mode auto
```

### 完全な設定を指定した起動

```bash
# カスタム設定・プラグイン・MCP・モデルをすべて指定
claude agents \
  --settings .claude/team-settings.json \
  --mcp-config .claude/mcp-config.json \
  --plugin-dir .claude/plugins \
  --add-dir ../shared-lib \
  --model claude-sonnet-4-6 \
  --permission-mode auto
```

### チーム開発での共通設定適用

```bash
# チーム共通の設定でエージェントビューを起動するスクリプト例
# scripts/start-agents.sh
#!/bin/bash
claude agents \
  --mcp-config config/team-mcp.json \
  --settings config/team-settings.json \
  --model claude-opus-4-7 \
  --permission-mode auto
```

## 注意点

- `--permission-mode` で指定したモードは、`claude agents` ビューから起動した新しいセッションのデフォルトになります（既存のセッションには影響しません）
- `--dangerously-skip-permissions` を使用する場合は、信頼できる環境でのみ使用してください
- `--mcp-config` と `--settings` はダッシュボード自体と、そこから起動されるすべてのバックグラウンドセッションに適用されます
- これらのフラグは `claude agents` 起動時にのみ有効です。セッション実行中の変更は反映されません
