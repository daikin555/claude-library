## 原文（日本語に翻訳）

長い会話でプロンプト入力が2行目に折り返した際、画面が真っ白になる問題を修正しました。

## 原文（英語）

Fixed screen going blank when the prompt input wraps to a second line in long conversations

## 概要

長い会話セッションでプロンプトを入力中に、テキストが2行目に折り返す際に画面全体が白くなってしまうUIのバグが修正されました。長い日本語テキストや長いコマンドを入力する際に発生していた問題です。

## 修正内容

プロンプトが複数行にわたる場合でも、ターミナル画面が正常に表示されるようになりました。

```
Before: 長いプロンプト入力 → 折り返し時に画面が白くなる
After:  長いプロンプト入力 → 折り返し時も正常に表示
```
