## 原文（日本語に翻訳）

`package.json`とlockfileを持つマーケットプレースプラグインのインストール・更新後に依存関係が自動的にインストールされない問題を修正しました。

## 原文（英語）

Fixed marketplace plugins with `package.json` and lockfile not having dependencies installed automatically after install/update

## 概要

マーケットプレースからインストール・更新されたプラグインが`package.json`とlockfileを含む場合、`npm install`（または対応するパッケージマネージャーのコマンド）が自動的に実行されず、プラグインが動作しない問題が修正されました。

## 修正内容

マーケットプレースからプラグインをインストール・更新した際に、`package.json`を含むプラグインの依存関係が自動的にインストールされるようになりました。

```
Before: プラグインインストール → 依存関係は未インストール → プラグインが動作しない
After:  プラグインインストール → 依存関係が自動インストール → プラグインが正常に動作
```
