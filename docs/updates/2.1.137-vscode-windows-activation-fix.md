# [VSCode] Windows での拡張機能起動失敗の修正

## 原文（日本語に翻訳）

[VSCode] Windows で拡張機能の起動に失敗する問題を修正しました。

## 原文（英語）

[VSCode] Fixed extension failing to activate on Windows

## 概要

Claude Code v2.1.137 では、Windows 環境において VSCode 拡張機能が正常に起動しない問題が修正されました。この問題により、Windows ユーザーが VSCode 内で Claude Code 拡張機能を使用できないケースが発生していました。今回の修正により、Windows 上の VSCode での Claude Code 拡張機能が安定して動作するようになります。

## 基本的な使い方

修正後は、Windows 上の VSCode で Claude Code 拡張機能を通常通りインストールし、使用できます。

```
# VSCode 拡張機能マーケットプレイスから Claude Code をインストール
# または VSCode 内で Ctrl+Shift+X を押して "Claude Code" を検索
```

拡張機能のインストール後、VSCode を再起動することで問題なく起動するようになります。

## 実践例

### Windows 環境でのインストール確認

```powershell
# VSCode のバージョン確認
code --version

# Claude Code 拡張機能のインストール（コマンドライン）
code --install-extension anthropic.claude-code

# 拡張機能の一覧確認
code --list-extensions | findstr claude
```

### 拡張機能が正常に起動しているかの確認

VSCode を起動した後、以下の手順で拡張機能の状態を確認できます：

1. VSCode を開く（Windows の場合: スタートメニューから "Visual Studio Code" を起動）
2. サイドバーの Claude Code アイコンをクリック
3. Claude Code パネルが正常に表示されることを確認

### 以前の問題の症状と対処法

修正前に以下の症状が見られた場合、v2.1.137 以降へのアップデートで解決します：

- VSCode 起動時に "Extension 'Claude Code' failed to activate" などのエラーが表示される
- サイドバーに Claude Code アイコンが表示されない
- Claude Code パネルが空白または応答しない

```powershell
# Claude Code CLI をアップデート（npm 使用の場合）
npm update -g @anthropic-ai/claude-code

# または npx で最新版を確認
npx @anthropic-ai/claude-code --version
```

### VSCode 設定の確認

拡張機能の設定を確認するには：

```json
// .vscode/settings.json
{
  "claude-code.enable": true
}
```

## 注意点

- この修正は Windows 環境の VSCode ユーザーに特に関係します
- macOS および Linux ユーザーへの影響はありません
- v2.1.137 以降へアップデートすることで問題が解消されます
- VSCode 拡張機能を最新版に保つことで、このような問題を避けられます
- 問題が解消しない場合は、VSCode を完全に終了して再起動してみてください
