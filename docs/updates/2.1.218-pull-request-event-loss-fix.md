## 原文（日本語に翻訳）

セッションがプルリクエストの作成またはリンクの直後に終了した場合、プルリクエストイベントが失われることがある問題を修正しました

## 原文（英語）

Fixed pull request events occasionally being lost when a session exited immediately after creating or linking a PR

## 概要

Claude Codeのセッションがプルリクエストを作成またはリンクした直後にすぐ終了すると、そのPR関連イベント（セッションとPRを紐付ける記録など、トラッキングや帰属に使われるもの）がイベント送信前にセッションが終わってしまうことで、まれに失われることがありました。この修正により、セッション終了前にイベントが確実に記録・送信されるようになりました。

## 注意点

- Claude Code v2.1.218で修正されました
- 影響は主にPRの作成・リンク直後に即座にセッションを終了するようなワークフロー（自動化・バッチ実行など)に限定されます
- ユーザー側で追加の設定変更や対応は不要です
