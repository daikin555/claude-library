## 原文（日本語に翻訳）

同じコマンドだが異なる環境変数を持つプラグインの MCP サーバーが誤って重複排除される問題を修正しました。

## 原文（英語）

Fixed plugin MCP servers with the same command but different environment variables being incorrectly deduplicated

## 概要

同じ実行コマンドを使用しているが、異なる環境変数（API キーや設定値など）を持つ複数のプラグイン MCP サーバーが、誤って同一のサーバーとみなされて片方が無効化される問題が修正されました。v2.1.152 以降は、環境変数が異なる場合は別々のサーバーとして正しく認識されます。

## 影響を受けていた状況

- 同じ MCP サーバー実行コマンドを持つが、異なる環境変数を持つ複数のプラグインをインストールしている場合
- 例：同じツールを異なる API キーで複数の環境（開発・本番）向けに設定している場合

## 注意点

- v2.1.152 以降では自動的に修正されており、特別な操作は不要です
- 既存の設定を変更する必要はありません
