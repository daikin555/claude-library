## 原文（日本語に翻訳）

エージェントのカラーパレットを更新

## 原文（英語）

Updated agent color palette

## 概要

エージェントビューやエージェント関連UIで使用されるカラーパレットが更新されました。複数のエージェントを同時に実行する際の視認性と区別のしやすさが改善されます。

## 注意点

- カラーパレットの更新により、エージェントビューでの各セッションの識別が視覚的に改善されます
- `claude agents`コマンドで起動するエージェントビューで変更を確認できます
- ユーザーによる設定変更は不要で、自動的に新しいカラーパレットが適用されます
