## 原文（日本語訳）

Fast mode が Opus 4.6 で利用可能になりました。詳細は https://code.claude.com/docs/en/fast-mode をご覧ください。

## 原文（英語）

Fast mode is now available for Opus 4.6. Learn more at https://code.claude.com/docs/en/fast-mode

## 概要

Claude Code の Fast mode が、最新かつ最も高性能なモデルである Opus 4.6 に対応しました。Fast mode を使用することで、同じ品質と機能を維持しながら、最大2.5倍高速なレスポンスを得ることができます。これは異なるモデルではなく、Opus 4.6 を高速化するための API 設定の最適化により実現されています。

## 基本的な使い方

Fast mode は `/fast` コマンドで簡単に切り替えることができます。

```bash
# Fast mode を有効化
/fast

# 通常モードに戻す
/fast
```

有効化すると、Opus 4.6 の高い知性を保ちながら、最大2.5倍高速な応答が得られます。

## 実践例

### ケース1: ライブデバッグセッション

リアルタイムでコードを修正しながらデバッグする場合、高速な応答が作業効率を大きく向上させます。

```bash
/fast

# エラーログを素早く分析
"このエラーログを分析して、原因を特定してください"

# 即座に修正案を取得
"修正コードを提案してください"

# 素早く反復
"別のアプローチも試してみてください"
```

### ケース2: 高速な反復開発

UI コンポーネントやアルゴリズムを何度も調整しながら開発する際に有効です。

```bash
/fast

# 最初のバージョン
"レスポンシブなナビゲーションバーを作成してください"

# 素早くフィードバック
"もっとモダンなデザインにしてください"

# さらに調整
"アニメーションを追加してください"
```

### ケース3: コードレビューの迅速化

大量のコードレビューや複数ファイルの確認を効率的に行えます。

```bash
/fast

# 複数ファイルを素早くレビュー
"このPRの全ファイルをレビューして、問題点を指摘してください"

# 即座に改善案を取得
"指摘された問題を修正してください"
```

### ケース4: 対話的な学習・探索

新しいフレームワークやライブラリを学習する際、素早い応答で効率的に理解を深められます。

```bash
/fast

"Next.js の App Router について教えてください"

# 即座に実例を確認
"具体的なコード例を見せてください"

# さらに深掘り
"Server Components との違いは何ですか？"
```

## 注意点

- **コスト**: Fast mode はプレミアム価格（$30/$150 per MTok）で提供されます。通常モードよりもコストが高くなる点に注意してください
- **品質**: Fast mode は速度を優先しますが、モデル自体は同じ Opus 4.6 であり、知性や能力は変わりません
- **用途**: インタラクティブな作業（ライブデバッグや迅速な反復開発）に特に適しています
- **実験的機能**: このリリースは早期の実験的なもので、高速な推論に焦点を当てています
- **切り替え**: `/fast` コマンドでいつでもオン/オフを切り替えられます。必要な時だけ有効化してコストを最適化できます

## Opus 4.6 について

Claude Opus 4.6 は Anthropic の最も高性能なモデルで、コーディング、エンタープライズエージェント、プロフェッショナルワークに最適化されています。2026年2月5日にリリースされた最新モデルです。
