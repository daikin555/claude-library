# Claude Fable 5登場：Mythosクラスの最高性能モデルが一般利用可能に

## 原文（日本語に翻訳）

Claude Fable 5のご紹介：Anthropicがこれまでに一般提供してきたどのモデルをも上回る能力を持つ、安全性に配慮したMythosクラスのモデルです。v2.1.170にアップデートするとアクセスできます。

## 原文（英語）

Introducing Claude Fable 5: a Mythos-class model that we've made safe for general use. Fable's capabilities exceed those of any model we've ever made generally available. Update to version 2.1.170 for access.

## 概要

Claude Fable 5は、AnthropicがこれまでにリリースしたMythosクラスの中で初めて一般利用可能になったモデルです。推論能力・コーディング・長文理解など、あらゆる面で従来の最高性能モデルを上回ります。Claude Code v2.1.170にアップデートすることで即座に利用できます。

## 基本的な使い方

### モデルの切り替え

Claude Codeのモデルピッカーで `claude-fable-5` を選択するか、`/model` コマンドで切り替えます。

```
/model claude-fable-5
```

または起動時にフラグで指定：

```bash
claude --model claude-fable-5
```

### APIでの利用

```python
import anthropic

client = anthropic.Anthropic()

message = client.messages.create(
    model="claude-fable-5",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "このコードをレビューしてください。"}
    ]
)
print(message.content)
```

## 実践例

### 複雑なコードベースの解析

Fable 5の高度な推論能力を活用して、大規模なコードベースを深く分析できます。

```bash
# プロジェクト全体の設計上の問題を発見する
claude --model claude-fable-5 "このリポジトリのアーキテクチャ上の問題点と改善提案を包括的に教えてください"
```

### 長文ドキュメントの理解と要約

```python
client = anthropic.Anthropic()

with open("large_spec.md", "r") as f:
    content = f.read()

response = client.messages.create(
    model="claude-fable-5",
    max_tokens=4096,
    messages=[{
        "role": "user",
        "content": f"以下の仕様書を分析し、実装上の注意点を洗い出してください:\n\n{content}"
    }]
)
```

### 高度な数学・論理問題への応用

```python
response = client.messages.create(
    model="claude-fable-5",
    max_tokens=8192,
    messages=[{
        "role": "user",
        "content": "この分散システムにおけるコンセンサスアルゴリズムの正確性を証明してください。"
    }]
)
```

### Fast モードとの組み合わせ

Claude Code内で `/fast` コマンドを使うと、Fable 5をより高速な出力で利用できます。

```
/fast
```

## 注意点

- **v2.1.170以上が必要**: Fable 5を利用するには Claude Code を v2.1.170 にアップデートしてください。
- **モデルID**: APIから利用する場合のモデルIDは `claude-fable-5` です。
- **コスト**: Mythosクラスのモデルはより高い処理コストがかかる場合があります。利用量に応じて計画的に使用してください。
- **安全性への配慮**: Anthropicは Fable 5 を安全に一般利用できるよう設計しており、強力な能力と責任ある利用のバランスを取っています。
