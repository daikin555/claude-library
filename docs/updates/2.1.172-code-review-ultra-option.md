## 原文（日本語に翻訳）

`/code-review` で claude.ai にサインインしていない場合でも `ultra` オプションが表示されるようになりました（クラウドレビューにはclaude.aiアカウントが必要である旨の説明付き）

## 原文（英語）

`/code-review` now keeps the `ultra` option visible when you're not signed in to claude.ai, with an explanation that the cloud review requires a claude.ai account

## 概要

`/code-review` コマンドでコードレビューを実施する際、最高品質の `ultra` オプション（クラウドベースのレビュー）が claude.ai にサインインしていない場合でも選択肢として表示されるようになりました。サインインが必要であることが明示されるため、ユーザーは `ultra` の存在と要件を把握した上で利用方法を検討できます。

## 基本的な使い方

```
/code-review

# サインイン状態に関わらず、以下のオプションが表示される:
# > low    - 素早いレビュー
# > medium - 標準的なレビュー（推奨）
# > high   - 詳細なレビュー
# > ultra  - 最高精度のクラウドレビュー（claude.aiアカウントが必要）
```

## 実践例

### ultraオプションを利用する方法

```
/code-review
# ultraを選択
# → claude.aiアカウントへのサインインが促される

# サインイン後:
# クラウドで実行される最高品質のコードレビューが利用できる
```

### ローカルレビューとクラウドレビューの使い分け

| オプション | 実行場所 | 必要条件 | 適用シーン |
|-----------|---------|---------|-----------|
| low/medium | ローカル | なし | 素早いチェック |
| high | ローカル | なし | 詳細なローカルレビュー |
| ultra | クラウド | claude.aiアカウント | 重要なPRの最終確認 |

## 注意点

- `ultra` オプションはclaude.aiのアカウントが必要です（無料プランでも利用可能か要確認）
- クラウドレビューにはコードがAnthropicのサーバーに送信される点に注意してください
- 機密性の高いコードには `high` オプションの使用を検討してください
