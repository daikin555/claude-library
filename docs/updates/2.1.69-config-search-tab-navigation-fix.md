## 原文（日本語に翻訳）

`/config` の検索ボックスにフォーカスが当たっているが空の状態で、Tab/矢印キーで設定タブを切り替えられない問題を修正

## 原文（英語）

Fixed Tab/arrow keys not cycling Settings tabs when `/config` search box is focused but empty

## 概要

`/config` コマンドで設定画面を開いた際、検索ボックスにフォーカスが当たって空の状態では、Tab キーや矢印キーを使っても設定タブを切り替えることができない問題がありました。この修正により、検索ボックスが空の場合はキーボード操作で設定タブを切り替えられるようになります。

## 基本的な使い方

```
# 設定画面を開く
/config

# 修正後: 検索ボックスが空の状態でも
# Tab または矢印キーでタブを切り替えられる
↹ Tab  → 次のタブへ
← →    → 前後のタブへ移動
```

## 実践例

### キーボードのみで設定を変更

```
# 設定画面を開く
/config

# 検索せずに直接タブを切り替える
Tab        # 次のタブ（Model, Tools, Behavior, など）
← →        # タブ間を移動

# 目的のタブに到達したら設定を変更
↓          # リスト内を下に移動
Space      # 設定を切り替え
```

### 設定の素早い確認

```
/config

# 修正前: タブを切り替えるには検索ボックスからフォーカスを外す必要があった
# 修正後: そのままTab/矢印キーで素早くタブ移動できる

Tab → [Model タブ]    # モデル設定
Tab → [Tools タブ]    # ツール設定
Tab → [Memory タブ]   # メモリ設定
```

### マウスなしでの設定操作

```
# マウスなしでClaudeの設定を変更する場合
/config
Tab    # タブ移動
↑↓     # 選択肢の切り替え
Enter  # 選択/確定
Esc    # 設定を閉じる
```

## 注意点

- 検索ボックスに文字が入力されている場合は、Tab/矢印キーは通常の動作（テキスト編集）になります
- この修正は空の検索ボックスにフォーカスがある状態での動作改善です
- 設定変更は即座に反映されます
