## 原文（日本語に翻訳）

Claude in Chromeのツール読み込みを改善しました: ブラウザツールが1ツールずつではなく、1回のバッチ呼び出しで読み込まれるようになりました

## 原文（英語）

Improved Claude in Chrome tool loading: browser tools now load in a single batched call instead of one per tool

## 概要

Claude in Chrome拡張機能が提供するブラウザツール（ページ操作、スクリーンショット取得等）の読み込みが、ツールごとに個別のAPI呼び出しをしていた方式から、1回のバッチ呼び出しで全ツールを読み込む方式に変更されました。これにより、Claude in Chromeの起動時間と初回ツール使用時の遅延が大幅に短縮されます。

## 基本的な使い方

この改善は自動的に適用されます。Claude in Chrome拡張機能の起動がより速くなります。

```
# Claude in Chrome拡張機能をインストール後
# Chromeでページを開く
# 修正後: ブラウザツールが高速に読み込まれてすぐ使用可能になる
```

## 実践例

### Webスクレイピング・自動化での活用

```
# Claude in Chromeが高速に初期化されるため、
# 起動直後からブラウザ操作ツールを使用できる

claude
> このページのすべてのリンクを取得してください
# 修正後: ツール読み込み待ちなしで即座に実行される
```

### 複数タブでの並列作業

```
# 複数のブラウザタブでClaude in Chromeを使用する場合
# 修正後: 各タブでのツール初期化が高速化され、並列作業がよりスムーズ
```

## 注意点

- この改善はClaude in Chrome拡張機能を使用している場合のみ効果があります
- 通常のClaude Code CLI（ターミナル）には直接影響しません
- ツールの機能・動作に変化はありません
