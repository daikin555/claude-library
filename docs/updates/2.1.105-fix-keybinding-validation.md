## 原文（日本語に翻訳）

`keybindings.json`の不正なキーバインディングエントリ値が明確なエラーなしにサイレントに読み込まれていた問題を修正しました。

## 原文（英語）

Fixed malformed keybinding entry values in `keybindings.json` being silently loaded instead of rejected with a clear error

## 概要

`~/.claude/keybindings.json`に不正な形式のキーバインディング設定が含まれている場合、エラーなしにサイレントに読み込まれていた問題が修正されました。現在は不正な設定に対して明確なエラーメッセージが表示され、設定の問題を素早く特定して修正できるようになりました。

## 修正内容

```
Before: 不正なkeybindings.json → サイレントに読み込まれる（何が問題か不明）
After:  不正なkeybindings.json → 明確なエラーメッセージを表示
```

**エラーメッセージの例:**
```
エラー: keybindings.json のエントリが不正です:
  { "key": "ctrl+invalid" }  ← 無効なキーの組み合わせ
設定を確認して修正してください。
```

## 注意点

- `~/.claude/keybindings.json`の書式を確認するには`/doctor`コマンドも利用できます
- 有効なキーバインディング設定の例は公式ドキュメントを参照してください
