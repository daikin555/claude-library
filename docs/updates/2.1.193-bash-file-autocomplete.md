## 原文（日本語に翻訳）

Bashモード（`!`）にライブファイルパス自動補完を追加した。

## 原文（英語）

Added live file path autocomplete to bash mode (`!`)

## 概要

Claude Code のBashモード（`!` で始まる入力）で、ファイルパスのライブ自動補完がサポートされました。コマンドを入力しながらTabキーでファイルやディレクトリのパスが補完されるため、長いパスを手入力する手間が省けます。ターミナルと同じような感覚でファイルパスを入力できます。

## 基本的な使い方

Bashモードは `!` で始めることで利用できます:

```
! cat src/components/<Tab>
```

`<Tab>` を押すと `src/components/` 以下のファイル/ディレクトリが補完候補として表示されます。

## 実践例

### ファイルの参照

```bash
# ! の後にコマンドを入力し、パスでTabを押す
! cat docs/<Tab>
# → docs/README.md, docs/API.md, docs/guides/ などが候補に表示される

! cat docs/gu<Tab>
# → docs/guides/ に補完される
```

### 深いディレクトリ構造のナビゲート

```bash
! ls src/components/ui/<Tab>
# → Button/, Input/, Modal/, Select/ などが候補に表示される

! cp src/utils/helper.ts src/utils/helper-v2<Tab>
# → .ts などの拡張子候補が表示される
```

### 複数のファイルパスを含むコマンド

```bash
! diff <Tab>  # 最初のファイルを補完
! diff src/old.ts <Tab>  # 2番目のファイルを補完
```

### よく使うパターン

```bash
# ファイルを開く
! code <Tab>

# ファイルをコピー・移動
! cp src/<Tab> dist/<Tab>

# ファイルを検索・確認
! grep -r "TODO" src/<Tab>
```

## 注意点

- 自動補完はカレントディレクトリを基準とした相対パスで動作します
- 隠しファイル（`.` で始まるファイル）も補完候補に含まれます
- シェルの補完とは異なり、Claude Code のBashモード専用の補完機能です
- Bashモードは `!` で始めることでアクセスできます（`/bash` コマンドとは別機能）
