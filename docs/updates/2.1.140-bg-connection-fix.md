## 原文（日本語に翻訳）

バックグラウンドサービスがアイドル終了しようとしている際に`claude --bg`が「connection dropped mid-request」で失敗する問題を修正

## 原文（英語）

Fixed `claude --bg` failing with "connection dropped mid-request" when the background service was about to idle-exit

## 概要

バックグラウンドサービスがアイドルタイムアウトによる終了処理中に`claude --bg`でリクエストを送ると、「connection dropped mid-request」エラーが発生する問題が修正されました。バックグラウンドサービスへの接続タイミングによる失敗が解消されます。

## 影響を受けるシナリオ

```bash
# バックグラウンドタスクを起動しようとしたとき
claude --bg "長時間実行のタスクを実行して"

# エラー（修正前）:
# Error: connection dropped mid-request

# 修正後は正常に動作する
```

## 注意点

- アイドルタイムアウト後のバックグラウンドサービス再起動時の接続エラーが解消されました
- エンタープライズのエンドポイントセキュリティを持つ環境では、バックグラウンドサービスの起動に追加の時間が許容されるようになりました（同バージョンで修正）
- バックグラウンドサービスが完全に停止している場合は自動的に再起動されます
