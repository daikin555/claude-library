## 原文（日本語に翻訳）

- エージェントタスクの進捗が「Initializing…」のまま停止する問題を修正

## 原文（英語）

- Fixed agent task progress stuck on "Initializing…"

## 概要

Claude Codeでエージェントタスクを実行した際に、進捗表示が「Initializing…」のまま更新されず、実際にはタスクが進行しているにもかかわらず進捗が把握できない問題が修正されました。この修正により、エージェントタスクの実行状況がリアルタイムで正しく表示されるようになります。

## 基本的な使い方

エージェントタスク（Taskツール）を使用すると、進捗状況が正しく表示されます:

```
claude
> このプロジェクト全体のテストカバレッジを確認して
# エージェントタスクが起動し、進捗が正しく表示される
# 「Initializing…」→ 実際の進捗メッセージに更新される
```

## 実践例

### バックグラウンドタスクの進捗確認

バックグラウンドで実行中のエージェントタスクの状態が正しく表示されるようになりました:

```
claude
> バックグラウンドでこのモジュールのリファクタリングを進めて
# タスクの進捗が「Initializing…」で止まらず、正しく更新される
# 例: 「ファイルを読み取り中...」「テストを実行中...」など
```

### ワークツリーでのタスク実行

ワークツリー分離（isolation: "worktree"）を使ったタスクでも、進捗が正しく表示されます:

```
claude
> 新しいワークツリーでfeature-branchの実装を進めて
# ワークツリーの作成 → 実装作業の進捗が段階的に表示される
```

## 注意点

- この修正はUIの進捗表示に関するもので、タスクの実行ロジック自体は変更されていません
- 進捗表示はエージェントからの応答に基づいて更新されるため、応答間隔によっては一時的に更新が遅れる場合があります
- バックグラウンドタスクの完了通知は別途正しく送信されます
