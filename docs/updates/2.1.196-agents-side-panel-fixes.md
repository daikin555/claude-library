## 原文（日本語に翻訳）

`claude agents` サイドパネルに関する複数の問題を修正: エージェントを開いた際にキーボードフォーカスが固着する問題、バックグラウンドジョブのサブエージェントタイプがパネルを開くたびに失われる問題、実行中のセッションのステータスが正しく表示されない問題

## 原文（英語）

Fixed multiple `claude agents` side panel issues: keyboard focus getting stuck when opening an agent, background jobs losing their subagent types on every open, and sessions showing incorrect status while actively running

## 概要

`claude agents` コマンドで表示されるサイドパネル（agent view）に存在していた3つの独立した不具合が修正されました。具体的には、エージェント詳細を開いた際にキーボードフォーカスが固まって操作できなくなる問題、バックグラウンドジョブのサブエージェントタイプ表示がパネルを開き直すたびに消えてしまう問題、そして実行中のセッションのステータスが実際の状態と異なって表示される問題です。いずれもUI上の表示・操作に関する不具合で、`claude agents` を日常的に使ってバックグラウンドジョブを管理するユーザーの操作性と信頼性が向上します。

## 基本的な使い方

```bash
# バックグラウンドエージェント/ジョブの一覧をサイドパネルで表示
claude agents
```

サイドパネル内では、矢印キーでジョブ一覧を移動し、Enter などでエージェントの詳細を開閉します。今回の修正により、この一連の操作が安定して行えるようになりました。

## 実践例

### キーボードフォーカスの固着が解消される

```
修正前の問題シナリオ:
1. claude agents でサイドパネルを開く
2. 一覧から特定のエージェントを選択して詳細を開く
3. ❌ キーボードフォーカスが詳細ビューに固着し、↑↓キーで一覧に戻れない
4. パネルを閉じて開き直す必要があった

修正後の動作:
1. claude agents でサイドパネルを開く
2. エージェントの詳細を開く
3. ✅ ↑↓キーで一覧操作に戻ったり、Esc で詳細を閉じたりが正常に行える
4. 複数のエージェントを連続して確認できる
```

### バックグラウンドジョブのサブエージェントタイプが保持される

```
修正前の問題シナリオ:
1. code-reviewer などのサブエージェントタイプを指定してバックグラウンドジョブを起動
2. claude agents でサイドパネルを開くと、最初は "code-reviewer" と表示される
3. ❌ パネルを閉じて再度開くと、サブエージェントタイプの表示が失われ空欄になる
4. どの種類のエージェントを起動したか一覧から判別できなくなる

修正後の動作:
1. サブエージェントタイプを指定してバックグラウンドジョブを起動
2. claude agents を何度開き直しても "code-reviewer" などの種別が保持される
3. ✅ 一覧から各ジョブの役割を常に判別できる
```

### 実行中セッションのステータス表示が正確になる

```
修正前の問題シナリオ:
1. 長時間のバックグラウンドジョブ（テスト実行やビルドなど）を起動
2. claude agents で確認すると、実際には実行中なのに
   ❌ ステータスが "完了" や "待機中" のように誤って表示される
3. ジョブが本当に動いているのか判断できず、誤って再実行してしまう

修正後の動作:
1. バックグラウンドジョブを起動
2. claude agents で確認すると、実行中は常に "実行中" のステータスが正しく表示される
3. ✅ ジョブの進行状況を信頼して確認できる
```

### 複数ジョブを管理する際のワークフロー例

```bash
# 複数のサブエージェントタイプでバックグラウンドジョブを並行起動
claude --background "フロントエンドのコードレビューをして" # code-reviewer
claude --background "テストスイートを実行して"             # general-purpose

# サイドパネルで状態を確認
claude agents
# ✅ 各ジョブのサブエージェントタイプとステータスが正確に表示される
# ✅ 詳細を開いてもキーボード操作が固着しない
```

## 注意点

- 本修正はサイドパネルの表示・操作に関するものであり、バックグラウンドジョブ自体の実行ロジックには影響しません
- フォーカスの挙動に違和感がある場合は、`claude agents` を開き直すか最新バージョンへのアップデートを確認してください
- サブエージェントタイプが表示されない場合、ジョブ起動時にタイプが指定されていたかもあわせて確認してください
- ステータス表示は引き続きジョブの実行状況に依存するため、ネットワークやシステム負荷によっては反映に多少の遅延が生じる場合があります
