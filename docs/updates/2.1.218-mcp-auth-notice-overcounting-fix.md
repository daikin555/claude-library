## 原文（日本語に翻訳）

「N個のMCPサーバーが認証を必要としています」という起動時通知が、claude.ai上で接続されていないclaude.aiコネクタまで過剰にカウントしていた問題を修正しました

## 原文（英語）

Fixed the "N MCP servers need authentication" startup notice over-counting claude.ai connectors that aren't connected in claude.ai

## 概要

Claude Code起動時に表示される「N個のMCPサーバーが認証を必要としています」という通知が、ユーザーがclaude.ai側でそもそも接続していないclaude.ai管理のコネクタまでカウントに含めてしまい、実際よりも多い数を表示していました。この修正により、実際にユーザーの対応が必要なサーバーのみが件数に反映されるようになりました。

## 注意点

- Claude Code v2.1.218で修正されました
- claude.aiのMCPコネクタ連携を利用しているユーザーに影響する修正です
- 通知に表示される件数がより正確になり、不要な確認作業が減ります
