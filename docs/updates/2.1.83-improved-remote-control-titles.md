## 原文（日本語に翻訳）

Remote Control セッションタイトルを改善しました。AI が生成するタイトルが最初のメッセージから数秒で表示されるようになりました。

## 原文（英語）

Improved Remote Control session titles: AI-generated titles now appear within seconds of the first message

## 概要

Remote Control セッションで AI が自動生成するセッションタイトルが、最初のメッセージを送信してから数秒以内に表示されるようになりました。以前は遅延してから（または長時間後に）タイトルが表示されることがあり、複数のセッションを管理する際に識別が困難でしたが、本改善でより素早くセッションの内容を把握できるようになります。

## 基本的な使い方

Remote Control セッションを開始すると、すぐにセッションタイトルが生成されます。

```
# Web UI でのセッション一覧（改善後）
[ コードレビュー: auth.py の最適化 ] ← 数秒で表示
[ データベースマイグレーション作業  ] ← 数秒で表示
[ バグ修正: ログイン認証の問題      ] ← 数秒で表示
```

## 実践例

### ユースケース: 複数の Remote Control セッション管理

複数のセッションを同時に管理する際、タイトルがすぐに表示されて識別しやすくなります。

```bash
# セッション 1: コードレビュー用
$ claude --channels
> auth.py のコードをレビューしてください

# Web UI では数秒後に
# "コードレビュー: auth.py" のようなタイトルが表示される
```

### ユースケース: セッション履歴の参照

過去のセッションを探す際、タイトルで内容を判断しやすくなります。

```
セッション履歴:
2026-03-20 コードレビュー: 認証モジュールの最適化
2026-03-19 バグ修正: セッションタイムアウト問題
2026-03-18 機能追加: OAuth2 統合
# タイトルからセッションの内容を素早く把握できる
```

## 注意点

- タイトルは最初のメッセージ内容から AI が自動生成します
- タイトル生成はバックグラウンドで非同期に行われます
- 既存のセッションタイトルは変更されません
- セッションタイトルは手動で編集することも可能です
