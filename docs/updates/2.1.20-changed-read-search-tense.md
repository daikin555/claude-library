## 原文（日本語訳）
折りたたまれた読み込み/検索グループが、進行中は現在形（"Reading"、"Searching for"）を、完了後は過去形（"Read"、"Searched for"）を表示するように変更しました

## 原文（英語）
Changed collapsed read/search groups to show present tense ("Reading", "Searching for") while in progress, and past tense ("Read", "Searched for") when complete

## 概要
ファイル読み込みや検索操作の進行状況がより直感的に分かるよう、表示テキストの時制が動作の状態に応じて変化するようになりました。

## 基本的な使い方

時制の変化は自動的に反映されます。特別な操作は不要です。

## 実践例

### ファイル読み込み中

```bash
claude
> プロジェクト内のすべてのTypeScriptファイルを確認してください

📖 Reading 47 files...  # 進行中（現在形）
```

完了後：

```bash
✓ Read 47 files  # 完了（過去形）
```

### コードベース検索中

```bash
claude
> このエラーメッセージがどこで使われているか調べてください

🔍 Searching for "ConnectionError"...  # 進行中（現在形）
```

完了後：

```bash
✓ Searched for "ConnectionError" (15 matches)  # 完了（過去形）
```

### 複数の操作が並行実行

```bash
📖 Reading config files...
🔍 Searching for dependencies...
```

それぞれが完了するたびに過去形に変わります：

```bash
✓ Read config files
🔍 Searching for dependencies...  # まだ進行中
```

最終的にすべて完了：

```bash
✓ Read config files
✓ Searched for dependencies
```

## 注意点

- **視覚的フィードバック**: 現在進行中の操作と完了した操作を一目で区別できます
- **進捗の把握**: 大量のファイル操作でも、どれが完了しているか明確です
- **ユーザー体験**: より自然な言語表現により、状態が理解しやすくなります
- **折りたたみ表示**: 複数のファイル操作は自動的にグループ化され、完了後は過去形で要約されます
