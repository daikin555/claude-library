## 原文（日本語に翻訳）

`claude update` がダウンロード開始前にターゲットバージョンを通知するようになりました（以前はサイレントでした）。

## 原文（英語）

`claude update` now announces the target version before downloading instead of going silent

## 概要

`claude update` コマンドを実行した際、ダウンロードを開始する前にアップデート先のバージョン番号が表示されるようになりました。以前はダウンロード中に何も表示されず、何が起きているか分かりにくい状態でしたが、改善によりどのバージョンにアップデートされるかが事前に確認できます。

## 基本的な使い方

```bash
claude update
```

実行すると、以下のような出力が表示されます：

```
Updating to v2.1.166...
Downloading...
Update complete.
```

## 実践例

### アップデート前の確認フロー

アップデート内容を確認してから適用する場合：

```bash
# 現在のバージョン確認
claude --version

# アップデートを実行（ターゲットバージョンが表示される）
claude update

# アップデート後のバージョン確認
claude --version
```

### スクリプトでのバージョン管理

CI/CD や自動化スクリプトでのアップデート確認：

```bash
#!/bin/bash
echo "アップデート前のバージョン: $(claude --version)"
claude update
echo "アップデート後のバージョン: $(claude --version)"
```

## 注意点

- 表示されるバージョン番号はダウンロード前に確定するため、ダウンロードの失敗などによって実際のインストールバージョンと異なる場合があります
- アップデートが利用可能でない場合は、既に最新バージョンである旨が表示されます
