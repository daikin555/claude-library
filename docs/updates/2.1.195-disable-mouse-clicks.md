## 原文（日本語に翻訳）

フルスクリーンモードでマウスクリック・ドラッグ・ホバーを無効化しつつ、ホイールスクロールは維持する `CLAUDE_CODE_DISABLE_MOUSE_CLICKS` を追加

## 原文（英語）

Added `CLAUDE_CODE_DISABLE_MOUSE_CLICKS` to disable mouse click/drag/hover in fullscreen mode while keeping wheel scroll

## 概要

フルスクリーンモード時に、マウスクリック・ドラッグ・ホバーイベントを無効化する環境変数 `CLAUDE_CODE_DISABLE_MOUSE_CLICKS` が追加されました。この設定を有効にしても、マウスホイールによるスクロールは引き続き利用できます。tmux や screen などのターミナルマルチプレクサ環境で Claude Code を使用する際の干渉問題を解消するのに役立ちます。

## 基本的な使い方

環境変数を設定するだけで有効になります。

```bash
# シェルの設定ファイル（~/.bashrc, ~/.zshrc など）に追加
export CLAUDE_CODE_DISABLE_MOUSE_CLICKS=1

# または一時的に有効化
CLAUDE_CODE_DISABLE_MOUSE_CLICKS=1 claude
```

## 実践例

### tmux / screen 環境での使用

tmux や screen のマウスモードと Claude Code のマウスイベントが競合する場合に有効です。

```bash
# tmux の設定でマウスを有効にしつつ Claude Code のクリックイベントを抑制
export CLAUDE_CODE_DISABLE_MOUSE_CLICKS=1
tmux new-session -s main
claude
```

### SSH リモート接続時の使用

SSH 越しのリモート環境でフルスクリーンモードを使用する際、マウスイベントが意図せず送信される問題を防げます。

```bash
# リモートサーバー側の .bashrc に設定
echo 'export CLAUDE_CODE_DISABLE_MOUSE_CLICKS=1' >> ~/.bashrc
```

### テキスト選択を優先する場合

フルスクリーンモードでも通常のターミナルと同様にテキストをマウスで選択したい場合に便利です。

## 注意点

- マウスホイールによるスクロールは **無効化されません**（スクロールは引き続き動作します）
- フルスクリーンモード（`--fullscreen` フラグ使用時）に適用されます
- 通常のターミナルモードでは影響を受けません
- この環境変数を設定すると、クリック・ドラッグ・ホバーイベントがすべて無視されます
