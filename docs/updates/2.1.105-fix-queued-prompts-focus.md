## 原文（日本語に翻訳）

フォーカスモードでキューに入れたユーザープロンプトが消えてしまう問題を修正しました。

## 原文（英語）

Fixed queued user prompts disappearing from focus mode

## 概要

フォーカスモード（Claudeが処理中の状態）でキューに入れたメッセージが、Claudeが作業を完了した後に表示されず、消えてしまう問題が修正されました。キューに入れたプロンプトが確実に処理されるようになりました。

## 修正内容

```
Before: フォーカスモード中にプロンプト送信 → Claudeの処理完了後にプロンプトが消える
After:  フォーカスモード中にプロンプト送信 → Claudeの処理完了後に正しく処理される
```
