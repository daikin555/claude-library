## 原文（日本語に翻訳）

リモートコントロールのフッターインジケーターを「/rc active」に短縮し、狭いターミナルでは非表示にするようになりました

## 原文（英語）

Shortened the Remote Control footer indicator to "/rc active" and hid it on narrow terminals

## 概要

リモートコントロール（`/rc`）が有効な際のターミナルフッターに表示されるインジケーターが、より短い「`/rc active`」に簡略化されました。また、ターミナル幅が狭い場合（モバイル端末や小さい分割ペイン等）には自動的に非表示になり、作業領域を確保できるようになりました。

## 基本的な使い方

```
# リモートコントロールを有効化
/rc

# フッター表示
# 修正前: "Remote Control is active" など長い表示
# 修正後: "/rc active" とコンパクトに表示
#         狭いターミナル幅では自動非表示
```

## 実践例

### 分割ペインでの作業

```
# ターミナルを複数ペインに分割して作業する場合
# 狭いペインでも作業領域が確保される
┌────────────────────┬──────────────┐
│ Claude Code        │ エディタ     │
│ ...                │              │
│ /rc active         │              │  ← 幅が十分な場合は表示
└────────────────────┴──────────────┘

# 狭いペインの場合:
┌──────────┬──────────────────────┐
│ Claude   │ エディタ             │
│ ...      │                      │
│          │                      │  ← 幅が足りない場合は非表示
└──────────┴──────────────────────┘
```

## 注意点

- リモートコントロール機能自体の動作に変更はありません
- インジケーターが非表示の場合でも、`/rc` は有効に動作しています
- `/status` コマンドでリモートコントロールの状態を確認できます
