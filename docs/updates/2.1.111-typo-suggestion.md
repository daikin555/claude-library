## 原文（日本語に翻訳）

- 近いタイポで `claude <word>` を実行したとき、最も近いサブコマンドを提案するようになりました（例: `claude udpate` → 「`claude update` ではありませんか？」）

## 原文（英語）

- Suggest the closest matching subcommand when `claude <word>` is invoked with a near-miss typo (e.g. `claude udpate` → "Did you mean `claude update`?")

## 概要

Claude Code のコマンドをタイポして入力した場合に、最も近い正しいコマンドを自動的に提案するようになりました。毎回ヘルプを確認したり正しいスペルを検索したりする手間がなくなります。

## 基本的な使い方

### タイポの例

```bash
$ claude udpate
# "Did you mean `claude update`?"（`claude update` ではありませんか？）と表示

$ claude upgade
# "Did you mean `claude upgrade`?" と表示

$ claude instll
# "Did you mean `claude install`?" と表示
```

## 実践例

### よくあるタイポ

| 入力したコマンド | 提案 |
|---------------|------|
| `claude udpate` | `claude update` |
| `claude upgade` | `claude upgrade` |
| `claude instll` | `claude install` |
| `claude logut` | `claude logout` |

### 提案に従って実行

```bash
$ claude udpate
Error: unknown command "udpate"
Did you mean "update"?

# 正しいコマンドで再実行
$ claude update
```

## 注意点

- 提案はあくまで候補であり、自動実行はされません
- 完全に無関係なコマンドを入力した場合は、提案が表示されない場合があります
- `claude --help` で利用可能なコマンド一覧を確認できます
