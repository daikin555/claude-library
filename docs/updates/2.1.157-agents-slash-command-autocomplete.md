## 原文（日本語に翻訳）

`claude agents`: ディスパッチ入力でのスラッシュコマンドオートコンプリートが、部分文字列（サブストリング）の一致もサポートするようになりました。

## 原文（英語）

`claude agents`: slash-command autocomplete in the dispatch input now matches substrings

## 概要

`claude agents` のディスパッチ入力において、スラッシュコマンドのオートコンプリート機能が強化されました。これまでは先頭からの文字列一致のみサポートしていましたが、コマンド名の途中の文字列でも一致するようになりました。より少ない入力でコマンドを見つけやすくなり、作業効率が向上します。

## 基本的な使い方

`claude agents` のディスパッチ入力で `/` を入力してコマンド名の一部を入力すると、その文字列を含むすべてのコマンドが候補として表示されます。

例えば、`/review` というコマンドがある場合、`/view` や `/rev` などの部分文字列でも候補として表示されるようになります。

## 注意点

- この機能は `claude agents` のディスパッチ入力に限定された改善です
- 通常のCLIセッションのスラッシュコマンド補完にも同様の動作が期待されますが、主に agentsモードでの改善として提供されています
- v2.1.157 以降で自動的に適用されます
