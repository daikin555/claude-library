# 修正：ターミナルタブタイトルが自動生成されたセッション説明で更新されない問題

## 原文（日本語に翻訳）

自動生成されたセッション説明でターミナルタブタイトルが更新されない問題を修正。

## 原文（英語）

Fixed terminal tab title not updating with an auto-generated session description

## 概要

Claude Code がセッションの内容から自動生成するセッション説明が、ターミナルのタブタイトルに反映されない問題が修正されました。複数のターミナルタブでClaude Codeを使っている場合、タブタイトルからどのセッションで何をしているかを把握しやすくなります。

## 基本的な使い方

修正後は、セッションが進むとターミナルのタブタイトルが自動的に更新されます：

```
# 最初のタブタイトル
Claude Code

# セッションが進むと自動的に更新
Claude Code - Pythonコードのリファクタリング

# 別のタブ
Claude Code - テストの修正
```

## 実践例

### ### 複数タブでの作業管理

複数のターミナルタブでClaude Codeを同時使用する場合：

```
タブ1: Claude Code - フロントエンドのバグ修正
タブ2: Claude Code - APIドキュメント生成
タブ3: Claude Code - データベーススキーマ設計
```

各タブのタイトルから作業内容が一目でわかるようになります。

## 注意点

- タブタイトルの更新はターミナルエミュレータが対応している必要がある（iTerm2、WezTerm、Ghostty等）
- セッション説明は最初のプロンプトの後しばらくして自動生成される
- `/rename` コマンドで手動でセッション名を変更することも可能
