## 原文（日本語に翻訳）

`/terminal-setup` コマンドで VS Code・Cursor・Windsurf の統合ターミナルの GPU アクセラレーションを無効化し、テキストのレンダリング文字化けを防ぐようになりました。

## 原文（英語）

`/terminal-setup` now disables GPU acceleration in VS Code/Cursor/Windsurf integrated terminals to prevent garbled-text rendering

## 概要

VS Code、Cursor、Windsurf の統合ターミナルでGPUアクセラレーションが有効になっている場合、Claude Codeのテキスト表示が文字化けすることがありました。`/terminal-setup` コマンドがこの問題を自動的に解決するため、実行時にこれらのエディタの統合ターミナルにおけるGPUアクセラレーションを無効化する設定を適用するようになりました。

## 基本的な使い方

Claude Codeのターミナル内で以下のコマンドを実行するだけです:

```
/terminal-setup
```

このコマンドを実行すると、使用しているエディタ（VS Code・Cursor・Windsurf）が検出され、統合ターミナルのGPUアクセラレーション無効化を含む最適な設定が自動的に適用されます。

## 注意点

- GPU アクセラレーションの無効化は統合ターミナルのみに適用され、エディタ全体のGPU設定には影響しません
- `/terminal-setup` を一度実行すれば、以降は設定が維持されます
- すでに `/terminal-setup` を過去に実行している場合は、再度実行することで最新の設定（GPUアクセラレーション無効化を含む）が適用されます
- テキストの文字化けが発生していない場合でも、予防的に `/terminal-setup` を実行しておくことを推奨します
