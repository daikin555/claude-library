## 原文（日本語に翻訳）

サブエージェントのチャットタブが開いている間に上矢印キーでプロンプト履歴を表示するとメインエージェントのプロンプトが表示される問題を修正しました

## 原文（英語）

Fixed up-arrow prompt history showing the main agent's prompts while a subagent's chat tab is open

## 概要

サブエージェントのチャットタブが開いている状態で上矢印キー（↑）を押してプロンプト履歴を遡ると、サブエージェントの履歴ではなくメインエージェントの履歴が誤って表示されていた問題が修正されました。今後は、アクティブなタブ（サブエージェント/メインエージェント）に対応した正しい履歴が表示されます。

## 基本的な使い方

```
# サブエージェントのチャットタブを開いている場合
# 修正前: ↑キーを押すとメインエージェントの履歴が表示されていた
# 修正後: ↑キーを押すとそのサブエージェントの履歴が正しく表示される
```

## 実践例

### マルチエージェント作業時の操作

```
# メインエージェントとサブエージェントを並行して操作する場合

[メインエージェント タブ]
> コードを書いてください
> テストを実行してください

[サブエージェントA タブ] ← このタブが開いている
> ファイルを確認してください
# ↑キーを押す
# 修正後: "ファイルを確認してください" がサブエージェントAの履歴から表示される
# 修正前: メインエージェントの "コードを書いてください" が表示されていた
```

## 注意点

- この修正はUI操作性の問題であり、エージェントの動作には影響しません
- 各タブは独立した履歴を持つようになりました
