## 原文（日本語に翻訳）

会話にメッセージがない場合や、同じ基礎値に解決される effort レベル間での切り替え時に effort 変更確認ダイアログが表示される問題を修正しました。

## 原文（英語）

Fixed the effort-change confirmation dialog appearing when the conversation has no messages or when switching between effort levels that resolve to the same underlying value

## 概要

effort レベル（思考量の設定）を変更する際に表示される確認ダイアログが、不必要な状況で表示されていた2つのケースが修正されました。（1）会話にまだメッセージがない状態でのレベル変更、（2）異なるラベルでも実際には同じ基礎設定値に対応する effort レベル間の切り替え。v2.1.152 以降は、本当に変更が必要な場合にのみ確認ダイアログが表示されます。

## 影響を受けていた状況

- セッション開始直後（会話履歴なし）に effort レベルを変更しようとした場合
- 実質的に同じ処理量に対応する異なる effort レベル名の間を切り替えた場合

## 注意点

- v2.1.152 以降では自動的に修正されており、特別な操作は不要です
