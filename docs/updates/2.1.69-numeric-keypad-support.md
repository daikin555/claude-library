## 原文（日本語に翻訳）

Claudeのインタビュー質問で選択肢を選ぶ際、テンキーの数字キーもサポートするようになりました（以前はQWERTYキーボード上部の数字行のみ動作していました）

## 原文（英語）

Added numeric keypad support for selecting options in Claude's interview questions (previously only the number row above QWERTY worked)

## 概要

Claude Codeがインタビュー形式で質問を出す際、選択肢を番号で選ぶ操作にテンキー（NumPad）の数字キーが使えるようになりました。これまでQWERTYキーボード上部の数字行しか認識されなかったため、テンキー専用のキーボードや外付けテンキーを利用しているユーザーが不便を感じていました。本修正により、あらゆるキーボードレイアウトで一貫した操作が可能になります。

## 基本的な使い方

インタビュー質問が表示された際、テンキーの数字を押して選択肢を選べます。

```
Claudeからの質問:
  どのフレームワークを使用しますか？
  1) React
  2) Vue
  3) Angular

[テンキーの 1] を押す → React を選択
```

## 実践例

### テンキー付き外付けキーボードでの操作

```bash
# Claude Codeを起動してプロジェクト初期化を依頼
claude

> 新しいWebアプリのプロジェクトをセットアップして

# Claude がインタビュー質問を出す
# 選択肢が表示されたら、テンキーで回答可能に
# NumPad 1, 2, 3... で選択肢を選べる
```

### ラップトップ + 外付けテンキーの構成

```bash
# テンキーを接続したラップトップでも
# インタビュー選択肢の回答にテンキーが使えるようになった
# NumLock が有効な状態であれば動作する
```

### キーボードショートカットの整理

| キー | 動作 |
|------|------|
| 上部数字行 (1-9) | 選択肢を選ぶ（従来から対応） |
| テンキー (NumPad 1-9) | 選択肢を選ぶ（今回追加） |

## 注意点

- NumLock が有効になっている必要があります（無効の場合はカーソルキーとして動作するため、数字入力にはなりません）
- テンキーの `0` は選択肢の番号には使用されません（選択肢は通常 1 始まりです）
- この対応はインタビュー形式の選択肢専用です。通常のテキスト入力では以前から両方使用可能でした
