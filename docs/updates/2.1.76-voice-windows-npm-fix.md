# Windows npm環境でのボイスモード動作不良の修正

## 原文（日本語に翻訳）

npm経由でインストールした場合にWindowsで `/voice` が動作しない問題を修正

## 原文（英語）

Fixed `/voice` not working on Windows when installed via npm

## 概要

Claude Code v2.1.76で、Windows環境においてnpm経由でインストールした場合に `/voice` コマンドが動作しない問題が修正されました。ネイティブインストーラーやその他のインストール方法では問題なく動作していましたが、`npm install -g @anthropic-ai/claude-code` でインストールした場合にのみボイスモードが起動できない状態でした。この修正により、Windowsのすべてのインストール方法でボイスモードが利用可能になりました。

## 基本的な使い方

修正後は、npm経由でインストールしたWindows環境でも `/voice` が正常に動作します：

```bash
# npmでClaude Codeをインストール（Windows）
npm install -g @anthropic-ai/claude-code

# Claude Codeを起動
claude

# ボイスモードを起動（修正後は正常に動作）
/voice
```

## 実践例

### Windows環境でのボイスモード利用

```bash
# Git Bash または PowerShell で Claude Code を起動
claude

# ボイスモードを有効化
/voice

# 音声で指示を出す
「プロジェクトの構造を説明して」
「このバグを修正して」
「テストを書いて」
```

### npmグローバルインストールの確認

現在のインストール方法を確認する場合：

```bash
# インストール元の確認
claude --version

# npmでインストールされているか確認
npm list -g @anthropic-ai/claude-code

# 必要に応じてアップデート
npm update -g @anthropic-ai/claude-code
```

### ネイティブインストーラーへの移行（任意）

npm以外のインストール方法を使いたい場合：

```powershell
# npmパッケージをアンインストール
npm uninstall -g @anthropic-ai/claude-code

# ネイティブインストーラーを使用（PowerShell）
irm https://claude.ai/install.ps1 | iex

# または WinGet を使用
winget install Anthropic.ClaudeCode
```

## 注意点

- **影響範囲**: この問題はWindows + npm環境の組み合わせでのみ発生していました。macOSやLinuxでのnpmインストールには影響ありません
- **Git Bashの必要性**: Windows環境ではGit for Windows（Git Bash含む）が必要です。インストールされていない場合は [git-scm.com](https://git-scm.com/downloads/win) からインストールしてください
- **マイクの権限**: ボイスモードの利用にはマイクへのアクセス許可が必要です。Windowsの設定でマイクのアクセスが許可されていることを確認してください
- **推奨インストール方法**: Windows環境では、ネイティブインストーラー（`irm https://claude.ai/install.ps1 | iex`）または WinGet の利用も推奨されています
