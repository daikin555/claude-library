## 原文（日本語に翻訳）

現在の認証設定で利用できないスラッシュコマンド（`/voice`、`/mobile`、`/chrome`、`/upgrade` など）が表示されなくなりました。

## 原文（英語）

Slash commands unavailable for the current auth setup (`/voice`, `/mobile`, `/chrome`, `/upgrade`, etc.) are now hidden instead of shown

## 概要

Claude Codeのスラッシュコマンドメニューに、現在の認証プランや環境では使用できないコマンドが表示されていた問題が改善されました。利用できないコマンドは非表示になり、実際に使用可能なコマンドのみが表示されるようになります。これによりUIがすっきりし、混乱が減ります。

## 基本的な使い方

変更は自動的に適用されます。`/` を入力してコマンド一覧を開くと、現在の環境・プランで利用可能なコマンドのみが表示されます。

```
/ を入力すると...

# 変更前（使用できないコマンドも表示されていた）
/voice    ← このプランでは使用不可
/mobile   ← このプランでは使用不可
/chrome   ← このプランでは使用不可
/upgrade  ← このプランでは使用不可
/help
/clear
...

# 変更後（使用可能なコマンドのみ表示）
/help
/clear
/compact
...
```

## 実践例

### プランと表示されるコマンドの対応

```
Claude.ai Pro プラン:
  /voice - 音声入力（サポート対象なら表示）
  /help, /clear, /compact など基本コマンド

Claude.ai Free プラン:
  /upgrade - 表示される場合あり
  /help, /clear など基本コマンド

企業向け / API キー直接使用:
  /help, /clear, /compact, /model など
  /voice, /mobile, /chrome は環境に応じて非表示
```

### コマンド検索での活用

```bash
# / を入力して検索
/mo → /model, /mobile（使用可能な場合）が表示される
/v  → /voice（使用可能な場合）、/version が表示される
```

## 注意点

- この変更は**表示の最適化**であり、機能のアクセス権限には影響しません
- 特定のコマンドが表示されない場合は、現在のプランや認証設定では利用できないことを示しています
- アップグレードにより利用可能なコマンドが増える場合があります
- `/help` コマンドで利用可能なすべてのコマンドを確認できます
