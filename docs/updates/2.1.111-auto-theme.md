## 原文（日本語に翻訳）

- ターミナルのダーク/ライトモードに合わせる「Auto（ターミナルに合わせる）」テーマオプションを追加 — `/theme` から選択可能

## 原文（英語）

- Added "Auto (match terminal)" theme option that matches your terminal's dark/light mode — select it from `/theme`

## 概要

Claude Code に新しいテーマオプション「Auto (match terminal)」が追加されました。ターミナルのダーク/ライトモード設定を自動的に検出し、Claude Code の表示テーマをそれに合わせます。OS やターミナルのテーマを変更すると Claude Code も自動的に追従します。

## 基本的な使い方

### テーマを設定する

```
/theme
```

テーマ選択メニューから「Auto (match terminal)」を選択します。

### 選択肢

| テーマ | 説明 |
|--------|------|
| `dark` | 常にダークテーマ |
| `light` | 常にライトテーマ |
| `auto` | **ターミナルのモードに自動追従（新機能）** |

## 実践例

### システム全体でテーマを統一する

```
/theme
# → "Auto (match terminal)" を選択
```

macOS の「システム設定 > 外観」でダーク/ライトモードを切り替えると、Claude Code も自動で追従します。

### 時間帯に合わせた自動切り替え

macOS の「自動」外観設定と組み合わせることで、夜間は自動的にダークモード、日中はライトモードに切り替わります。

```
# ターミナルも自動テーマ対応のものを使用（例: iTerm2, Alacritty）
# /theme で "Auto (match terminal)" を設定するだけで完了
/theme
```

### 設定ファイルで直接指定

`~/.claude/settings.json` で設定する場合:

```json
{
  "theme": "auto"
}
```

## 注意点

- この機能はターミナルがダーク/ライトモードの情報を提供している場合のみ機能します（ANSI エスケープシーケンスによる検出）
- 一部の古いターミナルでは自動検出が機能しない場合があります
- 自動検出が失敗した場合は、デフォルトのテーマにフォールバックします
