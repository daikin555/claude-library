# VSCode：リモートセッションが会話履歴に表示されない問題を修正

## 原文（日本語に翻訳）

VSCode：リモートセッションが会話履歴に表示されない問題を修正しました。

## 原文（英語）

VSCode: Fixed remote sessions not appearing in conversation history

## 概要

Claude Code VSCode拡張機能のv2.1.63で、リモートセッション（Remote SSH、WSL、Dev Containersなど）が会話履歴パネルに表示されない問題が修正されました。これにより、リモート環境でのClaude Code使用時も、ローカル環境と同様に過去の会話履歴にアクセスできるようになりました。

## 基本的な使い方

この修正はバックグラウンドで自動的に適用されます。ユーザー側での操作は不要です。

## 実践例

### 影響を受けるシナリオ

- VS Code Remote SSH で接続したリモートサーバーでClaude Codeを使用する場合
- WSL（Windows Subsystem for Linux）環境でClaude Codeを使用する場合
- VS Code Dev Containers でClaude Codeを使用する場合

### 修正後の改善

```
# VSCode Remote SSH での使用例
ssh user@remote-server
↓
VSCode Remote SSH で接続
↓
Claude Codeを使用（会話が発生）
↓
修正前：会話履歴パネルに表示されない
修正後：ローカルセッションと同様に会話履歴に表示される
```

## 注意点

- この修正はVSCode拡張機能専用です
- v2.1.63にアップデートすることで自動的に修正が適用されます
- リモート開発環境でClaude Codeを活用するユーザーに特に効果があります
