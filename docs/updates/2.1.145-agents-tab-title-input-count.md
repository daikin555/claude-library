## 原文（日本語に翻訳）

`claude agents` のターミナルタブタイトルに入力待ちカウントが表示されるようになりました。これにより別ウィンドウに切り替えていても、エージェントが操作を必要としているときにすぐわかります

## 原文（英語）

`claude agents` terminal tab title now shows the awaiting-input count so an alt-tabbed window tells you when an agent needs attention

## 概要

Claude Code v2.1.145から、`claude agents` コマンドを実行しているターミナルのタブタイトルに、入力待ちのエージェント数が表示されるようになりました。複数のターミナルウィンドウや作業を並行して進めている場合でも、別ウィンドウに切り替えることなく、Claudeがユーザーの確認や入力を必要としているかどうかをタブタイトルから即座に確認できます。

## 基本的な使い方

`claude agents` を起動すると、ターミナルのタブタイトルが自動的に更新されます。

```bash
# Claude Agentsを起動
claude agents

# タブタイトルが変化する:
# 通常時:  "claude agents"
# 1件待ち: "claude agents (1 awaiting input)"
# 3件待ち: "claude agents (3 awaiting input)"
```

これにより、別のウィンドウで作業中でも（Alt+Tab などで切り替えた状態でも）タブバーを見るだけでエージェントの状態が分かります。

## 実践例

### 複数タスクを並行処理する際の管理

複数のタスクをエージェントに並行して任せながら別の作業をする際の活用例です。

```bash
# ターミナルウィンドウ1: claude agentsを起動
claude agents

# 複数タスクをバックグラウンドで起動
# タスク1: コードレビュー
# タスク2: テスト実行
# タスク3: ドキュメント生成

# 別のウィンドウで作業中でも...
# タブに "(2 awaiting input)" と表示されたら
# → エージェントが確認待ちになっていることがわかる
```

### tmuxでの複数セッション管理

tmuxのウィンドウ名にも反映されるため、tmuxユーザーにとって特に便利です。

```bash
# tmuxセッションを作成
tmux new-session -d -s work

# ウィンドウ1でclaude agentsを起動
tmux send-keys -t work 'claude agents' Enter

# ウィンドウ2で別の作業
tmux new-window -t work
tmux send-keys -t work 'vim myfile.py' Enter

# tmuxのステータスバーでウィンドウ一覧を見ると:
# 0: work  1: claude agents (3 awaiting)  ...
```

### 通知ベースの作業フロー

入力待ちカウントを確認することで、割り込みなしに集中して作業できます。

```bash
# 長時間のタスクをエージェントに依頼
# "大規模なリファクタリングをお願いします"

# エージェントのウィンドウを最小化して他の作業に集中
# 定期的にタブバーを確認:
#   "(0 awaiting input)" → まだ作業中、引き続き他の作業を続けられる
#   "(1 awaiting input)" → エージェントが確認を求めている、切り替えて対応

# このサイクルにより、不必要な確認を減らしながら
# エージェントのニーズに素早く応答できる
```

### 複数モニター環境での活用

```bash
# セカンドモニターにclaude agentsウィンドウを配置
# タスクバーやdockのタイトルで確認可能

# Windowsの場合: タスクバーのプレビューでカウントを確認
# macOSの場合: Dockのウィンドウタイトルで確認
# Linux(GNOME等): アクティビティバーやタスクバーで確認
```

## 注意点

- タブタイトルの更新はターミナルエミュレーターがタイトル変更のエスケープシーケンスをサポートしている必要があります（多くの主要ターミナルは対応済み）
- タブタイトルをカスタマイズしているターミナル設定（`PROMPT_COMMAND` でタイトルを上書きする設定など）が有効な場合、この機能が正しく機能しない場合があります
- tmuxを使用している場合は、tmuxの `set-titles` および `set-titles-string` 設定が有効になっている必要があります
- 「入力待ち」カウントは、ユーザーの明示的な確認や入力を必要とするエージェントの数を表します
