## 原文（日本語に翻訳）

Autoモードがプラン制限で無効になっている場合、「一時的に利用不可」の代わりに「お客様のプランでは利用不可」と表示されるようになりました。

## 原文（英語）

Auto mode now shows "unavailable for your plan" when disabled by plan restrictions (was "temporarily unavailable")

## 概要

Claude Codeのモデル選択でAutoモードを選ぼうとした際、ユーザーが使用しているサブスクリプションプランでAutoモードが利用できない場合、以前は「一時的に利用不可（temporarily unavailable）」という誤解を招くメッセージが表示されていました。修正後は「お客様のプランでは利用不可（unavailable for your plan）」という正確なメッセージが表示され、ユーザーが状況を正しく理解できるようになります。

## 基本的な使い方

Autoモードを選択しようとした際に、プラン制限でアクセスできない場合：

修正前：
```
Autoモード: 一時的に利用不可
```

修正後：
```
Autoモード: お客様のプランでは利用不可
```

## 実践例

### ユースケース: 利用可能なモデルの確認

`/model` コマンドでモデル一覧を確認：

```
/model
```

プラン制限でAutoモードが使えない場合、「お客様のプランでは利用不可」と表示されることで、プランのアップグレードが必要だとすぐに判断できます。

### ユースケース: プランに応じたモデル選択の判断

「一時的に利用不可」というメッセージだと、しばらく待てば使えると誤解しますが、「プランでは利用不可」と表示されることで、適切なプランへのアップグレードを検討できます。

## 注意点

- この変更はUIメッセージの改善であり、機能的な変更はありません
- Autoモードが「一時的に利用不可」と表示される場合は、サービスの問題ではなくプランの制限です
- Autoモードを使用するには、対応するサブスクリプションプランが必要です
- Autoモードはモデルを自動的に最適なものに切り替える機能です
