## 原文（日本語に翻訳）

現在のプランでエージェントチームが利用できないという警告の表示を修正しました。

## 原文（英語）

Fixed warnings about agent teams not being available on your current plan

## 概要

エージェントチーム機能が利用できないプランでClaude Codeを使用する際、警告メッセージの表示が不適切だった問題が修正されました。これにより、ユーザーは自分のプランでエージェントチームが利用可能かどうかを正確に把握できるようになります。

## 基本的な使い方

この修正により、エージェントチーム機能を利用できないプランの場合、適切な警告メッセージが表示されるようになりました。

```bash
# エージェントチームを有効化しようとした場合
export CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1
claude
```

プランでサポートされていない場合、明確なメッセージが表示されるようになりました。

## 実践例

### プラン確認

エージェントチーム機能は現在研究プレビュー段階で、特定のプランでのみ利用可能です。

```bash
# 環境変数を設定してエージェントチームを試す
export CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1
claude

# プランで利用できない場合、明確な警告が表示される
```

### エンタープライズプランでの利用

エージェントチーム機能は現在研究プレビュー段階で、特定のプラン（エンタープライズプランなど）でのみ利用可能です。プラン情報の詳細については、Anthropicの公式サイトで確認できます。

## 注意点

- エージェントチーム機能は現在研究プレビュー段階です
- 利用には環境変数 `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1` の設定が必要です
- プランによっては利用できない場合があります
- この修正により、警告メッセージがより明確になりました
