## 原文（日本語に翻訳）

`/usage` がProとEnterpriseプランでは不要な「現在の週（Sonnetのみ）」バーを非表示にするようになりました。

## 原文（英語）

`/usage` now hides redundant "Current week (Sonnet only)" bar for Pro and Enterprise plans

## 概要

Claude Codeの `/usage` コマンドで使用量を確認する際、ProおよびEnterpriseプランのユーザーには関係のない「現在の週（Sonnetのみ）」という使用量バーが表示されていました。このバーは無料プランのユーザーがSonnet限定の使用量を確認するためのものですが、ProとEnterpriseプランではSonnet以外のモデルも利用できるため、混乱を招いていました。v2.1.89でプランに応じた適切な表示に改善されました。

## 基本的な使い方

```bash
# 使用量の確認
/usage

# ProまたはEnterpriseプランでの表示（修正後）
# ┌─────────────────────────────────────────────────┐
# │ Usage Statistics                                 │
# │                                                  │
# │ This month: 45,230 tokens                       │
# │ [██████░░░░░░░░░] 45%                          │
# │                                                  │
# │ ← "Current week (Sonnet only)" バーは非表示    │
# └─────────────────────────────────────────────────┘
```

## 実践例

### /usageコマンドの使い方

```bash
# 現在の使用量を確認する
/usage

# ProプランとEnterpriseプランでの表示内容（修正後）:
# - 今月の合計トークン使用量
# - 使用可能な残量
# - 関連するモデルの使用状況
# 
# 表示されなくなるもの:
# - "Current week (Sonnet only)" バー（フリープラン専用）

# フリープランの場合は引き続き表示される
```

### プランの確認方法

```bash
# 現在のプランを確認する
# Claude Code の設定 → アカウント情報

# または Anthropic Console で確認
# https://console.anthropic.com/settings/billing
```

### 使用量の詳細確認

```bash
# 詳細な使用量の確認
/usage

# モデル別使用量が表示される場合:
# Claude Opus: 10,000 tokens
# Claude Sonnet: 25,000 tokens
# Claude Haiku: 10,230 tokens
# Total: 45,230 tokens

# Enterpriseプランでは管理者がチーム全体の使用量を確認できる
# https://console.anthropic.com/usage
```

## 注意点

- フリープランのユーザーには引き続き「Sonnetのみ」の使用量バーが表示されます
- ProとEnterpriseプランでは複数のモデルへのアクセスが可能なため、モデル別の使用量表示が重要です
- 使用量の詳細はAnthropicコンソールでも確認できます
- 月次使用量はプランによってリセットタイミングが異なります
