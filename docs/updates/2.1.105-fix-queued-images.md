## 原文（日本語に翻訳）

Claudeが作業中に送信されたキューイングされたメッセージへの添付画像が失われる問題を修正しました。

## 原文（英語）

Fixed images attached to queued messages (sent while Claude is working) being dropped

## 概要

Claudeが既存のリクエストを処理中に、ユーザーが画像付きメッセージをキューに入れた場合、Claudeが次のターンでそのメッセージを処理する際に画像が失われていた問題が修正されました。

## 修正内容

Claudeが処理中でも画像を添付してメッセージを送信できるようになりました。キューに入れられた画像は、Claudeが次のターンに進む際に正しく処理されます。

```
Before: Claude処理中 → 画像付きメッセージ送信 → Claudeが画像なしで処理
After:  Claude処理中 → 画像付きメッセージ送信 → Claudeが画像を含めて正しく処理
```
