# 修正：最初のメッセージが短い挨拶の場合にセッションタイトルがプレースホルダーになるバグ

## 原文（日本語に翻訳）

最初のメッセージが短い挨拶の場合にセッションタイトルにプレースホルダーのサンプルテキストが表示されていた問題を修正した。

## 原文（英語）

Fixed session titles showing placeholder example text when the first message is a short greeting.

## 概要

Claude Codeのセッションタイトルは通常、最初のメッセージから自動生成されます。しかし「こんにちは」「Hello」など短い挨拶でセッションを開始した場合、タイトル生成に失敗してプレースホルダーのサンプルテキストが表示されていたバグが修正されました。

## 基本的な使い方

### 修正後の動作

```
# 短い挨拶でセッションを開始した場合
「こんにちは」
「Hello」
「やあ」

# 修正前: セッションタイトルが "New conversation" や "Example text" などのプレースホルダー
# 修正後: 挨拶に適したタイトルまたは適切なデフォルトタイトルが設定される
```

## 注意点

- **v2.1.108で修正済み**: アップデート後は自動的に修正が適用されます。
- **影響範囲**: 最初のメッセージが短い挨拶（数文字程度）のセッションのみが影響を受けていました。
