## 原文（日本語に翻訳）

[VSCode] チャットパネルでRTL（右から左）テキスト（アラビア語、ヘブライ語、ペルシア語）が逆順に表示されていた問題を修正（v2.1.63でのリグレッション）

## 原文（英語）

[VSCode] Fixed RTL text (Arabic, Hebrew, Persian) rendering reversed in the chat panel (regression in v2.1.63)

## 概要

VSCode拡張機能版のClaude Codeのチャットパネルで、アラビア語、ヘブライ語、ペルシア語などのRTL（右から左）言語のテキストが逆順に表示されるバグが、v2.1.63 で導入されたリグレッションによって発生していました。この修正により、RTLテキストが正しい方向で表示されるようになります。

## 基本的な使い方

この修正は自動的に適用されます。VSCode拡張機能をv2.1.69にアップデートするだけで、RTLテキストが正しく表示されます。

```
# VSCode のClaude Codeチャットパネルで
# アラビア語で入力/表示する場合

# 修正前（v2.1.63以降）: テキストが逆順に表示
# مرحبا → アバحرم（逆順）

# 修正後: 正しくRTLで表示
# مرحبا → مرحبا（正常）
```

## 注意点

- この問題はVSCode拡張機能版のみで発生していました（ターミナル版では問題なし）
- v2.1.63 で導入されたリグレッションで、v2.1.69 で修正されました
- RTL言語を使用していないユーザーには影響がありません
- VSCode拡張機能をv2.1.69以降にアップデートすることで自動的に修正されます
