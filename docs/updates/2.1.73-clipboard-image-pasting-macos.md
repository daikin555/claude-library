## 原文（日本語に翻訳）

- macOSでのクリップボード画像ペーストのパフォーマンスを改善しました

## 原文（英語）

- Improved clipboard image pasting performance on macOS

## 概要

macOS環境でクリップボードに画像をコピーしてClaude Codeにペーストする際のパフォーマンスが改善されました。スクリーンショットやブラウザからコピーした画像をClaude Codeに貼り付ける操作がより高速にレスポンスするようになり、画像を使ったやり取りがスムーズになりました。

## 基本的な使い方

```bash
# macOSでスクリーンショットを撮影
# Cmd + Shift + 4 でスクリーンキャプチャ → クリップボードにコピー
# Cmd + Shift + Control + 4 でクリップボードに直接キャプチャ

# Claude Codeのプロンプトにペースト
# Cmd + V で画像をペースト
# → v2.1.73以降はペースト処理がより高速に
```

## 注意点

- この改善はmacOS環境に特化したものです（LinuxやWindowsのクリップボード処理には影響しません）
- 大きな画像（高解像度のスクリーンショットなど）の場合でもペースト速度が改善されています
- クリップボードの画像はPNG形式でAPIに送信されます
- 画像のサイズが非常に大きい場合、APIの入力トークン制限に影響する可能性があります
- クリップボードにテキストと画像の両方がある場合、画像が優先される場合があります
