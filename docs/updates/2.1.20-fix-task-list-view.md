## 原文（日本語訳）
メイン会話ビューの外側にタスクリストが表示されることがある問題を修正しました

## 原文（英語）
Fixed task list sometimes showing outside the main conversation view

## 概要
タスクリストが会話ビューの外側に表示されてしまう表示バグを修正しました。これにより、タスクの状態を常に適切な位置で確認できるようになります。

## 注意点

- **表示領域の最適化**: 修正により、タスクリストは常にメイン会話ビュー内に収まります
- **視認性の向上**: タスク管理がより直感的になり、作業の進捗を見逃しにくくなります
- **ターミナルサイズ**: 小さなターミナルウィンドウでも適切に表示されます
