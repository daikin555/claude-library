## 原文（日本語訳）
ツール使用をキャンセルする際のクラッシュを修正しました

## 原文（英語）
Fixed crashes when cancelling tool use

## 概要
Claude がツールを実行している最中にキャンセル操作を行うとアプリケーションがクラッシュする問題を修正しました。これにより、より安定した操作が可能になります。

## 注意点

- **安定性の向上**: Esc キーでツール実行をキャンセルしても安全に中断できます
- **セッション保護**: クラッシュによる作業内容の喪失がなくなります
- **応答性の改善**: ツールキャンセル時の動作がより予測可能になります
