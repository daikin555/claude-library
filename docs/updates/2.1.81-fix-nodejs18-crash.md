# 修正：Node.js 18 でのクラッシュ

## 原文（日本語に翻訳）

Node.js 18 でのクラッシュを修正。

## 原文（英語）

Fixed a crash on Node.js 18

## 概要

特定の状況下で Node.js 18 を使用している環境で Claude Code がクラッシュする問題が修正されました。Node.js 18 は LTS（長期サポート）バージョンとして多くの企業環境で使われているため、この修正により安定性が向上します。

## 基本的な使い方

修正後は、Node.js 18 環境でも Claude Code が安定して動作します：

```bash
# Node.js バージョン確認
node --version  # v18.x.x

# 通常通り起動可能
claude
```

## 実践例

### ### Node.js バージョン管理ツールとの組み合わせ

nvmや他のバージョン管理ツールを使ってNode.js 18を使用している場合：

```bash
# nvmでNode.js 18を使用
nvm use 18
node --version  # v18.x.x

# Claude Code が正常に動作
claude -p "このコードをレビューしてください"
```

### ### Dockerコンテナでの利用

Node.js 18ベースのDockerコンテナ：

```dockerfile
FROM node:18-alpine
RUN npm install -g @anthropic-ai/claude-code
CMD ["claude"]
```

## 注意点

- Claude Code の推奨 Node.js バージョンは最新の LTS バージョン
- Node.js 18 のサポート期限は2025年4月まで。それ以降は Node.js 20 以降へのアップグレードを検討
- Node.js のバージョンが古い場合は、念のため最新バージョンへのアップグレードを推奨
