## 原文（日本語に翻訳）

Fableアドバイザーの失敗を引き起こしているサーバー側の問題を修正する間、アドバイザーピッカー内でFableが一時的に「利用不可」と表示されます

## 原文（英語）

Fable temporarily shows as unavailable in the advisor picker while a server-side issue causing Fable advisor failures is fixed

## 概要

Claude Codeのadvisor（アドバイザー）機能は、メインで使うモデルとは別に、より高性能なモデルをキーとなる判断のタイミングで呼び出せる仕組みです。Fable 5をアドバイザーとして使うには、Claude Code v2.1.170以降とFable 5へのアクセス権が必要ですが、サーバー側で発生していた問題によりFableアドバイザーの呼び出しが失敗するケースがありました。この問題が修正されるまでの一時的な対応として、`/advisor` ピッカーの中でFableが「利用不可」として表示され、選択できないようになっています。これは恒久的な機能廃止ではなく、サーバー側の障害対応が完了するまでの一時的な措置です。

## 基本的な使い方

### 現在の挙動

```bash
claude

> /advisor

# ピッカーに候補が表示される
# - No advisor
# - Opus
# - Sonnet
# - Fable  (Currently unavailable)   ← 一時的に選択不可として表示

# Fableを選ぼうとしても選択できない、
# もしくは選択してもアドバイザーとしては呼び出されない
```

### 既にFableをアドバイザーに設定していた場合

```bash
claude --advisor fable

# サーバー側の問題が解消されるまでの間、
# Fableアドバイザーへの呼び出しは行われない
# （設定自体はエラーにならず保持される場合がある）
```

## 実践例

### 一時的に代替のアドバイザーへ切り替える

```bash
claude

> /advisor opus

# Fableが利用不可の間は、Opusをアドバイザーに設定して
# 同等の「セカンドオピニオン」的な運用を継続する
```

### settings.jsonのadvisorModelを一時的に変更する

```json
{
  "advisorModel": "opus"
}
```

Fableの復旧後にFableへ戻したい場合は、再度 `/advisor fable` を実行するか、この値を `"fable"` に書き戻します。

### アドバイザーを一時的にオフにする

```bash
claude

> /advisor off

# アドバイザー呼び出し自体を止め、
# メインモデル単体での作業に切り替える
```

### 状況を確認しながら復旧を待つ

```bash
claude

> /advisor

# Fableの表示が "Currently unavailable" から
# 通常の選択可能な状態に戻っているかを都度確認する
# 復旧后は特別な操作なしにFableが再び選択できるようになる想定
```

## 注意点

- これはFable自体の恒久的な提供終了ではなく、Fableアドバイザー呼び出しの失敗を引き起こしていたサーバー側の問題に対する一時的な緩和策です。修正が完了すれば、通常どおりピッカーから選択できるようになる見込みです
- Fable 5をアドバイザーとして使うには、そもそもClaude Code v2.1.170以降とFable 5へのアクセス権（組織単位）が必要です。利用不可の表示が出た場合、まずこの前提条件を満たしているかも確認してください
- advisorは実験的機能であり、Anthropic API経由でのみ利用可能です。Amazon Bedrock、Claude Platform on AWS、Google CloudのAgent Platform、Microsoft Foundryでは利用できません
- アドバイザーはメインモデルと同等以上の能力を持つ必要があります（例: Sonnet 5メインにSonnet 4.6アドバイザーは拒否されます）。Fableが使えない間は、Opusなど許容されるペアリングの中から選ぶ必要があります
- `/advisor` の選択内容はユーザー設定の `advisorModel` に保存されます。組織の `availableModels` の許可リストでFableが除外されている場合も同様に選択できない状態になります
