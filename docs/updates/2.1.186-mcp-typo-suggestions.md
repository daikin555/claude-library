# claude mcp get/remove でサーバー名のタイポ修正候補を表示

## 原文（日本語に翻訳）
`claude mcp get` と `claude mcp remove` を改善し、タイポ時に最も近い設定済みサーバー名を提案し、長いサーバーリストを省略表示するようになりました。

## 原文（英語）
Improved `claude mcp get` and `claude mcp remove` to suggest the closest configured server name on a typo and truncate long server lists

## 概要
`claude mcp get` や `claude mcp remove` コマンドで存在しないサーバー名を指定した場合、最も近い名前の候補を自動的に提案するようになりました。また、登録済みサーバーが多数ある場合のリスト表示が省略形式になり、見やすくなっています。タイポによるエラーを素早く修正できます。

## 基本的な使い方

### タイポ時の修正候補表示
```bash
# 「github」の代わりに「githb」を入力した場合
claude mcp get githb

# → エラーメッセージとともに候補を提案:
# エラー: サーバー「githb」が見つかりません
# もしかして: github
```

### サーバー情報の取得
```bash
claude mcp get github
```

### サーバーの削除
```bash
claude mcp remove github
```

## 実践例

### タイポ修正フロー
```bash
# 誤入力
claude mcp remove githb
# エラー: サーバー「githb」が見つかりません。
# 候補: github

# 正しい名前で再実行
claude mcp remove github
```

### 長いサーバーリストの管理
多数のMCPサーバーが設定されている環境でも、コマンドのエラーメッセージが読みやすく省略表示されます：
```bash
claude mcp list
# server1
# server2
# server3
# ... (他15件)
```

### よくあるタイポパターン
```bash
# タイポ例 → 候補
claude mcp get githab    # → github
claude mcp get slak      # → slack
claude mcp get filessytem  # → filesystem
```

## 注意点
- タイポ候補はレーベンシュタイン距離などの編集距離アルゴリズムで計算されます
- 候補が複数ある場合は最も近い1つが表示されます
- 候補の提案はあくまで参考です。実際の操作は正しいサーバー名で確認して実行してください
- 長いサーバーリストの省略は表示のみで、実際のサーバー設定は変更されません

## 関連情報
- MCPサーバーの管理: `claude mcp --help`
- サーバー一覧: `claude mcp list`
- サーバー追加: `claude mcp add --help`
