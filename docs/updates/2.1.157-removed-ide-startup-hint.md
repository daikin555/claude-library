## 原文（日本語に翻訳）

「/ide で…」という起動時のヒントトースト通知を削除しました。

## 原文（英語）

Removed the "/ide for …" startup hint toast

## 概要

Claude Code の起動時に表示されていた `/ide` コマンドに関するヒントトースト通知が削除されました。起動画面をよりシンプルにするためのUI改善です。`/ide` コマンド自体は引き続き使用可能で、IDE連携機能は変わらず利用できます。

## 基本的な使い方

IDE連携機能を使用する場合は、引き続き `/ide` コマンドを使用できます。ヒントが表示されなくなっても、機能に変更はありません。

```
/ide
```

## 注意点

- ヒントトーストが削除されても `/ide` コマンドは引き続き利用可能です
- IDE連携機能（VS Code、JetBrains等との連携）は変わらず動作します
- 起動時のメッセージが減り、インターフェースがよりすっきりします
- v2.1.157 以降で自動的に適用されます
