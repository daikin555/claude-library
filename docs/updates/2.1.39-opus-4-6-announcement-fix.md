## 原文（日本語）
Opus 4.6のリリース告知がBedrock/Vertex/Foundryユーザーに誤って表示されていた問題を修正しました。

## 原文（英語）
Fixed Opus 4.6 launch announcement showing for Bedrock/Vertex/Foundry users

## 概要
Claude Opus 4.6モデルのリリース告知が、AWS Bedrock、Google Cloud Vertex AI、またはAnthropic Foundryを使用しているユーザーに対して誤って表示されていた問題が修正されました。これらのプラットフォームでは、モデルの利用可能性が異なるため、不要な告知が表示されないようになりました。

## 基本的な使い方

この修正は自動的に適用されるため、特別な操作は不要です。

```bash
# Bedrock/Vertex/Foundryを使用している場合
claude --provider bedrock  # Opus 4.6の告知は表示されない
claude --provider vertex   # Opus 4.6の告知は表示されない
claude --provider foundry  # Opus 4.6の告知は表示されない

# 通常のAnthropic APIを使用している場合
claude  # Opus 4.6が利用可能な場合は告知が表示される
```

## 実践例

### AWS Bedrockでの使用
```bash
# Bedrockを使用している環境
export CLAUDE_PROVIDER=bedrock

# Claude Codeを起動
claude

# 修正前: Opus 4.6の告知が表示される（利用不可能なのに）
# 修正後: 告知は表示されない（適切）
```

### Google Cloud Vertex AIでの使用
```bash
# Vertexを使用している環境
claude --provider vertex

# 修正により、Vertex AI固有の利用可能なモデルのみが案内される
# Opus 4.6の不要な告知は表示されない
```

### 複数プロバイダーの使い分け
```bash
# 開発環境: Anthropic API（Opus 4.6利用可能）
claude --profile dev
# → Opus 4.6の告知が表示される

# 本番環境: AWS Bedrock（Opus 4.6未対応）
claude --profile prod --provider bedrock
# → 不要な告知は表示されない
```

## 注意点

- この修正は告知表示の制御のみで、モデルの利用可能性自体は変更しません
- 各プロバイダーでのモデル提供状況は、それぞれのプラットフォームのドキュメントを確認してください
- 将来的にBedrock/Vertex/FoundryでOpus 4.6が利用可能になった場合は、適切に告知が表示されます
