## 原文（日本語に翻訳）

- 長いセッションでフィードバックアンケートが頻繁に表示される問題を修正

## 原文（英語）

- Fixed feedback survey appearing too frequently in long sessions

## 概要

Claude Codeの長時間セッションにおいて、フィードバックアンケートが過度に頻繁に表示される問題が修正されました。アンケートは製品改善のために重要ですが、作業の流れを妨げるほど頻繁に表示されるのは望ましくありません。この修正により、適切な間隔でアンケートが表示されるようになり、集中して作業を続けられます。

## 基本的な使い方

この修正は自動的に適用され、ユーザーの操作は不要です:

```bash
# 通常通りClaude Codeを使用
claude
> 長時間の開発セッション...
# フィードバックアンケートが適切な頻度で表示される
```

## 実践例

### 長時間の開発セッション

数時間にわたる開発セッションでの体験が改善されます:

```
claude
> 大規模なリファクタリングを進めて
> テストを修正して
> ドキュメントを更新して
# ...数時間の作業を続けても、アンケートが何度も表示されない
```

## 注意点

- フィードバックアンケート自体が廃止されたわけではなく、表示頻度が適切に調整されました
- アンケートへの回答はClaude Codeの改善に活用されるため、余裕がある際は回答をお勧めします
- セッションの長さに応じた適切な間隔でアンケートが表示されます
