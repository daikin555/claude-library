## 原文（日本語に翻訳）

セッションに不正なテキストブロックが含まれる場合の再開時クラッシュを修正しました。

## 原文（英語）

Fixed crash on resume when session contains malformed text blocks

## 概要

保存されたセッションに不正な形式のテキストブロックが含まれている場合、`/resume`またはセッション再開オプションでセッションを再開しようとした際にClaude Codeがクラッシュしていた問題が修正されました。不正なテキストブロックが存在してもセッションの再開が安全に行えるようになりました。

## 修正内容

```
Before: 不正なテキストブロックを含むセッション → 再開時にクラッシュ
After:  不正なテキストブロックを含むセッション → 安全に再開（問題部分はスキップまたは修復）
```
