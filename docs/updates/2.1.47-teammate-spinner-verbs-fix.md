## 原文（日本語に翻訳）

チームメイトスピナーが設定のカスタムspinnerVerbsを反映していなかった問題を修正 (anthropics/claude-code#25748)

## 原文（英語）
Fixed teammate spinners not respecting custom spinnerVerbs from settings (anthropics/claude-code#25748)

## 概要

Claude Codeの設定でカスタム `spinnerVerbs` を設定していても、チームメイト（並列エージェント）のスピナーに反映されていなかった問題が修正されました。この修正により、チームメイトモードでも個人の設定に合わせたスピナーテキストが正しく表示されるようになります。

## 基本的な使い方

```json
// ~/.claude/settings.json でのカスタムspinnerVerbs設定例
{
  "spinnerVerbs": [
    "考え中",
    "処理中",
    "調査中",
    "作業中"
  ]
}
```

```bash
# 設定後にClaude Codeを起動すると、チームメイトスピナーにもカスタム動詞が表示される
claude
```

## 実践例

### 日本語のスピナーテキストを設定する

```json
// ~/.claude/settings.json
{
  "spinnerVerbs": [
    "考え中",
    "検討中",
    "分析中",
    "実行中",
    "確認中"
  ]
}
```

修正後は、チームメイトモードで複数のエージェントが動作する際も、上記のカスタムテキストがスピナーに表示されます。

### プロジェクト固有のスピナーVerbs設定

```json
// プロジェクトルートの .claude/settings.json
{
  "spinnerVerbs": [
    "ビルド中",
    "テスト中",
    "デプロイ中",
    "検証中"
  ]
}
```

```bash
# チームメイトを使った並列処理でもプロジェクト固有のスピナーが表示される
claude
```

### デフォルトのspinnerVerbsとカスタム設定の比較

```json
// デフォルトのspinnerVerbs（設定しない場合）
// "Thinking", "Working", "Processing" などの英語テキスト

// カスタム設定（修正後はチームメイトスピナーにも反映される）
{
  "spinnerVerbs": [
    "Analyzing",
    "Coding",
    "Reviewing",
    "Optimizing"
  ]
}
```

## 注意点

- この修正はv2.1.47以降で有効です
- `spinnerVerbs` の設定はユーザーレベルの `~/.claude/settings.json` またはプロジェクトレベルの設定ファイルで行えます
- チームメイト機能はClaude Codeの並列エージェント機能で、複数のClaudeインスタンスが協調して作業する場合に使用されます
- スピナーのカスタマイズは見た目の変更のみで、動作には影響しません
