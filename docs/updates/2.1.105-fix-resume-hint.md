## 原文（日本語に翻訳）

`/resume`、`--worktree`、または`/branch`の後に終了した際に「Resume this session with...」ヒントが表示されない問題を修正しました。

## 原文（英語）

Fixed "Resume this session with..." hint not printing on exit after `/resume`, `--worktree`, or `/branch`

## 概要

`/resume`コマンドでセッションを再開した後、`--worktree`フラグを使って起動した後、または`/branch`コマンドを使用した後にClaude Codeを終了した際、通常表示されるはずの「このセッションを再開するには...」というヒントメッセージが表示されなかった問題が修正されました。

## 修正内容

```
Before: /resume → 作業 → 終了 → ヒントが表示されない
After:  /resume → 作業 → 終了 → "Resume this session with: claude --resume <session-id>" を表示
```

## 注意点

- セッションを後で再開できるように、このヒントメッセージを確認・保存しておくことを推奨します
