## 原文（日本語に翻訳）

[VSCode] Opus 4.8 でファストモードインジケーターが表示されない問題を修正しました。

## 原文（英語）

[VSCode] Fixed the fast mode indicator not appearing on Opus 4.8

## 概要

VS Code拡張機能において、Opus 4.8 モデルを使用している際にファストモード（高速モード）のインジケーターが正しく表示されないバグが修正されました。この問題により、Opus 4.8 使用時にユーザーが現在ファストモードで動作しているかどうかを視覚的に確認できない状況が発生していました。v2.1.157 以降ではインジケーターが正常に表示されます。

## 影響を受けていた状況

- VS Code 拡張機能を使用しているユーザー
- Opus 4.8 モデルを選択して使用している場合
- ファストモードが有効な状態でのUI表示確認時

## 注意点

- v2.1.157 以降では自動的に修正されており、特別な操作は不要です
- この修正はVS Code拡張機能に限定されます
- Opus 4.8 以外のモデルには影響しない修正です
