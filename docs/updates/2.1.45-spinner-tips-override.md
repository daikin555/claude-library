## 原文（日本語に翻訳）

スピナーTipsをカスタマイズする `spinnerTipsOverride` 設定を追加 — カスタムTipの文字列配列を `tips` に設定し、オプションで `excludeDefault: true` を設定することでデフォルトのTipsの代わりに独自のTipsのみを表示できます

## 原文（英語）

Added `spinnerTipsOverride` setting to customize spinner tips — configure `tips` with an array of custom tip strings, and optionally set `excludeDefault: true` to show only your custom tips instead of the built-in ones

## 概要

Claude Code 2.1.45 では、処理中に表示されるスピナーのTips（ヒント文）をカスタマイズできる `spinnerTipsOverride` 設定が追加されました。チームの規約や社内ドキュメントへの参照、モチベーションメッセージなど、独自のTipsを表示することができます。`excludeDefault: true` を設定することで、デフォルトのTipsを非表示にして独自Tipsのみを表示することも可能です。

## 基本的な使い方

`~/.claude/settings.json`（またはプロジェクトの `.claude/settings.json`）に `spinnerTipsOverride` を追加します。

```json
{
  "spinnerTipsOverride": {
    "tips": [
      "コードレビューのチェックリストを確認しましょう！",
      "テストを書くことを忘れずに。",
      "ドキュメントを最新の状態に保ちましょう。"
    ]
  }
}
```

## 実践例

### カスタムTipsをデフォルトに追加する

デフォルトのTipsに加えて独自のTipsを表示する場合（`excludeDefault` を省略または `false` に設定）：

```json
{
  "spinnerTipsOverride": {
    "tips": [
      "社内 Wiki: https://wiki.example.com",
      "困ったときは #dev-help チャンネルへ",
      "PRのタイトルは日本語で書いてね！"
    ]
  }
}
```

### カスタムTipsのみを表示する

デフォルトのTipsを非表示にして、完全にオリジナルのTipsのみ表示する場合：

```json
{
  "spinnerTipsOverride": {
    "tips": [
      "チームのコーディング規約: https://standards.example.com",
      "毎日15分のコードレビュータイムを設けましょう",
      "コミットメッセージは動詞で始める（Add / Fix / Update / Remove）",
      "型エラーは早めに解決すると後で楽になります"
    ],
    "excludeDefault": true
  }
}
```

### 動機付けメッセージを表示する

長時間の処理中に表示するモチベーションアップのメッセージ：

```json
{
  "spinnerTipsOverride": {
    "tips": [
      "今日も良いコードが書けています！",
      "コーヒーを一杯どうぞ ☕",
      "良い休憩は生産性を高めます"
    ],
    "excludeDefault": false
  }
}
```

## 注意点

- `tips` は文字列の配列です。各要素が1つのTipsとして表示されます。
- `excludeDefault` を省略した場合、デフォルトは `false`（デフォルトのTipsも表示）となります。
- `excludeDefault: true` にすると、Claude Code 標準のTipsは一切表示されなくなります。
- 設定はユーザーレベル（`~/.claude/settings.json`）またはプロジェクトレベル（`.claude/settings.json`）どちらでも使用できます。
- Tipsはランダムに表示されます。順番は保証されません。
