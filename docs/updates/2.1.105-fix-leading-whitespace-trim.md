## 原文（日本語に翻訳）

アシスタントメッセージから先頭の空白が除去され、ASCIIアートや字下げされた図が壊れていた問題を修正しました。

## 原文（英語）

Fixed leading whitespace being trimmed from assistant messages, breaking ASCII art and indented diagrams

## 概要

Claudeの応答に含まれるASCIIアート、テキストベースの図、インデントされた構造図などが、先頭の空白文字が削除されることで正しく表示されなかった問題が修正されました。ASCII絵文字、ツリー図、アーキテクチャ図などが意図通りに表示されるようになります。

## 修正内容

アシスタントメッセージの先頭空白が保持されるようになりました。

**修正前（先頭空白が削除されて崩れた例）:**
```
src
├── components
│   ├── Button.tsx
│   └── Input.tsx
└── pages
    └── index.tsx
```

**修正後（先頭空白が保持された例）:**
```
    src
    ├── components
    │   ├── Button.tsx
    │   └── Input.tsx
    └── pages
        └── index.tsx
```
