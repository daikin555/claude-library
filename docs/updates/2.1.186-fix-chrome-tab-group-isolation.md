# 並行CLIセッションで製品内権限ゲートが無効な場合にChromeタブグループの分離が適用されない問題を修正

## 原文（日本語に翻訳）
並行CLIセッションで製品内権限ゲートが無効な場合に、Chromeタブグループの分離が適用されない問題を修正しました。

## 原文（英語）
Fixed Chrome tab-group isolation not applying when the in-product permissions gate is off for concurrent CLI sessions

## 概要
Claude CodeをChromeのブラウザ統合（`claude in chrome` 機能）で使用し、複数のCLIセッションを並行実行している場合に、製品内権限ゲートが無効になっていると、タブグループによる分離が機能しない問題がありました。この修正により、並行CLIセッションでもChromeタブグループの分離が正しく適用されます。

## 基本的な使い方

この修正は自動的に適用されます。Chrome統合機能使用時の分離が改善されます。

### 影響を受けていたシナリオ
- 複数のCLIセッションを並行して実行
- Chrome統合（claude in chrome）を使用
- 製品内権限ゲートが無効な設定

### 修正後
上記の条件下でも、Chromeタブグループの分離が正しく機能します。

## 注意点
- Chrome統合機能を使用しているユーザーに適用されます
- 並行CLIセッションのセキュリティ分離が改善されます
