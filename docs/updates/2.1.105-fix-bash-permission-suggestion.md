## 原文（日本語に翻訳）

プランモードを終了する際に高い権限レベルからダウングレードになる`acceptEdits`パーミッションモードをBashツールが誤って提案する問題を修正しました。

## 原文（英語）

Fixed Bash tool suggesting `acceptEdits` permission mode when exiting plan mode would downgrade from a higher permission level

## 概要

プランモードを終了する際に、Bashツールが現在の権限レベルよりも低い`acceptEdits`モードを提案してしまう問題が修正されました。例えば、`bypassPermissions`モードで動作中にプランモードを終了する場合、`acceptEdits`へのダウングレードが提案されなくなりました。

## 修正内容

```
Before: bypassPermissions で動作中 → プランモード終了 → acceptEdits を提案（ダウングレード）
After:  bypassPermissions で動作中 → プランモード終了 → 現在のモードを維持（正しい提案）
```

## 注意点

- パーミッションモードのレベル順序: `default` < `acceptEdits` < `bypassPermissions`
- 高いレベルのモードから低いレベルへの不必要なダウングレードが提案されなくなりました
