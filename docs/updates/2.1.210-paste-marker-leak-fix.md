## 原文（日本語に翻訳）

Claude Codeから開いた外部エディタに貼り付けマーカーが漏れてしまう不具合を修正しました。この不具合は、貼り付けたテキストの前後に見慣れないÈ/É文字が現れる形で発生することがありました

## 原文（英語）

Fixed paste markers leaking into external editors opened from Claude Code, which could appear as stray È/É characters around pasted text

## 概要

Claude Codeでは `Ctrl+G` などを使ってプロンプトを外部エディタ（Vim、VS Codeなど）で編集できます。この際、Claude Codeの入力欄では貼り付けられたテキストを内部的に識別するため、ターミナルの「ブラケットペーストモード」の開始・終了マーカーを利用しています。しかし外部エディタを起動する際にこのマーカーが正しく除去されず、貼り付けたテキストの前後に `È` や `É` といった見慣れない文字がそのまま混入して表示されてしまう不具合がありました。Claude Code v2.1.210ではこの不具合が修正され、外部エディタに渡されるテキストからペーストマーカーが適切に除去されるようになりました。

## 基本的な使い方

### 修正前の問題

```bash
claude

# 長いプロンプトをクリップボードから貼り付け、
# Ctrl+G で外部エディタ（例: vim）を開く
> Ctrl+G

# vimが開いた際、貼り付けたテキストの前後に
# 次のような見慣れない文字が混入することがあった
#
# È function calculateTotal(items) {
#   return items.reduce((a, b) => a + b, 0);
# }É
```

### 修正後の動作

```bash
claude

> Ctrl+G

# vimが開いた際、貼り付けたテキストは
# 余計な文字が混入せずクリーンな状態で表示される
#
# function calculateTotal(items) {
#   return items.reduce((a, b) => a + b, 0);
# }
```

## 実践例

### VS Codeを外部エディタとして使う場合

```bash
export EDITOR="code --wait"
claude

> （長いプロンプトを貼り付けてから）Ctrl+G

# 修正後: VS Codeで開かれるファイルにも
# È/É などのマーカー文字が混入しない
```

### コード片をそのままプロンプトへ貼り付けるワークフロー

```bash
# クリップボードにコードスニペットをコピーしておき、
# Claude Codeのプロンプトへ貼り付けてから外部エディタで編集する場面

claude

> このコードをレビューして: (Ctrl+V で貼り付け)
> Ctrl+G で編集画面を開く

# 修正後は貼り付けたコードの構文が
# マーカー文字によって壊れることがない
# → シンタックスハイライトやインデントも正しく機能する
```

### /vim モードとの併用

```bash
claude
> /vim
# プロンプト入力にvimキーバインドを有効化

# その状態でクリップボードからの貼り付け＋Ctrl+Gによる
# 外部エディタ起動を行っても、修正後は
# ペーストマーカーの文字化けは発生しない
```

## 注意点

- この修正はClaude Code v2.1.210（2026年7月15日リリース）で行われました
- 根本原因は、外部エディタ起動時にターミナルの「フォーカスレポート」機能は正しく無効化・再有効化されていた一方、「ブラケットペーストモード」の無効化が行われていなかったことにあります
- ブラケットペーストモードが有効なままだと、ペーストの開始・終了を示すエスケープシーケンス（`ESC[200~` / `ESC[201~`）がエディタ側にそのまま渡され、一部の環境ではこれが `È`/`É` のような文字として画面に表示されてしまいます
- この問題は特にVimやNeovimなど、ブラケットペースト対応が独自実装になっているターミナルエディタで顕在化しやすい傾向がありました
- 修正後は外部エディタ起動前にブラケットペーストモードを確実に無効化し、エディタ終了後に元の状態へ復元するようになっています
- 同様の外部エディタ機能として、貼り付けた長いテキストを再度貼り付け操作することでインライン展開できる「paste again to expand」機能もあわせて提供されています
